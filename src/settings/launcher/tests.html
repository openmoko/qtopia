Test cases 



 Background

Test Purpose
 Verify that the background can be set for each launcher category individually.
Test Description
  1. Select the Applications category, change to the Background tab, and change the Style to "Solid Color". Pick a color from the color picker. Click OK.
  2. Select the Applications tab. Verify the background color is now the color selected from the color picker.
  3. Restart Settings->Launcher
  4. Pick the "Image" Style and select an image. Click OK.
  5. Select the Applications tab. Verify that the background shows the selected image (one or multiple times).
  6. Restart Settings->Launcher
  7. Pick the "Faint Ruled" Style. Click OK.
  8. Select the Applications tab. Verify the background is a subtle combination of light and darker horizontal lines.
  9. Repeat for the other categories and inspect the apropriate tab (e.g. Games, Settings, etc) each time.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test


 Font

Test Purpose
 Verify that the font size can be set for each launcher category individually.
Test Description
  1. Select the Applications category, change to the Font tab, and change the Family, Size and / or Color. Click OK.
  2. Select the Applications tab. Verify the icon font has been changed according to your selection.
  3. Restart Settings->Launcher, select the Applications category again and check that the Font tab reflects the latests changes.
  4. Change the Family, Size and Color again. Click OK.
  5. Select the Applications tab. Verify the icon font has been changed according to your selection.
  6. Repeat for the other categories.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test


 Icon Size

Test Purpose
 Verify that the icon size can be adjusted for each tab of the launcher individually.
Test Description
  1. Select the Applications category, and check the "Large" icon size from the Icons tab. Click OK.
  2. Select the Applications tab and verify that the icons are large.
  3. Start Settings->Launcher again.
  4. Verify that the "Large" radiobutton is selected.
  5. Change the icon size to "Small". Click OK.
  6. Select the Applications tab. Verify the icons are small.
  7. Start Settings->Launcher again.
  8. Verify that the "Small" radiobutton is selected.
  9. Repeat for the other categories.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test
