Launcher Settings Test List

Feature Tests

All launcher tabs

  1. Start the Launcher Settings application.
  2. Verify that all launcher tabs are available in the Category listview.

Icon size

  1. Select the Applications category, and check the "Large" icon size from the Icons tab. Click OK.
  2. Select the Applications tab. Verify the icons are large.
  3. Change the icon size to "Small". Click OK.
  4. Select the Applications tab. Verify the icons are small.
  5. Repeat for the other categories.

Background

  1. Select the Applications category, change to the Background tab, and change the Style to "Solid Color". Pick a color from the color picker. Click OK.
  2. Select the Applications tab. Verify the background color is now the color selected from the color picker.
  3. Pick the "Faint Ruled" Style. Click OK.
  4. Select the Applications tab. Verify the background is the same as it was before the test started.
  5. Repeat for the other categories. Also try using the Image style to tile the background with an image.

Font size

  1. Select the Applications category, change to the Font tab, and change the Family, Size and / or Color. Click OK.
  2. Select the Applications tab. Verify the icon font has been changed according to your selection.
  3. Repeat for the other categories.
