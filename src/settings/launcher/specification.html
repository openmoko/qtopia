Launcher Settings Specification

The Launcher Settings application modifies the appearance of the launcher tabs.

The Launcher Settings application modifies the appearance of the launcher tabs. This allows the user to customize the each launcher tab as they choose. Each tab can have a different appearance.

Status: RT

Use Cases
  • Icon names are too small

    The names below the icons are either too small to read, or induce eye strain. The user uses the launcher settings application to increase the font size to make the icon descriptions easier to read.

Features
  • All launcher tabs: All launcher tabs can be customized.
  • Icon size: Large or small icons can be displayed.
  • Background: The background can be ruled, a solid color, or have an image tiled across it.
  • Font size: The icon Font can be modified - different family, size and color.
