Sound Settings Specification

The Sound Settings application allows the user to configure sound settings on the Qtopia device.

The Sound Settings application allows the user to configure sound settings on the Qtopia device. Screen and keyboard taps can be disabled, and the audio out volume can be adjusted.

Status: RT

Use Cases
  • Feedback required when using the keyboard

    A user who is unfamiliar with the amount of force required to press keys on the keypad can enable keyboard clicks. This helps the user become acquainted with how the keypad works.

  • Feedback required when using the screen

    A user who is unfamiliar with the amount of force required to press on the touchscreen can enable screen clicks. This helps the user become acquainted with how the touchscreen works.

  • Volume control

    A user who rarely changes the sound volume may choose not to install or activate the Volume Applet. In this case, he will use the Sound Settings to change the volume.

Features
  • Keyboard sound: The keyboard sound can be enabled or disabled.
  • Screen sound: The screen sound can be enabled or disabled.
  • Volume control: The volume of the audio output port can be adjusted from mute to maximum.

Prerequisites
  • Sound hardware required: The Qtopia device must have some means of emitting sound.
  • Keyboard: Required for keyboard sounds.
