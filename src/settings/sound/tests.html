Qtopia/Sound

Test cases 



 Audible keyboard clicks

Test Purpose
 Verify that keyboard clicks can be made audible.
Test Description
  1. From the Sound Settings application, check Keyboard sound and tap OK. 
  2. Open the keyboard and press some keys.
  3. Verify that for each keypress a sound is emitted.
  4. Start Sound again, Uncheck Keyboard sound and tap OK. 
  5. Verify that keyboard presses are silent.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test

 Volume control

Test Purpose
 Verify that the volume can be controlled.
Test Description
  1. Check Screen sound move the volume slider all the way to the top, and click OK. 
  2. Verify that screen taps are now at high volume.
  3. Open Sound again and move the volume slider midway and click OK. 
  4. Verify that screen taps are now half as loud.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test
