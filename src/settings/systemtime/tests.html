Date / Time Settings Test List

Feature Tests

Date and Time adjustment

  1. Start the Date / Time Settings application from the Settings tab.
  2. Change the Hour and Minute. Tap . Note the new time in the clock applet in the taskbar.
  3. Select a different Timezone and tap . Note that the time in the clock applet has changed to match that of the timezone.

Six available timezones

  1. Tap the World Time icon .
  2. Tap on one of the six city buttons, then tap and hold around the world map. Different cities are displayed with their local time. Release on any city. The city button now has that cities' time in it.
  3. Tap to close the World Time viewer.
  4. Tap the Time Zone selector. Note that the timezone previously selected from theWorld Time application is now in the list.

Display configuration

  1. Toggle the Time Format listview between 24 and 12 hour. Note that the AM/PM listview will toggle between enabled (12-hour) and disabled (24-hour). Also note that when this is performed in the afternoon, the Hour field will change automatically.
  2. Weeks start on can either be Monday or Sunday. Change to Monday and tap . Start the Calendar application from the Applications tab, and switch to the week or month views. Note that the weeks now begin on Monday.
  3. Use the Date Format listview to select which format the date should appear in. The changes are reflected in the Date listview.
