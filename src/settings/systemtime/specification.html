Date / Time Settings Specification

The Date / Time Settings application modifies the device's date and time.

The Date / Time Settings application allows the user to set the date and time. Multiple timezones can be setup for the traveller.

Status: BT

Use Cases
  • Moving between timezones

    Timezone information for multiple cities can be set. When travelling, the user need only change the timezone to match the date and time in the new timezone.

Features
  • Date and Time adjustment: Both the date and time can be changed for any particular timezone.
  • Six available timezones: Date and time information can be stored for up to six different localities around the world.
  • Display configuration: The time can be shown in 12-or-24 hour format. The first day of the week can be set, and the date format can be modified.

Prerequisites
