Appearance Specification

The appearance settings allow the user interface appearance to be customized.

The appearance settings allow the user interface color scheme, widget style, window decoration style and font face and size to be customized,

Status: BT

Use Cases
  • Color Scheme

    A fashion consious user likes their PDA UI to match the clothes they wear.

  • Widget Style

    A user finds the default widget style to be boring. They must be able to select a style that better suits their tastes.

  • Window Decoration

    Similar to widget styles, the decoration around the windows is an expression of the users individuality. The user must be able to select a window decoration that suits their own tastes.

  • Font

    A visually impaired person must be able to select a larger font in a serif face.

Features
  • Color scheme: allow selection from a list of color schemes.
  • Widget Style: allow selection from a variety of widget style plugins.
  • Window Decoration: allow selection from a variety of window decoration plugins.
  • Font: allow selection of font face and size.
  • Preview: provide a preview of the combination of the selected color scheme, widget style, window decoration and font.

Prerequisites
