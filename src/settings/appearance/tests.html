Feature Tests

Color Scheme

  1. Select each color scheme in the list. Confirm that the preview is updated to show the new color scheme.
  2. Select a color scheme different to the current scheme and press OK. Confirm that the Launcher is using the new scheme. Run sysinfo and confirm that it is using the new color scheme.

  2. Select a widget style different to the current style and press OK. Confirm that the Launcher is using the new style. Run sysinfo and confirm that it is using the new widget style.

  1. Select each window decoration in the list. Confirm that the preview is updated to show the new window decoration.
  2. Select a window decoration different to the current style and press OK. Run sysinfo and confirm that it is using the new window decoration.

Font

  1. Select each font face in the list. Confirm that the preview is updated to show the new face. Confirm that the font size combo shows the font sizes available for each font.
  2. Select the Helvetica font. Select each of the font sizes available. Confirm that the preview is updated to show the new font size.
  3. Select a font face and size different to the current and press OK. Confirm that the Launcher is using the new font in the tab title. Run sysinfo and confirm that it is using the new font.
