Qtopia/Appearance

Test cases 



 Color Scheme

Test Purpose
 Verify that the Color scheme can be changed.
Test Description
  1. Select the "Colors" tab
  2. Select the available options (e.g. Blues, Bright, Desert, etc) one ofter the other.
  3. Observe the results in the sample shown below the colors listbox.
  4. Select a color style and close the app using OK.
  5. Check that the color style of Qtopia has changed as well.
  6. Start an application and check that the chosen color style is applied.
  7. Open Appearance again and check that in the Colors tab the current selection is highlighted.
  8. Select a different color style and do the same check again: are the colors changed again for all applications and Qtopia launcher?
Expected TestResult
 All applications including Qtopia launcher should use the selected color style as defined by Appearance.

Available Color schemes:
  1. Blues
  2. Bright
  3. Desert
  4. Gray
  5. Green
  6. Purple
  7. Yellow
Testnorm/Criteria
 Exact Match

 Fonts

Test Purpose
 Verify that the standard Font can be changed for all applications.
Test Description
  1. Select the Font tab and select each Font name (e.g. Helvetica, Micro, etc) one after the other.
  2. Observe the sample dialog just below the listbox and verify that the displayed font is updated with each selection and that the displayed font matches your expectation.
  3. Select the Utopia font and check that the Font size can be varied. 
  4. Ceck that the sample dialog adjusts the presentation according to the selected font size.
  5. Select a large font size and press OK.
  6. Verify that the new font/size is used in all Qtopia applications and in the launcher.
  7. Check that the presented names of applications are still readable (it is possible that a name is split up in two lines if the chosen font is 'too' large).
  8. Open Appearance again and check that in the Font tab the currently selected font is highlighted and the current font size is shown.
  9. Select a different Font + Font size and redo the test.
Expected TestResult
 The selected font should be used in all applications (where text is to be shown) including the launcher.

Available fonts and font sizes:
  1. Babelfish: 8, 9 10, 11, 12
  2. Fixed: 7, 12
  3. Helvetica: 8, 10, 12
  4. Micro: 4 only
  5. Smoothtimes: 10 only
  6. Unifont: (fixed font)
  7. Utopia: 8, 9, 10, 11, 12
Testnorm/Criteria
 Exact Match
Test Suitability

 Widget Style

Test Purpose
 Verify that a Widget Style can be selected and will be used by all Qtopia applications.
Test Description
  1. Select the Style tab.
  2. Select the different offered styles (e.g. Qtopia, Windows, Flat, etc) one after the other and observe the results in the Sample dialog below the listbox.
  3. Select a different style than the one that is currently used and press OK
  4. Open "Application Services" and check that the scrollbar, listview and radiobuttons look as expected.
  5. Open "Backups" and check that the buttons look as expected.
  6. Open Appearance again and check that in the Style tab the current selection is highlighted.
  7. Select a different style (or the original style) and press OK.
  8. Check again with some applications if the used style matches your expectations.
Expected TestResult
 The used style typically has an effect on everything _inside_ a mainwidget or dialog. So it defines how buttons, scrollbars, editors, etc look like. It does not specify how the outline of the widget (the title bar, and bounding rectangle of the mainwidget/dialog look like).

Available Widget styles:
  1. Qtopia
  2. Windows
  3. Flat
  4. Fresh
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test


 Window Decoration

Test Purpose
 Verify that the Window Decoration can be changed.
Test Description
  1. Select the "Windows" tab
  2. Select the available options (e.g. Qtopia, Flat, etc) one after the other.
  3. Observe the sample dialog just below the listbox and verify that the output matches your expected result.
  4. Select a different Window Decoration than the one that was currently being used and press OK.
  5. Open another application and check that the new decoration is applied (mostly noticeable by the title bar and buttons inside it).
  6. Open Appearance again and check that in the Windows tab the current selection is highlighted.
  7. Make a new selection and repeat the test.
Expected TestResult
 The decoration is mostly visible in the title bar of an application and the bounding rectangle of the mainwidget/dialog.

Available Decorations:
  1. Qtopia
  2. Flat
  3. Polished
