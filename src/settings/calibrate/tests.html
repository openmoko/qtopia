Recalibrate Test List

Feature Tests

Multiple hitpoints

  1. Start the Recalibrate application from the Settings menu.
  2. Repeatedly tap the target in the centre as it moves around the screen. After 5 taps, the Recalibrate application will quit, indicating successful screen calibration.
  3. Start the Recalibrate application again.
  4. Repeatedly tap in the top left corner of the screen. Note that the Recalibrate application doesn't quit after 5 taps; each tap must be within a specific tolerance for correct calibration.
