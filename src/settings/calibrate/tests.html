Qtopia/Calibrate

Test cases



 First Use

Test Purpose
 Verify that the calibrate app is started automatically when first using the device.
Test Description
  1. Flash the device using the current ROM or upgrade to a newer ROM.
  2. Restart the device.
  3. Verify that the Calibrate app is started automatically after the system has booted.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test


 Normal Use

Test Purpose
 Verify that the screen can be calibrated.
Test Description
  1. Start Calibrate
  2. Verify that the app presents a + at the lower left corner of the screen.
  3. Tap on the Qt logo e.g. NOT on the +.
  4. Repeat for all other locations.
  5. Verify that after the 5th tap the calibrate app restarts in the left lower corner, e.g. the entries you made are rejected.
  6. Tap as accurately as possible on the + and repeat 4 times.
  7. Verify that Calibrate closes automatically.
  8. Tap on several elements on the device and verify that the system responds as expected, e.g. buttons must be depressed, scrollboxes can be dragged, etc.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test
