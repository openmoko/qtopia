Qtopia/Button Editor

There are four hw buttons that can be configured in this app.
We'll name them "Notes", "Person", "Envelop" and "House" from top to bottom.
Each hw button can be configured for two modes: a normal mode and a hold mode.
The following testcases will refer to "Notes", "Person", ... and "Normal" and "Hold" all the time.

Test cases 



 Assigning applications

Test Purpose
 Verify that applications can be assigned to the special hardware keys and are opened when the keys are pressed.
Test Description
  1. Assign "File Manager" to "Notes/Normal" 
  2. Assign "Todo list" to "Notes/Hold"
  3. Assign "Contacts" to "Person/Normal"
  4. Assign "Calculator" to "Person/Hold"
  5. Assign "EMail" to "Envelope/Normal"
  6. Assign "New EMail" to "Envelope/Hold"
  7. Assign "System Info" to "House/Hold"
  8. Press OK
  9. Press the "Notes" hw button and verify that "File Manager" is opened.
  10. Press the "Person" hw button and verify that "Contacts" is opened.
  11. Press and hold for a little while the "Envelope" button and verify that the "New E-mail" dialog is opened.
  12. Do similar checks for all buttons/modes
  13. Start Buttons again
  14. Assign "Calendar" to "Notes/Normal" and press OK
  15. Press the "Notes" hw button and verify that "Calendar" is opened.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test
