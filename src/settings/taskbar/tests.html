Taskbar Settings Test List

Feature Tests

Add / Remove applets

  1. Start the Taskbar Settings application.
  2. Uncheck the Battery and Clock applets. Click OK. The Battery and Clock applets will be removed from the taskbar.
  3. Start the Taskbar Settings application and check the Battery and Clock applets. Click OK. The Battery and Clock applets appear in the taskbar.

Launcher tabs in Q-Menu

  1. Uncheck the Show Launcher tabs in Q-Menu box in the Taskbar Settings application. Click OK.
  2. Click the Q-Menu . Only applets that use the Q-Menu will be shown.
  3. Check the Show Launcher tabs in Q-Menu box in the Taskbar Settings application. Click OK
  4. Click the Q-Menu. The menu will show entries for each of the launcher tabs in addition to any applets that use the Q-Menu.
