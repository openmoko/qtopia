Taskbar Settings Specification

TaskBar Settings allow taskbar applets to be loaded and unloaded while the Qtopia-powered device is running.

Taskbar applets are programs that run in the background of Qtopia. They show information about the running system, as well as providing a shortcut to their owner application.

Status: BT

Use Cases
  • Removing the Clipboard applet from the taskbar

    A user can remove the Clipboard applet from the taskbar by unchecking the box next to Clipboard in the Taskbar Settings application.

Features
  • Add / Remove applets: Applets can be added and removed from the taskbar by checking or unchecking the box next to the applet name.
  • Launcher tabs in Q-Menu: The launcher tabs' application links can be shown or not shown in the Q-Menu.

Prerequisites
  • Adding / removing: Requires applets to be available to Qtopia.
