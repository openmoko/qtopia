Test cases 



 Set Rotation

Test Purpose
 Verify that the rotation can be globally set for qtopia.
Test Description
  1. Open the Clock application (so you can use it for verification).
  2. Go to the Rotation app again and click one of the four offered rotation options and click OK.
  3. Verify that the launcher has updated the output to match the selected rotation.
  4. Open Clock and verify that the rotation is as expected.
  5. Repeat the same for all four offered options.
Expected TestResult
 When you select a rotation option, physically rotate the device until you can normally read the 'abc' you've selected and then click OK.
While holding the device in the same physical position all screen output should be readable without the need to turn your head.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test
