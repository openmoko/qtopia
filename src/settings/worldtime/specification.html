World Time Specification

The World Time application displays the current time in various places around the world.

The World Time application can be used to see what the current day and time it is in other cities around the world. The day and time for up to 6 cities can be remembered without having to search for the city on the map.

Status: BT

Use Cases
  • International Traveller

    An international traveller who needs to know the time in different cities in order to coordinate meetings.

  • Checking timezone differences

    Handy as a check to see how many hours behind / ahead a particular city is in relation to another.

Features
  • World Map: Displays a clickable map of the world.
  • Reference Cities: Up to 6 cities' times can be shown at a single time.
  • Map Zooming: Useful for closely-packed cities.

Prerequisites
