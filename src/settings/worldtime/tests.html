Qtopia/World time

Test cases 



 World map

Test Purpose
 Verify that time zone information is presented in the world map.
Test Description
  1. Start the World Time application from the Applications tab. 
  2. Verify that a map of the world is displayed above a set of six buttons, each with a different city name and time.
  3. Click and hold anywhere on the map. If the click position was close enough to a city, the city name and the current day and time will be displayed in a popup window. If the click position wasn't close enough, drag until a popup appears.
  4. Verify the presented information for a number of known cities and time zones.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test


 Reference cities

Test Purpose
 Verify that six cities can be selected as reference cities.
Test Description
  1. Click any pushbutton. The button will remain depressed.
  2. In the same manner as above, find a city on the world map and then release the pointer.
  3. Verify that the city's name and time will appear on the depressed pushbutton. 
  4. Verify that the pushbutton will return to it's normal state.
  5. Verify that the presented date and time info behind the button is according to the newly entered city.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test


 Map zooming

Test Purpose
 Verify that the map of the world can be enlarged.
Test Description
  1. Click the Zoom button in the bottom right of the world map. 
  2. Verify that the world map is enlarged and that scrollbars are added to the view.
  3. Using the scrollbars, move around the world map. Selecting a city is done in the same fashion as previously.
  4. Click the Zoom button to return to normal view.
  5. Verify that the normal map view is restored.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test


 Day and time presentation

Test Purpose
 Verify that the presented date and time info for the reference cities is correct.
Test Description

  2. Start Date and Time and set a new time.
  3. Verify that the presented day and time info is updated accordingly.
  4. Start Date and Time again and reset the time to the correct value.

 All the verification steps as described in the Test Description must be met.
