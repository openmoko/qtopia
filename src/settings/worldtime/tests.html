World Time Test List

Feature Tests

World Map

  1. Start the World Time application from the Applications tab. A map of the world will be displayed above a set of six buttons, each with a different city name and time.
  2. Click and hold anywhere on the map. If the click position was close enough to a city, the city name and the current day and time will be displayed in a popup window. If the click position wasn't close enough, drag until a popup appears.

Reference Cities

  1. Click any pushbutton. The button will remain depressed.
  2. In the same manner as above, find a city on the world map. Release the pointer, and that city's name and time will appear on the depressed pushbutton. The pushbutton will return to it's normal state.

Map Zooming

  1. Click the Zoom button in the bottom right of the world map. This will enlarge the world map, and add scrollbars.
  2. Using the scrollbars, move around the world map. Selecting a city is done in the same fashion as previously.
  3. Click the Zoom button to return to normal view.
