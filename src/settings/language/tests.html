Qtopia/Language


Note: The execution of this test depends on the availability of certain languages.  If a language mentioned in the test doesn't exist then go on with the next language. The basic idea is that each language available on the device should be tested.

Test cases 



 Language selection

Test Purpose
 Verify that the language can be changed and that all text presented on the device (except the information in e-mail and private documents of course) is translated to the selected language.
Test Description
  1. Select "German" as the language
  2. Press OK
  3. Verify that all the tab pages of the launcher show German titles e.g. "Anwendungen", "Spiele", "Einstellungen", "Documenten".
  4. Verify that all application names have been replaced by German names, do likewise for Games and Settings.
  5. Start applications and verify that all text has been replaced to German text.
  6. Start the Help application and verify that help documentation is available for all/most applications in the selected language.
  7. Repeat the test by selecting Japanese and finally English.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Language must match the language specific syntax rules.
Test Suitability
 Manual test
