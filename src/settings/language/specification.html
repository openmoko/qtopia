Language Setting Specification

Allows the user to choose User Interface language.

The user can choose the language to be used for the interface and the documentation of applications. If the language chosen is not available for a particular application, a default is used.

Status: RT

Use Cases
  • Initial Purchase

    A Norwegian-speaking user purchases her device in Austria. The default language for the device is German. By using the Language Settings, shown as "Sprache" in the "Einstellungen" tab, but fortunately with a recognization icon, she changes the language to Norwegian, shown as "Norwegisch (Norsk)". (not implemented: the first use application gives the user the option to choose language).

  • Language Learner

    User is an American recently relocated to Italy. In order to facilitate his learning of Italian, he switches his device to Italian. Occassionally though, he switches back to English for some more complicated applications.

Features
  • Choose language: all available languages are listed for selection.
  • Language translation: each language option is shown in that language, and in the current language.
  • Documentation translation: the language selection also determines the language used for Help Documentation.
  • No change to current applications: applications must be restarted before they use the newly selected language.

Prerequisites
  • Translations: only applications which have appropriate translation files will be translated - others will use a default language.
