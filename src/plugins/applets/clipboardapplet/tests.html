Clipboard Applet Test List

Feature Tests

Text Cut

  1. Start the Text Editor application.
  2. Open a new document and enter some text. Highlight the text by dragging, and select Cut from the Clipboard Applet popup menu. The highlighted text will disappear from the document. Close the document.
  3. Open another new document. Select Paste from the Clipboard Applet. The text from the original document is pasted into the new document.
  4. Run the Email application, and choose Compose to start a new letter. Tap in the message area, and select Paste from the Clipboard Applet popup menu. The previously cut text will appear.

Text Copy

  1. Start the Text Editor application.
  2. Open a new document and enter some text. Highlight the text by dragging, and select Copy from the Clipboard Applet popup menu. Close the document.
  3. Open another new document. Select Paste from the Clipboard Applet. The text from the original document is pasted into the new document.
  4. Run the Email application, and choose Compose to start a new letter. Tap in the message area, and select Paste from the Clipboard Applet popup menu. The previously copied text will appear.
