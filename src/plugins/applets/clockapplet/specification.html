Clock Applet Specification

The Clock applet displays the current time and provides access to the clock application

The Clock applet displays the current time in the taskbar. The Clock applet also allows the user to set the time, access the in-built clock application and toggle the setting of the daily alarm.

Status: BT

Use Cases
  • View the current time

    The user can see the current time by looking at the taskbar.

  • Modify the daily alarm

    The user is going on holiday and doesn't wish to be woken. The user is able to turn the daily alarm off from the clock applet popup.

  • View the clock fullscreen

    The clock application can be started from the clock applet. The default screen is a fullscreen view of the time.

Features
  • Time display: The current time is displayed in the taskbar.
  • Time set: The current time can be set.
  • Clock application: The clock application can be started from the applet.
  • Daily alarm: The daily alarm can be modified from the applet.
