Clock Applet Test List

Feature Tests

Time display

  1. The current time is displayed in the bottom right corner.

Time set

  1. Tap the clock applet. From the popup menu, select Set Time.
  2. The Set System Time dialog will appear. Change the displayed time or change the timezone and click OK. The applet will change to display the time selected.

Clock application

  1. Tap the clock applet. From the popup menu, select Clock.
  2. The Clock application will run.

Daily alarm toggle

  1. Tap the clock applet. From the popup menu, select Alarm.... The Clock application will run with the Alarms page showing.
  2. Tap Enabled to enable or disable the daily alarm.
