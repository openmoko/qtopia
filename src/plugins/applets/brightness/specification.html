Brightness Applet Specification

The Brightness Applet changes the screen brightness.

The Brightness Applet allows the user to adjust the brightness of the screen. This is precisely equivalent to running the Light & Power settings program, adjusting the brightness for the current power mode, and quiting the Light & Power settings program.

Status: RT

Use Cases
  • Travelling user on a train enters a tunnel

    The user increases the brightness of the screen to compensate for the darkness of the tunnel.

Features
  • Adjusts brightness: User-adjustable screen brightness.
