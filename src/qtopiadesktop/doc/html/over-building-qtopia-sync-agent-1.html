Building Qtopia Sync Agent

These instructions tell you how to build Qtopia Sync Agent, how to run a build and how to deploy Qtopia Sync Agent.

Requirements

Qtopia Sync Agent is known to build with Microsoft Visual C++ 6 (VC6), Microsoft Visual Studio .NET 2003 Standard Edition (VC7) and Microsoft Visual C++ Express 2005 (VC8).

You must have the Platform SDK installed to build Qtopia Sync Agent.

You must have Outlook installed to build a fully-functional Qtopia Sync Agent.

Setup

Ensure that a suitable compiler is installed and the appropriate environment variables are set. You will also need to install the platform SDK to build Qtopia Sync Agent.

You should set QMAKESPEC to a suitable value.

CompilerQMAKESPEC
VC6win32-msvc
VC7win32-msvc.net
VC8win32-msvc2005

Build

    configure
    nmake
    nmake install

Execute Unit Tests

There is only minimal test coverage at this time, mostly aimed at the Outlook interface.

    cd tests\src\qtopiadesktop
    nmake
    nmake test

Running

    dimage\qtopiadesktop.exe

Deployment

You can build an installer for Qtopia Sync Agent. Please see src/qtopiadesktop/dist/README for details.
