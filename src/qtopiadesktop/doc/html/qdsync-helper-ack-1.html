HELPER_ACK

Usage

    HELPER_ACK

Description

Sent from the device to the desktop in response to any command (if the HELPER_INIT command enabled it). This is required by some connection types (which block unless input is available).

Note: This command is removed from the command stream by the connection helper.

See also qdsync Protocol.
