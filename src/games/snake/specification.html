Snake Specification

Guide the snake around the screen eating mice.

Move the snake around the game screen eating mice. The game ends if the snake hits a wall or it's own tail.

Status: RT

Use Cases
  • User with time to kill.

    A user has some spare time and decides to play a game.

Features
  • New Game: A new game may be started at any time.
  • Keypad Control: The snake is moved by using the keypad.

Prerequisites
  • Keypad: Required for snake control.
