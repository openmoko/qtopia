ParaShoot Specification

Shoot the parachuters before they hit the ground.

Shoot the parachuters before they hit the ground. If a parachuter makes it to the ground, he will destroy part of the base. Three parachuters will completely destroy the base and the game will end.

Status: RT

Use Cases
  • User with time to kill.

    A user has some spare time and decides to play a game.

Features
  1. New Game: A new game may be started at ay time.
  2. Keypad Control: The turret is controlled by using the keypad.

Prerequisites
  • Keypad: Required for turret control.
