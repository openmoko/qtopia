Text Editor Test List

Feature Tests

Create new text file

  1. Start Text editor from the launcher
  2. Click the New file button
  3. Verify that an empty file is display with the name "Unnamed"
  4. Enter some text eg: "1234 test"
  5. Click the OK button. This
  6. Rename the created file by clicking and holding the stylus on the last file listed. Enter an unique file name in the "Name" field

Edit an existing text file

  1. Start Text editor from the launcher
  2. Select a existing file from the list of file, as named in previous step
  3. Verify that the file contains "1234 test"
  4. Change the text in the file and click OK button
  5. Verify that the changes made have been saved
  6. An alternate path from step 2 is to make changes to the file click the Close button And verify that the changes made were not saved

Find text

  1. Start Text editor from the launcher
  2. Select a existing file from the list of file, as named in previous step
  3. Click Find button
  4. Enter "t" in the topmost edit field
  5. Verify that the first "t" is now displayed in highlighted font

Repeat last text find

  1. Followimg on from previous test case
  2. Click the Find nextButton and verify that the second "t" is now displayed in highlighted font

Copy text

  1. Start Text editor from the launcher
  2. Select a existing file from the list of file, as named in previous step
  3. Select the first line text by in the file draging stylus across from left to right
  4. Click the Copy button

Paste text

  1. Following on from previous test case, click the Paste button
  2. Verify the text previously copied has now been pasted into where the text cursor is postioned

Cut text

  1. Following on from previous test case, select the first line text by in the file draging stylus across from left to right
  2. Click the Cut button
  3. Verify that the select text has been removed from the file

Line Wrapping

  1. Click and drag the entire line of text. Click Copy from the toolbar or select Copy from the Edit menu.
  2. Place the cursor at the end of the line, and click Paste from the toolbar or select Paste from the Edit menu. Repeat until text appears on the second line.
  3. Uncheck Wrap lines from the View menu. The second line disappears, and it's contents are placed back on the first line.

Variable Level Zoom

  1. Select Zoom in from the View menu. The text will appear slightly larger. Repeatedly selecting Zoom in from the View menu will cause the text to increase in size.
  2. Select Zoom out from the View menu. The text will appear slightly smaller. Repeatedly selecting Zoom out from the View menu will cause the text to decrease in size.
