Today Specification

The Today application displays information related to "today".

The Today application provides a summary of events related to the current date. This can include such things like appointments (from the Calendar application), new or unsent email (from the Email application) or pending tasks (from the Todo application).

Status: BT

Use Cases
  • Summary of pending appointments

    A list of appointments for the next week can be displayed.

Features
  • Default Plugins: Summaries from Calendar, Email and Todo applications.
  • Dynamic Plugin Loading: Plugins can be added / removed from the current Today view at run-time.
  • Plugin Configuration: Each plugin has it's own configuration dialog, which can be customised.
  • Application Launcher: Each plugin can launch it's owner application.
  • Event Reloading: Adjustments made to events via owner applications will be reflected in the plugin summary.
  • Completely Plugin Driven: New plugins can be written to display any sort of information.
