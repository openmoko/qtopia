Feature Tests

Default Plugins

  1. Start the Today application from the Applications tab.
  2. The application will be broken up into 4 parts:
    • The current date will be displayed at the top.
    • The default plugins (Calendar, Email and Todo) will provide summaries, each in their own section. Summaries will be shown only for those applications that have any relevant information.
  3. Clicking on the date at the top of the screen will show display the current month as a popup.

Dynamic Plugin Loading

  1. Select Edit Today View from the corner box in the bottom right of the Today screen.
  2. Click on the Calendar plugin, followed by the Down arrow on the right of the screen. This will move the Calendar summary below the Email summary. Click OK in the top right of the window. The Email summary will now appear before the Calendar summary.
  3. Return the Calendar plugin to the top of the list.

Plugin Configuration

  1. Select Calendar plugin... from the corner box in the bottom right of the Today screen.
  2. Modify how many days over which events should be shown (Events the next...) and the maximum number of events (Maximum listing). Click Ok in the top right of the window.
  3. Depending on the number of events in the Calendar, the contents of the Calendar summary may or may not change.
  4. Other plugins have different options, but are all accessed in the same fashion.

Application Launcher

  1. Clicking on the icon representing each plugin will launch that application. (eg: clicking on the Calendar icon will launch the Calendar application).
  2. Clicking on an entry in a summary list will also launch the owner application (eg: clicking on an entry in the Calendar list will launch the Calendar application, with a day view of the date of the entry).

Event Reloading

  1. Click on the Calendar icon on the Today screen to launch the Calendar.
  2. Enter a new event in the current day view. Close the Calendar.
  3. The Today application will update the Calendar summary with the new event.
