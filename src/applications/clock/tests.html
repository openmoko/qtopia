ClockTest List

Feature Tests

Clock

  1. Start the clock.
  2. Confirm that the Clock tab is displayed.
  3. Check that the time displayed on the analog clock matches the device system time.
  4. Check that the time displayed on the digital clock matches the device system time.
  5. Check that the date displayed matches the device system date.

Stopwatch

  1. Start the clock.
  2. Click on the Stopwatch tab.
  3. Confirm that the stopwatch reads 00:00:00.00 and that the analog hands indicate 00:00.
  4. Press the Start button and confirm that the analog and digital timers count up (resolution 1 sec).
  5. Press the Stop button and confirm that the timers have stopped. The digital timer should show 1/100 sec resolution.
  6. Press the Reset button and confirm that the stopwatch reads 00:00:00.00 and that the analog hands indicate 00:00.
  7. Press the Start button and wait 3 seconds. Press the Reset button and confirm that the stopwatch resets to 0, and continues to count up correctly.

Daily Alarm

  1. Start the Clock.
  2. Click on the Alarms tab.
  3. Set the alarm time 1-5 minutes in the future and check the Enabled box.
  4. Confirm that when the alarm time is reached the alarm sounds and a message box is displayed with the alarm details.
  5. Set the alarm time 1-5 minutes in the future and uncheck the Enabled box.
  6. Confirm that when the alarm time is reached no alarm is sounded.
  7. Set the alarm time 1-5 minutes in the future, check Enabled, and close the clock application.
  8. Confirm that when the alarm time is reached the alarm sounds and a message box is displayed with the alarm details.
