Help Browser Specification

The Help Browser provides the user with short help for applications, settings and information about Qt/E and Qtopia.

The Help Browser provides the user with short help for applications, settings and information about Qt/E and Qtopia.

Status: RT

Use Cases
  • First use

    The user has started Qtopia for the first time. After looking at the index page they browse the pages that describe the commonly used applications and settings.

  • How do I

    The user has a specific task in mind, they look at the index page and browse the topic that best matches what they need to do.

  • Provide help when using an application

    The user is in the middle of using an application, and they what help using the application. They then click the help button and the applications help is displayed in the Help Browser. They then close the Help Browser and continue using the application.

  • Navigate to home page

    In the middle of viewing a page the user selects to navigate to the home page.

  • Navigate to previous page

    In the middle of viewing a page the user selects to navigate to a previously viewed page.

  • Add bookmark

    The user finds a page of interest in the Help Browser and chooses to add bookmark.

  • Browse to bookmark

    The user selects the bookmark to browse to from all possible bookmarks.

  • Remove bookmark

    The user selects the bookmark to be removed from all possible bookmarks.

Features
  • Navigation: All navigation operations can be achieved by a single tap of stylus.
  • Page display : Web pages are display in the highest detail and colour suitable by the devices Qtopia is installed onto. Changes to appearance settings that the user makes are applied to Help browser.
  • History : A history of viewed pages is maintained such that most recently viewed page is added to the top of the history list.
  • Bookmarks : Bookmarks can be added and removed using the Bookmarks menu in Help Browser.
