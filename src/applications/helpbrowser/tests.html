Help Browser Test List

Feature Tests

First use

  1. Start Qtopia, and proceed to launcher
  2. Start Help Browser.
  3. Check that it is possible to browse to applications and use help for
    • Help Browser
    • Address Book
    • Email
  4. Clock home button and check that it is possible to browse to setup and use help for
    • Add/Remove software
    • Appearance
    • Date/Time
    • Network

How do I

Consider what types of tasks the user will want to do. Check that there is clear where to find information about that task.

Provide help when using an application

For each application and setting accessable via the launcher check that there the help button is available at the top of the applications window that when the clicked causes the browser to

Navigate to home page

For every page accessable in the Help Browser check that the home button is available. This test case would only need to be done before a major release.

Navigate to previous page

  1. Browse to applications index page
  2. Browse to address book page
  3. Click the Home button
  4. Verify that clicking the Back button
    5. twice causes the applications index page to be displayed

Add bookmark

  1. Browse to the applications index page
  2. Click add bookmark item from the Bookmark menu

Browse to bookmark

  1. Browse to the home page
  2. Click the book mark added in previous test case
  3. Verify that the applications index page is now show

Remove bookmark

  1. Click the book mark added in previous test case
  2. Click the remove bookmark item from the Bookmark menu
  3. Verify that the applications index page has been removed from list of bookmarks
