Qtopia/Sys Info

Test cases 



 Memory

Test Purpose
 Verify that presented memory information is correct
Test Description
  1. Confirm that the Memory tab is displayed as the default.
  2. Check that the Total memory displayed matches the device memory size.
  3. Check that the graphical display matches the numertical values for Used, Buffers, Cached and Free.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test


 Storage

Test Purpose
 Verify that the presented storage information is correct.
Test Description
  1. Click on the Storage tab.
  2. Check that the top section reads "Internal Storage"
  3. Check that there is one section for each removable storage card installed on the device.
  4. For each section, check that the total/used/available information matches what is actually installed on the device.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test


 CPU

Test Purpose
 Verify that the presented CPU usage is correct.
Test Description
  1. Click on the CPU tab.
  2. Check that the Type info matches the CPU type on the device.
  3. Verify that the Application/CPU usage is shown as a graph.
  4. Verify that the graph is scrolling slowly sideways.
  5. Start media player and play a video.
  6. Verify that the System CPU usage drops and the Application CPU usage rises.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test


 Version

Test Purpose
 Verify that sys info is presenting the correct version information.
Test Description
  1. Click on the Version tab.
  2. Check that the Qtopia version info matches the version of Qtopia supposed to be installed on the device.
  3. Check that the Qtopia Compiled By info is correct, and not "Unknown". It should be "Trolltech" if this is a version supplied directly by Trolltech, otherwise it should be the name of the vendor.
  4. Check that the Qtopia "Built on" date is correct.
  5. Check that the Linux Kernel info matches what is supposed to be installed on the device.
  6. (If there is more info displayed) Check that any other information displayed matches what is supposed to be installed.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test
