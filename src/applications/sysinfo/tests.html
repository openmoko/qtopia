System InfoTest List

Feature Tests

Memory

  1. Start system info.
  2. Confirm that the Memory tab is displayed.
  3. Check that the Total memory displayed matches the device memory size.
  4. Check that the graphical display matches the numertical values for Used, Buffers, Cached and Free.

Storage

  1. Start system info.
  2. Click on the Storage tab.
  3. Check that the top section reads "Internal Storage"
  4. Check that there is one section for each removable storage card installed on the device.
  5. For each section, check that the total/used/available information matches what is actually installed on the device.

CPU

  1. Start system info.
  2. Click on the CPU tab.
  3. Check that the Type info matches the CPU type on the device.
  4. Verify that the Application/CPU usage is shown as a graph.
  5. Verify that the graph is scrolling slowly sideways.

Version

  1. Start system info.
  2. Click on the Version tab.
  3. Check that the Qtopia version info matches the version of Qtopia supposed to be installed on the device.
  4. Check that the Qtopia Compiled By info is correct, and not "Unknown". It should be "Trolltech" if this is a version supplied directly by Trolltech, otherwise it should be the name of the vendor.
  5. Check that the Qtopia "Built on" date is correct.
  6. Check that the Linux Kernel info matches what is supposed to be installed on the device.
  7. (If there is more info displayed) Check that any other information displayed matches what is supposed to be installed.
