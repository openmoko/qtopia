ImageViewer Specification

The Imageviewer applications allows viewing of pictures.

The ImageViewer can be used to load and view images in different formats.

Status: BT

Use Cases
  • Viewing pictures taken from a digital camera

    Pictures taken on a digital camera can be stored on removable media that may be compatible with a handheld device running Qtopia (eg. SD or Compact FLASH card ports available on the Sharp Zaurus). The media can be inserted into the handheld, and pictures can be viewed in a slideshow fashion.

Features
  • File list: All available pictures are listed in one place.
  • Image Formats: All supported Qt formats, including PNG, BMP, XBM, XPM, PNM, JPEG, MNG, GIF and NPD.
  • Rotation and Fullscreen: Pictures can be rotated and sized to make better use of available display space.
  • Slideshow: Displays images one after another, with a configurable pause in-between.
  • Document View: Launchable from the document view.
  • Fast Load: load large images faster.
  • Keypad Control: Traverse the list of images using the keypad.
  • Rotate-to-fit: Have the image automatically rotated on load to fit the available screen space.

Prerequisites
  • Fast load: Only available for JPEG images.
