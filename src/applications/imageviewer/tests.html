Imageviewer Test List

Feature Tests

File List

  1. Start the ImageViewer application from the Applications tab. A list of available images will show up.
  2. Click any image file. The image will be displayed.
  3. Return to the file list by closing the window (clicking the X in the top right corner), pressing ESCape, or clicking the File button in the toolbar.
  4. To view the next image in the list, click the Next button in the toolbar, select Next from the View menu, or the Right key on the keypad.
  5. To view the previous image in the list, click the Prev button in the toolbar, select Previous from the View menu, or the Left key on the keypad.

Image Formats

  1. Create or locate at least one image in each of the following formats: PNG, BMP, XBM, XPM, PNM, JPEG, MNG, GIF and NPD.
  2. Make these files available to the QPE server program.
  3. Start the ImageViewer application, and view each picture.

Rotation and Fullscreen

  1. Load any image.
  2. Click the rotation buttons in the toolbar. The image should rotate and flip accordingly.
  3. Select Horizontal flip and Vertical flip from the Image menu. The image should flip accordingly.
  4. Use the Up and Down keys on the keypad to rotate the image.
  5. To make the image fullscreen, click the fullscreen button in the toolbar, select Fullscreen from the View menu or click anywhere on the image. The image will resize to the size of the screen. Images that are not sized in proportion to the screen will have a surrounding black edge. Click in the window or press ESCape on the keypad to return to normal view.

Slideshow

  1. To start the slideshow, click the slideshow button in the toolbar or select Slideshow from the Image menu. The first image in the list will be displayed. After a pause, the next image in the list will be displayed.
  2. To change the amount of time between each image, select Settings from the View menu, and move the slider to increase or decrease the amount of time to wait between each image.
  3. To have the images loop (the first image is shown after the last image) check the Repeat slideshow option in the Settings dialog.
  4. To reverse the direction of images that are shown, check the Show pictures in reverse option.
  5. To view the next image in the list, click the Next button in the toolbar, select Next from the View menu, or the Right key on the keypad. The timeout period is reset after each image is loaded.
  6. To view the previous image in the list, click the Previous button in the toolbar, select Previous from the View menu, or the Left key on the keypad. The timeout period is reset after each image is loaded.

Document View

  1. Select an image from the document view. The ImageViewer will start and load the selected image.

Fast load

  1. Uncheck Fast load pictures from the Settings dialog. Large images will take some time to load (around 8 seconds for a 2048x2048 image).
  2. Check Fast load pictures from the Settings dialog. Large images will load much faster (around 2 seconds for a 2048x2048 image).

Rotate-to-fit

  1. Check Rotate pictures to fit screen from the Settings dialog. Any picture loaded will be rotated to best fit the available space.
  2. Check Rotate clockwise to specify a clockwise rotation on load, or leave it unchecked to specify a counter-clockwise rotation.
