File Manager Specification

The File Manager allows the user to browse the filesystem.

The File Manager allows the user to browse the filesystem.

Status: BT

Use Cases

Features
  • File and Directory View: Displays files and directories.
  • Directory Traversal: Allows traversal up and down the directory tree.
  • File Management: High-level editing of files can be performed (cut, copy, paste, delete, rename).
  • Application launching: Applications associated with a file can be launched.
  • Document Shortcut: Documents can be added to the Document view.

Prerequisites
