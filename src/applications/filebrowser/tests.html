File Manager Test List

Feature Tests

File and Directory View

  1. Start the File Manager from the Applications tab of the launcher. A list of files and directories will be displayed.

Directory Traversal

  1. Move up the directory tree by selecting Dir, followed by a sub-directory, or click on the Up button on the toolbar.
  2. Move to a previously viewed directory by clicking the Left button on the toolbar.
  3. Move down the directory tree by clicking on a directory.
  4. The size of a file is displayed on the right of the listview, as well as the last date of modification.
  5. Directory contents can be sorted by clicking on the Sort menu, followed by by Name, by Size, by Date, or by Type. Sort order can be changed by checking or unchecking Ascending. Note that a directories' contents are always displayed with sub-directories before files.

File Management

Cut

  1. Select the file(s) to be moved and click on the Cut button in the toolbar. Alternatively, select the file(s), click and hold, and select Cut from the popup menu. The file(s) will disappear from the directory view.

Copy

  1. Select the file(s) to be copied and click on the Copy button in the toolbar. Alternatively, select the file(s), click and hold, and select Copy from the popup menu.

Delete

  1. Select the file(s) to be copied, click and hold, and select Delete from the popup menu. The files will disappear from the directory view.

Paste

  1. Select the file(s) to be pasted and click on the Cut button in the toolbar. Alternatively, select the file(s), click and hold, and select Cut from the popup menu. The files will disappear from the directory view.
  2. Navigate to the required destination directory. Click the Paste button in the toolbar, or click and hold in the listview, and select Paste from the popup menu. The previously cut files appear in the current directory.
  3. Navigate to a different directory and try to paste. An error message will appear indicating that the paste was unsuccessful (no files were previously cut).

Rename

  1. Click and hld the file to be renamed, and select Rename from the popup list. Type the new name. Click on the listview to finish renaming.

Application Launching

  1. Navigate to a directory containing files of a mime-type handled by an application in Qtopia (eg: navigate to a directory containing a text file or JPEG image). Double click on the file and the appropriate application will be launched with the file as the default.
  2. Alternatively, click and hold the file. The popup menu will have an entry Open in X, where X is the default application for that file type.

Document Shortcut

  1. Navigate to a directory containing files of a mime-type handled by an application in Qtopia (eg: navigate to a directory containing a text file or JPEG image). Click and hold on the file, and select Add to Documents from the popup menu. Close the File Manager and select the Documents tab to verify the file was added. Clicking on the new addition will launch the application associated with the file.
