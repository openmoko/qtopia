Qtopia/Media player

Test cases 



 Play music

Test Purpose
 Verify that music files can be played.
Test Description
  1. Select music file to play.
  2. Click on play button.
  3. Verify that music plays correctly.
  4. Test with files of different output bit rates (48kHz, 44kHz, 22kHz etc).
  5. Test with files of different volumes (classical, jazz, rock etc).
  6. Test with files of different number of channels (stereo and mono).
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test

 Hard keys

Test Purpose
 Verify that the essential functions can be controlled using hard keys.
Test Description
  1. Verify that you can use the Left and Right hardkeys to toggle between the Audio, Video and Playlist tab pages.
  2. Verify that you can use the Up and Down hardkeys to select a file inside a tab page.
  3. Verify that you can use the Centre hardkey to start the selected file.
  4. Verify that after the Centre hardkey is pressed (start play) the media player starts playing the selected file. The GUI that is applied depends on whether an audio file or a video file is selected.
  5. Verify that you can use the Up and Down hardkeys to control the volume.
  6. Verify that you can use the Left and Right hardkeys to select the previous/next music or video file.
  7. Verify that you can use the Centre hardkey to start/stop the current music or video file.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test


 Full screen video

Test Purpose
 Verify that a video file can be played in full screen mode.
Test Description
  1. Select a video file to play
  2. Click on play button
  3. Click on the video output view e.g. somewhere on the video screen.
  4. Verify that the video is played in fullscreen mode, e.g. not title bar, no menu bar, no status bar and the video as large as possible.
  5. Click on the Home button and start another application.
  6. Tap on the Video player icon on the taskbar.
  7. Verify that the media player is restored again in full screen mode.
  8. Tap on video image again to return from fullscreen mode
  9. Verify the same as above happens when in normal mode that it restores to normal mode.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test

 Editing the playlist

Test Purpose
Test Description
  1. Go to the Audio tab in the file selection view of media player.
  2. Select a file and click on the add icon in the toolbar.
  3. Go to the Playlist tab and verify that the file is added to the playlist.
  4. Repeat for a video file and verify that the playlist now shows the audio and video file.
  5. Click on Playlist > Clear  and verify that the playlist is empty again
  6. Click on Playlist > Add all audio and verify that the playlist contains all files mentioned in the Audio tab.
  7. Clear the playlist again.
  8. Click on Playlist > Add all Videos and verify that the playlist contains all files mentioned in the Video tab.
  9. Clear the playlist again.
  10. Click on Playlist > Add all and verify that all audio and video files are add to the playlist.
  11. Go to the Playlist tab and select a file.
  12. Click on the Remove from playlist button and verify that the selected file is removed from the playlist.
  13. Quit the media player and open it again and ensure the play list is reloaded as it was before quitting.
Expected TestResult
 All the verification steps as described in the Test Description must be met.
Testnorm/Criteria
 Exact Match
Test Suitability
 Manual test
