HOWTO update zoneinfo data

- Ideally the system integrator should add the matching zoneinfo data to match the glibc version of the toolchain.
  Root file system default zoneinfo directory (/usr/share/zoneinfo)

- To update with new version of zoneinfo here:
  +Remove all file in zoneinfo directory except this one
  +Copy /usr/share/zoneinfo directory contents from host to this directory
  +Remove right and posix directories from this location
  +Estimated size 2M
