Edit Ring Profiles

  • Go to Settings and launch Ring Profiles by highlighting it and pressing .
  • Use to highlight the ring tone, then press and select Edit from the menu.
  • Use the Up / Down arrows to move between fields and the Left / Right arrows to move between tabs.
  • To select a field highlight it and press .
  • Use the keypad to enter text and to tick / untick check-boxes.
  • Press to finish a field.
  • Press to save the profile and exit.
