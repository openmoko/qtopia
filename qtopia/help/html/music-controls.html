Music Controls

The following controls are available: play / pause, stop, rewind / fast forward, previous / next and close to return to the playlist.

%%PHONE%%

The functions of the change when an audio file is being played. The new functions are displayed on the screen.

@@PHONE@@

Play / Pause

When playing, the play button is used to pause, and when paused, pressing it again resumes playing. The hardware select button on the device can also be used for this.

Stop

If pressed the track will cease being played. If Play /Pause is pressed now the track will start from the beginning.

Rewind / Fast Forward

These buttons are used to seek through a file. The hardware left / right buttons can also be used: hold down the button in the direction you wish to seek, then release the button when it reaches the desired position (the track will continue playing from there).

It is also possible to use the position slider at the bottom of the window to instantly position the playback to a given position.

Previous / Next

The previous / next buttons are for moving through the playlist. The hardware buttons can also be used (left / right buttons).

Close

%%PHONE%%

Press to stop playing and return to the list of music files.

@@PHONE@@ %%PDA%%

Tap to stop playing and return to the list of music files. The hardware cancel button can also be used.
