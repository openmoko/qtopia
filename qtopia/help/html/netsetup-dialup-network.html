Network Tab

Used to define internet services for the Qtopia device. To allow the server to determine the best settings select Auto (to allow Qtopia to detect the values) for each option.

If you cannot resolve names untick the name servers check-box and enter the IP addresses for your primary and secondary DNS (Domain Name Servers).

%%PHONE%%

Highlight tab and select an option. Press to tick/untick option.

Auto routing: routing decided by server.

Auto name servers: automatic DNS look-up.

Domain Name Servers only have to be defined if Auto name servers option is not selected.

First DNS: enter preferred DNS.

Second DNS: enter 2nd choice DNS.

@@PHONE@@ %%PDA%%

Tap Services to show this tab.

To add a new network service:

  • Tap the Add button to add a new network service using the Add Network Service dialog.

To change the settings of an existing service:

  • Tap the existing service, then Properties to change its settings using the Dial-Up or LAN dialog depending on the service.

To remove a service:

  • Tap an existing service, then tap Remove
