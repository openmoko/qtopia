Help Browser

You're using it! The Help Browser is a hypertext help system that works in a similar way to a web browser. Hypertext links appear in colored underlined text.

To get help within an application:

%%PDA%%
  • Tap on the title bar.

Some applications provide brief descriptions for different areas on the screen. Tap and hold in the title bar until What's this... is displayed, then tap an area of the screen.

@@PDA@@ %%PHONE%%
  • Press and select Help by highlighting (use ) it and pressing .

To quickly exit from help: press and hold until appears. @@PHONE@@
