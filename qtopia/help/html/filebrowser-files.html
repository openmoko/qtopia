Files Help

A number of files already exist on your device. Mostly these are the files that make your system and applications work.

Store your own files in the Documents folder to make locating and working with them easier. Once you have created a file using the Text Editor or another application it can be viewed in a text format or using the application it was created with.

%%PDA%%

To View the contents of a file:

  • Tap and hold the file to view.
  • Tap Text Format or the name of the application listed.

To make documents visible in the Documents screen:

  • Tap and hold the file/folder.
  • Tap Add to Documents.
@@PDA@@ %%PHONE%%

To View the contents of a file:

  • Select the file to view.
  • Select Text Format or the name of the application listed.

To make documents visible in the Documents screen:

  • Select the file/folder.
  • Select Add to Documents from the menu.
@@PHONE@@

Note: The document remains stored in it's original location, changes to the document from the Documents screen will affect the original file in it's original location.

Use caution when working with files as accidental deletion or renaming of a file may cause errors one or more of your applications.
