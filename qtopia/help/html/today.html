Today Help

The Today application provides a summary of events occurring today. This includes:

Calendar summary: displaying events occurring today.

  • Select an event to start the Calendar application.

Email summary: displaying a number of email headers.

  • Select a message to start the Email application.

Task summary: displaying all open tasks.

  • Select a task to start the Tasks application.
%%PDA%%

The information that each summary provides can be customized by tapping in the top left of the summary page. This shows the Edit Today View page.

@@PDA@@

Today will start automatically each day (at about midnight, or when you first turn the device on in the morning). This can be disabled by unchecking the Start automatically every day check-box on the Edit Today View page.
