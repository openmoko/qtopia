Music Help

The Music application is used to play music and other audio files.

%%PDA%%

To play an audio file tap the file.

The properties information of the file can be displayed by selecting the entry and tapping .

You can also construct playlists.

Details about the music controls and loading new files are also available.

@@PDA@@ %%PHONE%%

To play an audio file select the file. Any available properties about the file can be displayed by highlighting it and pressing and selecting Properties.

Press to access the menu and read about the options for playing files.
