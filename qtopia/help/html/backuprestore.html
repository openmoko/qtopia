Backups

To perform a backup:

  • Select the location you wish to backup to (e.g. SD or CF card).
  • Tap the Next button to continue.
  • Enter a series name in the Add new series box to store multiple unrelated backups on the one device, then tap Add.

OR

  • Simply use the default series.
  • Tap the Next button to continue.
  • A summary of the backup operation will be displayed.
  • Tap the Backup button to proceed.
  • Backup Successful will be displayed when the backup is complete.
  • Tap the Close button to exit the Backup application.

To restore from a backup:

  • Select the location of the backup.
  • Tap the Next button to continue.
  • Select the series name to restore from.
  • Select Restore from most recent backup to restore everything (useful if your device is seriously damaged).

OR

  • Select Restore specific files in, then the specific files you want restored (useful if you have accidentally broken a single component, but wish to keep other work).

Note:If you choose to restore specific components, you will be presented with a list of available backup dates for each component and will be able to choose which backup is to be restored. Tap the version to restore, then tap the button to return to the Restore Selection screen.

  • Tap the Next button to continue.
  • A summary of the restore operation will be displayed.
  • Tap the Restore button to proceed.
  • Restore Successful will be displayed when the restore is complete.
