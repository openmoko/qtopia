Alarm Settings

Alarm settings are used to specify if an alarm should be preset for newly created events and if yes, then how many minutes prior to an event's starting time should the alarm go off.

To activate Preset Alarms:

  • Select the Preset checkbox to place a tick in it.
  • Specify the number of minutes by $${SetMinutes}.
  • $${Tap} to save your changes or to cancel.
