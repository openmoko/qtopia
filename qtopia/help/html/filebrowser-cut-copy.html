Moving / Copying

Files and folders can be moved or copied from one location to another.

Use caution when moving files. Accidentally moving a system or application file may cause errors in one or more of your applications.

%%PHONE%%

To move or copy a file or folder:

  • Select the file or folder to move/copy.
  • Select Move from the menu to move the file to another folder.

OR

  • Select Copy to create a copy the file in another folder.
  • Navigate to the folder that you want to move/copy the file or folder to.
  • Press to place the file there.
