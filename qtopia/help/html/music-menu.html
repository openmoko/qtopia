Music Menu

Press to access the menu.

Properties: view the properties of a selected music file.

Move Up: press to shift the file up in the list as it appears on te screen.

Move Down: press to shift the file down in the list as it appears on te screen.

Select All: ticks all of the files. Use this to play all files in sequence. (Press to play the files one after the other.

Add to Category...: use this to add the highlighted file to a category.

View Category...: use this feature to limit the list of displayed music files to those in a certain category.
