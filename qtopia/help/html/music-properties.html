Music Properties

The properties of file are displayed in a scrolling banner across the top of the screen while the file is playing.

%%PHONE%%

To access the properties, highlight the file in the list (don't play it) and press then select Properties from the menu.

@@PHONE@@ %%PDA%%

To access the properties of a music file, press and hold the stylus on the file in the playlist.

@@PDA@@

The Properties screen has three tabs that provide detailed information about each file available from Music.

The information provided on the tabs:

  • General Tab: type, category, name and location.
  • ID3 Tags Tab: title, artist, album etc.
  • Details Tab: filename, path, format etc.
