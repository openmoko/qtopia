Video Properties - Details Tab

File: name of the file.

Path: directory path of file.

Size: of the video file in bytes.

Type: MIME type (read only).

Date: last modification date.

Audio: audio file format.

Rate: data (audio) rate in baud.

Chans: number of audio channels.

Freq: sampling frequency in Hz (related to quality).

Video: file format of video.

Rate: data (video) rate in baud. 0 means variable or unknown.

Width: of display in pixels.

Height: of display in pixels.

Length: playing time in minutes:seconds.
