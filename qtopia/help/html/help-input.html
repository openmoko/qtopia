Input Methods Help

%%PDA%%

Initially, an 'abc' input method icon appears at the bottom left of the screen.

  • Tap the triangle to the right of the input method icon to pop-up a list of input methods.
  • Tap the input method you prefer. The icon will change to reflect your choice.

Tapping the input method icon alternately pops-up the input area and hides the input area.

  • Handwriting
  • Keyboard
  • Pickboard
  • Unicode
@@PDA@@ %%TOUCH%%

Input is via handwriting. Simply write letters on the display, or use one of the special strokes for other input.

  • Handwriting
@@TOUCH@@ %%KEYPAD%%

Input is via the keypad. Press the key corresponding to the letter you wish to type (eg. press 3 for "D", "E", or "F").

Press 1 for punctuation.

Press 0 for a space.

Press-and-hold digits to enter the digit.

Press # to change between:

  • upper case, and
  • lower case.

Press-and-hold # to change input mode between:

  • your primary input language,
  • your other selected input languages,
  • character by character, and
  • international character by character.

Press * to choose between possible letters or words.

Press-and-hold * to select a special character. @@KEYPAD@@
