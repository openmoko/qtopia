Calendar Settings

Use the Calendar Settings to change the display start time for the calendar Day %%PDA%% and Week views @@PDA@@ and to set your alarm preferences for events.

%%PHONE%%

Day starts at: first blank time slot to display; events set for an earlier time will still show.

Alarm Settings: to set an alarm for every event entered, select this and set number of minutes for alarm to go off before every event. You can still set different alarms for individual events.

@@PHONE@@ %%PDA%%

To change the start time:

  • Tap View, then Settings...
  • Adjust the Start Time by tapping the up / down arrows beside the current start time.
  • Tap to save your changes or to cancel.
@@PDA@@
