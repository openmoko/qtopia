Editing Categories

Categories can only be edited from an application that uses categories. Default categories cannot be edited.

%%PHONE%%
  • Launch Documents by using and pressing .
  • Select a document and view it.
  • Press and select Properties.
  • Use to highlight the Cat field and press , then select the ... option.
  • When the list of available categories is displayed, use to highlight the category to be edited.
  • Press and select Edit.
  • Press to open the field, then change the name of the category.
  • Press to complete the field.
  • Use and to tick/untick the Global check-box.
  • Press to save and exit.
  • Unless your are going to use the new category immediately, press again and exit Documents.
@@PHONE@@ %%PDA%%
  • Launch Documents.
  • Select a document from the list and view it.
  • Tap File and select Properties.
  • Select the category field and choose the ... option.
  • Select the name of the category, then tap File and select Edit.
  • Change the name or global availability of the category.
  • Save and exit.
