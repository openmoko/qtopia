Dial-Up Help

The dial-up dialog has four tabs for providing the information to get a dial-up connection setup:

  • Account Tab
  • Modem Tab
  • Dialing Tab
  • Network Tab
  • Proxies Tab
%%PDA%%

Enter the values required for each tab then tap to add the service or to cancel.

@@PDA@@ %%PHONE%%

Enter the values required for each tab then press to add the service or to cancel.
