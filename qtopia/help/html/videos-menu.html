Videos Menu

%%PHONE%%

Press to access the menu.

Properties: view the properties of a selected video.

Move Up: select to shift the file up in the list as it appears on the screen.

Move Down: select to shift the file down in the list as it appears on the screen.

@@PHONE@@ %%PDA%%

Tap File to access the menu.

Properties: view the properties of a selected video.

Select All: ticks all of the files. Use this to play all files in sequence. (Press to play the files one after the other.

Add to Category...: use this to add the highlighted file to a category.

View Category...: use this feature to limit the list of displayed video files to those in a certain category.

%%PDA%%

Use the stylus to move press and drag a file to a new position in the list.
