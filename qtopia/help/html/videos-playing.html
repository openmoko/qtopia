Playing Videos Details

Videos Application is Running

If Videos is already running tap a file to select it and then use to begin playing the selected file. When the file finishes playing, Videos will play the next in the list of files. This allows you to play all video files in a category without having to create a playlist.

%%PDA%%

You can rearrange the order of the files by pressing on the file and moving it up or down the list with the stylus.

@@PDA@@

Video Application is not Running

You can use Documents to find and play a video:

  • Return to the main launcher view (pressing the hardware home button).
  • Continue to press the home button until the documents tab is the current tab, or by selecting the documents tab.
  • Find and select the video you wish to play in the list of documents.
  • The Videos application will launch and begin playing the file.
