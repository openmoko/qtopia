Synchronizing with a PC

  • Run the Qtopia Desktop application on your desktop PC.
  • Place the Qtopia device in its cradle.
  • Qtopia Desktop will automatically establish a connection to the Qtopia device.
  • Click the Sync All button on the Qtopia Desktop toolbar.

Finer control is available for synchronizing individual applications, and installing software.

See the Qtopia Desktop online help for more information.

You can restrict the synchronization access to the device by modifying or selecting the subnet in the Accept sync from network list. You will need to specify either the IP address of a single machine, or specify a network by entering an IP address with a slash (/) and the number of bits (e.g. 10.0.0.0/8 means all addresses from 10.0.0.0 to 10.255.255.255).
