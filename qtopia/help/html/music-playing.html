Playing Media Files

Music Application is Running

If Music is already running then select a file and then use to begin playing.

When the file finishes playing, Music will play the next in the list of files. This process allows you to play all music files in a specific category without having to create a playlist.

To play a mixture of music files in succession you have to create a Playlist.

You can rearrange the order of the files by pressing on the file and moving it up or down the list.

Music Application is not Running

You can use Documents to find and play a music track:

  • Return to the main launcher view (pressing the hardware home button).
  • Continue to press the home button until the Documents tab is the current tab, or by using the tab in the top right hand corner.
  • Find and select the media file you wish to play in the list of documents.
  • The Music application will launch and begin playing the file.
