Event Alarm

Event Alarms are used to remind you of upcoming events. They can be Silent (a message box will appear when the alarm is activated) or Audible (a bell will sound).

To set a reminder for an event:

  • Select either Silent Alarm or Audible Alarm from the alarm drop-down list.
  • Enter the number of minutes before the event that you want the alarm to be activated.
