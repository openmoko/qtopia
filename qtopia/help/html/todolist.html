Tasks

Tasks is used to create and maintain a list of prioritized tasks. You can add, edit, delete, view and beam tasks.

shows a check mark if the task has been completed.

shows task's priority (1 is highest priority).

Description: title of the task.

To view or edit a task: highlight and select a task, to edit choose Edit from the menu.

To add a task: choose New from the menu.

To quickly edit the priority or description of a task, select it.
