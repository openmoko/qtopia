Folders

Folders are used to store and organize the files on your device.

%%PHONE%%

To view the contents of a folder:

  • Select the folder.

To create a new folder:

  • Select New Folder from the menu.
  • The New Folder will be highlighted.
  • Enter an appropriate name for it using one of the Input Methods.
  • Press .
@@PHONE@@ %%PDA%%

To view the contents of a folder:

  • Tap the name of the folder.

To create a new folder:

  • Tap on the toolbar.
  • The New Folder will be highlighted.
  • Enter an appropriate name for it using one of the Input Methods.
  • Tap a blank area of the screen.
