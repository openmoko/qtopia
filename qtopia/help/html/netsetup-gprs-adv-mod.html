GPRS Advanced - Modem

Used to define internet services for your Qtopia device. Highlight and select to change.

  • AT-dial: the AT-dial dial code supplied by your ISP, e.g. ATDT.
  • Speed: at which modem should connect (influenced by IP).
  • Flow Control data transfer protocol. Recommended.
  • Wait Time acceptable delay for other end of connection to communicate. If data transfer to phone is delayed more than this time, then auto disconnect. Use the default 5 seconds, or increase the time if you have problems connecting.
  • Demand Dialing - select your preferred connection option and idle timeout.
