This image tool allows you to view, edit, delete, and beam pictures.

To view an image: tap the image. To return to the thumbnails press . Alternatively you could use the to move through the images.

To edit an image: select the image and tap the icon.

To delete an image: select the image and tap the icon.

To beam an image: select the image and tap the icon.

To edit image properties: select the image and tap the icon. For more on properties go to Picture Properties.

To view a slide show: select the first image for the slide show and tap the icon. For more on slide shows go to Picture Slide Show.
