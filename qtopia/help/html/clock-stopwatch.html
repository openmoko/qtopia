Stopwatch

Top display shows total time.

Number above Start/Stop shows current lap number.

Split shows end time for this lap, Lap shows duration of this lap.

%%PHONE%%

Press * button to start/stop timer.

Press # button to mark lap time, or to reset.

When stopped, use / to change lap number.

@@PHONE@@ %%PDA%%

To start the stopwatch:

  • Tap the Start button.
  • The Start button will be replaced with a Stop button.

To pause the stopwatch:

  • Tap the Stop button.

To restart the stopwatch:

  • Tap the Start button again.

To reset the stopwatch to zero:

  • Tap the Reset button

To record lap times:

  • Start the stopwatch.
  • Tap the Lap/Split button at the end of each lap.
  • The lap and split times will pause for a moment to record the details.
  • The lap number above the Stop button will be incremented to record the next lap.
  • At the end of the final lap tap the Stop button.

To review individual lap times:

  • Tap the up/down arrows to the left of the lap number.

Split time is the total time elapsed until the end of the current lap.

Lap time is the time recorded for the current lap.

The stopwatch can also be controlled by using a keypad (if available). The button is used to start and stop the stopwatch, and the Up and Down buttons can be used to view the previous and next lap times.
