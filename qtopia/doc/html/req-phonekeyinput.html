The Phone Key Input plugin allows the user to input text using the 0-9, # and * keys of a phone.

Features

Smart Activation

Phone Key Input is only active when an element that accepts text input has focus.

Input Hinting

Phone Key Input is able to take input hints from text input elements and use the appropriate word look-ups and key to character set mappings.

Word Recognition

Phone Key Input looks-up words based on the combination of characters for the keys entered. If more than one possible word matches the set of characters the plugin selects the one the user marked as preferred, then the one more common in the language selected and finally any remaining matched words.

Word Selection

The user is able to open a pop-up of all possible word matches to the set of characters for the keys they have pressed.

Next Word

The user is able change the recognized word to the next most likely word.

Dictionary Selection

The user is able to specify an alternate language to be used for word recognition.

Direct Character Input

The user is able to spell-out words that are not included in the set of recognized words for the selected language. Words entered this way consisting of more than two characters are added to the list of recognized words.

Symbol Input

The user is able to open a pop-up of additional symbols.

Dependencies

This application is dependent on the device being equipped with a keypad with the keys 0-9, # and *.
