World Time Specification

  • Features

    • World Map

    • Zooming

The World Time application allows the user to see what the current day and time is in other cities around the world. The day and time for up to six cities can be stored for easy look-up both in World Time and in applications accessing time zone information.

Features

World Map

A map of the world is displayed showing the current city.

Keypad Control

The user is able to select cities using the directional controls.

World Map

The user is able to select cities by clicking on the world map.

Reference Cities

The user is able to store up to six cities for easy look-up and use in other applications accessing time zone information.

Zooming

The user is able to zoom in the world map to aid in selecting cities.
