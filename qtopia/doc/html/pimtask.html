Detailed Description

The PimTask class holds the data of a todo entry.

This data includes the priority of the task, a description, an optional due date, and whether the task is completed or not.

See also Qtopia PIM Library.

Member Type Documentation

PimTask::PriorityValue

These enums describe the current priority() of the Task.

The values are:

  • PimTask::VeryHigh
  • PimTask::High
  • PimTask::Normal
  • PimTask::Low
  • PimTask::VeryLow

PimTask::TaskStatus

These enums describe the current status() of the Task.

The values are:

  • PimTask::InProgress
  • PimTask::Completed
  • PimTask::Waiting
  • PimTask::Deferred
