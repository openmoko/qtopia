Qtopia - 2.0.0 Release Notes

  • Introduction
  • Other New Features

Introduction

The Qtopia 2.0 release launches the Qtopia Phone Edition onto the commercial phone development market. In addition to enhancing existing features, the package now also offers the following phone-specific functions:

  • Call history
  • Call options including barring
  • Contacts linked to dialer
  • Call waiting
  • Dialing recently called/received numbers
  • Sending/receiving SMS

This release constitutes a broadening of the platform and extension of functionality that provides manufacturers of embedded linux phones with a flexible and powerful software suite.

Other New Features

  • System Information application.
  • SMS and Qtopia Mail integration: a single storage system for messages on the device; and a common message composition framework.
  • ImageViewer and Pictures integration.
