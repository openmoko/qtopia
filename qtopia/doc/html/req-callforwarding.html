Call Forwarding Specification

  • Features

The Call Forwarding application allows the user to set how the forwarding of calls should be handled under various circumstances.

Features

Call Forwarding allows incoming calls to be forwarded depending upon any of the following conditions:

  • Always forward all calls
  • Only forward calls when phone is busy
  • Only forward calls when call is unanswered
  • Only forward calls when signal is unavailable

For each of the above conditions it is possible to specify forwarding of calls according to the type of call:

  • Voice calls
  • Data calls
  • FAX calls
  • SMS calls

It is possible to forward a call to any number stored in Contacts.
