The Voice Notes application allows the user to record audio notes using an integrated microphone. In addition the user is able to browse through audio notes on their device, playback audio notes, delete audio notes, and adjust recording settings.

Features

Supported Formats

Voice Notes supports the following audio formats:

  • PCM WAV
  • GSM WAV
  • MMS AMR

Browse Audio Notes

The user is able to browse through audio notes on their device. The user is able to browse through all audio notes or only audio notes belonging to specific categories.

Record Audio Note

The user is able to record an audio note using the integrated microphone. The user is able to determine the duration and dynamic of the recording using a visualizer.

Playback Audio Note

The user is able to playback an audio note using the integrated sound device. The user is able to determine the duration and dynamic of the recording using a visualizer.

Delete Audio Note

The user is able to delete audio notes taken by Voice Notes.

Adjust Recording Settings

The user is able to adjust the settings of how audio notes are recorded. The user is able to toggle between mono and stereo recording, adjust the sample rate, and change the audio encoding method.

Dependency

Integrated Microphone

Voice Notes requires that the device be equipped with an integrated microphone in order for it to function as specified.
