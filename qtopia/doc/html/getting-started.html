Welcome to Qtopia.

This section provides information about how to download and install Qtopia products. It is assumed that it is already known which packages are going to be installed. For more information about available products please read the sections about Qtopia Products and Qtopia Packages.

Please contact sales@trolltech.com with any questions regarding the Qtopia product to best meet your requirements.

Downloading and Installing a Package

How to Obtain Qtopia Products

After becoming a Qtopia customer, Trolltech will create an FTP directory for each customer account. The user can log in to that FTP directory using any browser and download packages. The FTP directory will be available for the full duration of the contract, and will be updated regularly with new releases.

An email will be sent from sales@trolltech.com once you have acquired a Commercial Qtopia license or been granted an Evaluation Qtopia license. The email will state the following details:

  • Customer ID
  • License ID
  • Qtopia package Download Location.

Each new release will be announced to the e-mail address that is associated with the customer account.

For information on language-specific packages refer to: Language Packages.

Package Naming Conventions

A Trolltech FTP download directory usually contains multiple packages and it may, at first, be unclear which packages are required. Depending on the license type one or more of the following packages can be found:

Package Description
qtopia-[edition]-src-[x.y.z].[zip] The OEM source package. This package is required to build Qtopia from source. For Qtopia 2.2.0 and later, this package contains all source required to build Qtopia, including the prerequisite source for Qt and tmake.
qtopia-[edition]-sdk-[x.y.z]-[distro].i386.rpm The SDK package containing a combination of pre-built binaries and library header files.
qtopia-[edition]-sdk-devel-[x.y.z]-[distro].i386.rpm An SDK extension package containing Qtopia library source files.
qtopia-desktop-[x.y.z]-[distro].[exe] These packages all are pre-built binary packages that can be used to install a working version of Qtopia Desktop on a Linux, Windows or Mac desktop machine. Qtopia Desktop is an add-on tool that provides PIM like functionality on the desktop and can be used to synchonize files, events, todo-tasks and contact information between the desktop machine and a Qtopia (handheld) device. Installing Qtopia Desktop is optional, that is, it is not a prerequisite to start developing Qtopia or Qtopia-based applications.

In the above list the following tags have been used:
Tag Description
[edition] Qtopia product name, one of: core, pda, phone or refboard.
[x.y.z] Product version number, for example 2.2.0
[zip] indicates that a package is made available as both a "tar.gz" and as a "tar.bz2".
[distro] Operating system, for example, redhat9, suse82, suse9, macosx, win, etc.
[exe] Package type, one of: .rpm, .dmg, .tar.gz or .exe.

At first sight the vast number of possibilities may appear to be a bit daunting, but in practice it usually means the FTP site will provide either a qtopia-source package or a qtopia-sdk package, and a qtopia-desktop package.

Depending on the purchased licenses the FTP site provides access to many more files such as various Qt versions, snapshots of packages, language packages and much more. The rest of this section refers only to the Qtopia specific files available on the FTP site.
