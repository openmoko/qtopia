Qtopia - 2.1.2 Release Notes

  • Introduction
  • Bug fixes

Introduction

The Qtopia 2.1.2 release is a maintenance release and includes the Qtopia Core, PDA and Phone Editions.

Bug fixes

The following bugs have been fixed in Qtopia 2.1.2:

  • [Language] Change Qtopia's default font when a new language is loaded.
  • [Language] Much improved language support.
  • [Themeing] Significantly improve support for themeing by ensuring that all used pixmaps are available to be edited without need to change source code.
  • [configure] Enable automatic language detection for source packages.
  • [qtopia] Fixed/improved layout in some dialogs.
  • [qtopia] avoid warnings when an img is null.
  • [qtopia] Enable space and OK in non-phone mode for selecting an event.
  • [qtopia] Allow pseudo semi-transparent backgrounds in all applications.
  • [qtopia] fixed a segfault when $QPEDIR is not set correctly.
  • [qtopia] Avoid crashing when clicking on an empty menu.
  • [qtopia] Show a wait icon while network access is being performed.
  • [Calculator] Fix calculation of (-27) root 3.
  • {Calculator] Fixed tan operator when using Gradians measure.
  • [Calculator] Prevent a crash when changing to fraction plug-in.
  • [Calculator] Fix a font problem for languages like Chinese and Japanese.
  • [Calculator] Correct division by zero behavior for fraction and integer mode.
  • [Calculator] Add missing desktop files for calculator plug-ins.
  • [Calculator] Remove the limits of integer data type. Fraction calculations use longlong type.
  • [Calculator] Converts the data type of the memory to the data type of the engine.
  • [Calculator] Fixes a crash in the fraction plug-in after "out of range error".
  • [BatteryMonitor] Improved BatteryMonitor status.
  • [Messages] Speed improvement (~20%) by making ImapProtocol::parseFetch more asynchronous.
  • [Messages] Fixed a crash when having a completely empty email list.
  • [Messages] Delete all mails retrieved from an account when deleting the account and delete the info for this account in the settings file.
  • [Messages] Speed up retrieval of large messages by an order of magnitude (10x).
  • [Contacts] Contacts dropped any names after a third name; they are now added on as part of the last name.
  • [Contacts] Support the sending of vCard via SMS from Contacts.
  • [Contacts] Modified the contact editing dialog to delete a contact if all details are removed.
  • [Contacts] Stopped Phone Edition's address book from destroying PDA Edition's information display settings.
  • [Call History] Get day of month correct in call history.
  • [Call OPtions] Make it easier to add new/fewer sections to the Call Options setting application.
  • [Themeing] Add name attribute to background tag in theme.
  • [Callscreen] Sync callscreen and homescreen background: Previously callscreen had no background image.
  • [performance] Optimized performance by more efficient scaling of images.
  • [performance] Optimize performance of fileinfolist.
  • [performance] Optimize performance by not scaling the keypad images.
  • [performance] Improve performance by making qcop startup quicker (disabling/delaying GUI features).
  • [performance] Optimised translation loading.
  • [Sound] Extend support for CUSTOM_SOUND_ALARM function to multiple target devices.
  • [Sound] Allow setting profile volume to also affect volume of other non-priority sounds.
  • [Phone] Add "+EXT ERROR" as one of the recognised "ERROR" responses,
  • [Phone] Improve suppport for supplementary services.
  • [Phone] Add user interface support for the preferred operator list.
  • [Phone] Parse numeric format operator names from AT+CPOL correctly.
  • [Phone] Fix: backspace implicitly deleted an existing operator.
  • [Phone] Added functionality to convert numeric preferred operator identifiers into real operator names.
  • [Phone] Improved support for setPassword and clearPassword functions in combination with SIM PIN2.
  • [Phone] Added query/modify types for AT+CPOL, AT+CPLS, and AT+COPN.
  • [Phone] Hook up the speed dial "dial voice number" feature to the AT+CSVM command in the phone settings app and the server.
  • [Phone] Add missing query/modify/notification types for advice of charge, price per unit, accumulated call meter, etc.
  • [Phone] Add support for unsolicited advice of charge notifications, and better explain how "AdviceOfCharge" should be used.
  • [Phone] Added new options for CLCK and CPWD from SM 27.007.
  • [Phone] Added a vendor-specific facility to modify the GPRS AT commands when a data connection is started.
  • [Phone] Added extra commands to initialize GPRS sessions.
  • [Phone] Send "AT+CMMS=1" before transmitting multiple SMS messages to keep the radio link open for faster transmission.
  • [Phone] Repeat the AT+CNMI command up to 5 times, because some modems don't properly execute it just after AT+CPIN due to "SIM busy".
  • [Phone] Add a vendor option for empty field in the AT+CPBW command.
  • [Phone] Use the standard 27.007 AT+CSIL command instead of the Ericsson-specific AT*ESIL command.
  • [Phone] Fix up "decodeSimCommand" and "encodeSimResponse" to match the GSM 11.14 specification better.
  • [Phone] Reordered execution of AT%SATC, AT%CPI and AT+CFUN.
  • [Phone] No longer strip the '+' from international numbers in formatForwarding.
  • [Phone] Added a "SMSFull" notification, based on CIND notifications.
  • [Phone] Add support for +CIEV notifications of indicator changes, especially signal quality and battery charge.
  • [Phone] Add support for CSMI multiplexing.
  • [Phone] Use the AT+CSVM command to get and set the voice mail number.
  • [Phone] Add support for SS and USSD notifications.
  • [Phone] Added routines to encode and decode raw SIM toolkit packets, where the modem doesn't provide any decoding support of its own.
  • [Phone] Use SMS_Address_Unknown instead of SMS_Address_National for non-'+' addresses, because some networks are picky.
  • [Phone] Added phonevendor plug-in support
  • [Phone] Fixed a crash after pressing p or w at the call screen.
  • [Phone] Fix a crash when user removes last broadcast channel.
  • [libqpe] Fix a font problem in PhoneDecoration where the window title wasn't properly shown in Chinese due to static use of Helvetica.
  • [libqpe] Comments added to TimeStringFormat in order to give the translator a better understanding what he's translating.
  • [Qpe-server] Enforce translation of previously untranslated default button names.
  • [Qpe-server] If the BatteryMonitor is receiving the battery status via the modem this fix will ensure that the battery widget changes to charging mode when the modem is powered by an external power source. TimerEvents will be checked for their validity.
  • [Qpe-server] Avoids that the server kills an application that has been terminated by lazy application shutdown (this could happen if the server is in critical memory condition)
  • [Plug-in Manager] Load the correct name and comment entry in desktop files for plug-ins.
  • [libqtopia1] Fixes the missing translation for name of the month in Datepicker. This problem is not obvious in languages which use the latin alphabet.
  • [libqtopia2] Optimizes the use of available space for speeddial dialog.
  • [Snake] Fix incorrect font usage which prevents Asian languages from being properly shown.
  • [QAsteriods] Fix incorrect font usage which prevents Asian languages from being properly shown.
  • [Parashoot] Fix incorrect font usage which prevents Asian languages from being properly shown.
  • [Solitaire] Fix incorrect font usage which prevents Asian languages from being properly shown.
  • [Worldtime] Fix a font problem and clean up translation code.
  • [Today] Convert old-style translation in desktop files to new style. Now the Plug-in Manager shows the correct (and translated) description for all today plug-ins.
  • [Applets] Convert old-style translation in desktop files to new style. Now the Plug-in Manager shows the correct (and translated) description for all applets.
  • [Inputmethod plug-ins] Convert old-style translation of desktop files to new style. Now the Plug-in Manager shows the correct (and translated) description for all inputmethod plug-ins.
  • [PhoneProfile-config file] String change that prevented that Outdoor was not translated in ringprofiles application.
  • [FirstUse] Fixes the incorrect display of translated time zone names.
  • [FirstUse] Reload default font for each language (as stated in FontMap).
