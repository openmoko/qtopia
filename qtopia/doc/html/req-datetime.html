The Date/Time application allows the user to set current time, date and time zone.

Features

Date and Time Adjustment

The user can modify the current date and time.

Time Zone Adjustment

The user can set the current time zone.

Date and Time Format

The user is able to set the time to be displayed as either 12 or 24 hour time. The user can also set the starting day of the week to either Sunday or Monday.

Date and Time Format

The Date/Time application notifies other applications of any changes to the current date and time.
