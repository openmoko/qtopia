The Appearance settings application allows the user to select a color scheme, theme, font and background image.

Features

Select Color Scheme

The user is able to choose a color scheme. A color scheme defines what colors will be used for visuals such as text and background areas. For example, a color scheme may be comprised of colors that are shades of green.

Select the Theme

The user is able to select a theme to define the arrangement and appearance of the homescreen, title bar, context bar and window decorations. For more information on theming see Qtopia Themeing.

Select the Font

The user is able to select a font style and size for drawing.

Select the Background Image

The user is able to choose a background for sections in the launcher suchs as the home screen and application list. It is possible to choose from both images and wallpapers when selecting a new background.

It is possible to select the backgrounds from those supplied with the application or from the user's own store of images. Any image that is chosen is scaled to fit the window size whilst maintaining its aspect ratio.
