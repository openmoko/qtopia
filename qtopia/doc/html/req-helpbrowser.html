The Help Browser application allows the user view help pages. The user is able to browse through system help pages or application specific help pages. The user is able to view help pages in rich text and navigate using hyperlinks. The user is able to jump backwards and forwards through the browser page history.

Features

Context Sensitivity

The user is able to view pages specific to the current context by launching Help Browser from the system menu.

Rich Text

The user is able to view pages written using rich text elements. Support for rich text is provided by the 

QStyleSheet
class. See the QStyleSheet reference for more detail.

Hyperlinks

The user is able to navigate through help pages by following hyperlinks. The navigation path of the user is saved in the browser page history.

Step Back

The user is able to step back one page in the browser page history.

Step Forward

The user is able to step forward one page in the browser page history.

Home Page

The user is able to jump directly to the system help home page.
