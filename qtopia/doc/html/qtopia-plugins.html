Plug-ins

Qtopia Plug-ins can be categorized as:

  1. Application Plug-ins
  2. Server Plug-ins
  3. System Plug-ins

Plug-ins contained in each of these categories are defined below:

Application Plug-ins

The following plug-ins are used by applications:

  • Media Player (Music/Video)
  • Media Recorder
  • Today
  • Qtopia Desktop

Server Plug-ins

The following plug-ins will be loaded by the Qtopia server on bootup and restart:

  • Input methods
  • Network Services
  • Taskbar Applets

System Plug-ins

The following plug-ins are loaded at application startup:

  • Font Factories
  • Image Codecs
  • Text Codecs
  • Widget Styles
  • Window Frame Styles

See Writing Qtopia Plug-ins for information on how to write a plug-in.
