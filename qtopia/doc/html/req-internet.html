Internet Settings Specification

  • Features
    • GPRS
    • WAP

The Internet settings application allows the user to configure the network connections for their device.

Features

GPRS

It is possible to configure the following GPRS parameters:

  • Username
  • Password
  • APN

WAP

WAP settings are associated with each GPRS connection. The following WAP sever settings are supported:

  • Type:
    • IPV4
    • IPV6
    • E164
    • Alpha
  • Port
  • Proxy address
  • Login Type:
    • Basic
    • Digest
  • Login Name
  • Login Password
  • Connection Mode:
    • Connectionless (CL)
    • Connection Orientated (CO)
    • Secure CL
    • Secure CO
    • WTA secure CL
    • WTA secure CO
    • OTA HTTP
  • Default WAP page
  • MMS server URL
