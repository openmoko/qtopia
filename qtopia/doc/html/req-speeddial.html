The Speed Dial application allows users to associate applications, contacts and other favourite operations with a particular number which can then be run / accessed by entering the associated speed dial number. It provides the user with a way to create shortcuts to frequently used functions and records.

Features

Associations

The user is able to associate an application, phone number, contact, as well as other special functions such as viewing the current day's calendar or changing the ring tone with a number. It is possible to create speed dial associations for the following:

  • Dialing a contact's phone number
  • Running an application
  • Viewing Today's calendar
  • Creating a new event
  • Beaming a business card
  • Calling Voice mail
  • Composing an email
  • Composing an SMS
  • Viewing the most recent SMS
  • Viewing the SMS list
  • Creating a new task
  • Setting plain mode

Removing Association

The user is able to remove a speed dial association.

Replacing Association

The user is able to replace an existing speed dial association with another speed dial association.

Speed Dialing

The user is able to run the associated speed dial action from the home screen by simply pressing and holding the associated number(s) for the action.

Speed Dialing (fullscreen input method)

The user is able to run an associated speed dial action from the home screen by writing the associated number(s) or character(s) on the screen.

Dependencies

Contacts

Speed Dial uses services provided by Contacts for:

  • Dialing a contact's phone number
  • Beaming a business card

Calendar

Speed Dial uses services provided by Calendar for:

  • Viewing Today's calendar
  • Creating a new event

Messages

Speed Dial uses services provided by Messages for:

  • Composing an email
  • Composing an SMS
  • Viewing the most recent SMS
  • Viewing the SMS list

Tasks

Speed Dial uses services provided by Tasks for:

  • Creating a new task

Dialer

Speed Dial uses service provided by the Dialer for:

  • Calling voice-mail
