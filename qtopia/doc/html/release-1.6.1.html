Qtopia - 1.6.1 Release Notes

Qtopia 1.6.1 was the first maintenance release for the Qtopia 1.6 series. Since it is a maintenance release, all changes are bug fixes for existing functionality. No new features have been added.

The total number of bugs fixed between 1.6.0 and 1.6.1 is 51. The bug fix table is shown below.
