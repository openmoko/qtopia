Ring Profiles Specification

The Ring Profiles setting application allows the user to personalize the behavior and sounds of the phone for incoming calls and messages. The range of ring profile features that can be activated includes volume, tone, and vibrate responses. An airplane safe mode is also available and it is possible to create new profiles.

Features

Activate Ring Profile

The user is able to set a ring profile as the currently active profile for the device.

Edit Ring Profile

The user is able to edit the selected ring profile. The settings for a ring profile include Volume, Vibrate Alert and Auto answer. The user is able to customize the Call and Message alert behaviors and melodies. The user is able to set Alert behavior to be Off, Once, Continuous, and Ascending. Ascending starts from the set volume and rises in volume to the maximum volume setting. It is possible to choose melodies from uncompressed PCM audio data files either provided with the device or from the user's document files.

Add/Remove Ring Profiles

The user is able to add or remove additional ring profiles. The user is not able to remove the original system ring profiles.

Add to Speed Dial

The user is able to add a ring profile to Speed Dial to enable fast activation of the profile from the home screen of the phone.
