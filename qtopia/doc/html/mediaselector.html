Detailed Description

This class is specific to the mediaplayer apps.

It is currently private to those apps, and its API, and therefore binary compatibilty may change in later versions.

This class lists MediaSelectorItems. Media items could be listed by various media specific attributes, id3 tags or other criteria. Phone version only lists the document name due to space restrictions.
