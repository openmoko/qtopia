Power Management is a settings application. It allows the user to set up the device's display and change settings to regulate power use.

Features

The features of the Power Management settings application allow the user to configure the device to behave differently under different usage conditions and to suit individual tastes. The configuration extends only to the display screen.

Automatic Behaviour When phone is Not in Use

The user is able to configure a number of settings to take effect when the device is not in use.

Set Display Lighting to Automatically Change

The user may enable features to dim or turn off the display lighting when the device has not been used for a time. If enabled each feature is further configurable by changing a timer that controls the delay before activation. The timer is specified in increments of 15 seconds from 10 seconds up to 3600 seconds (an hour) by using the up and down arrows or by typing in an exact figure in seconds.

The timers commence counting as soon as the user ceases interacting with the device (by pressing keys or using the stylus). There is no visible feedback that the timers are running. Any interaction with the device restarts the timer.

Automatic display lighting features are:

  • Dimming of the display lighting: after a user specified amount of time.
  • Turning display lighting off: after a user specified amount of time.
  • It is possible for the user to enable / disable the above.

Set Display to Automatically Return to the Home Screen

With this feature the device is configured to quit applications and display the home screen when the device is not in use. The home screen is the blank desktop display of the device that appears when the device is powered on. The feature is controlled by a timer in a similar fashion to the display lighting in the previous section.

If an application such as Notes is left open, editing a note, the user is not prompted to save the note: the note application simply quits. For this reason care needs to be used with this setting to avoid data loss if the device is to be used for creating documents.

The feature allows the user to:

  • Set the amount of time to wait before returning to the home screen.
  • Turn this feature off or on.

Display brightness

The user is able to control the display backlighting to suit the brightness of their surroundings and their individual tastes. Higher settings will use more power.

The feature allows the user to:

  • Adjust the brightness of the display.
