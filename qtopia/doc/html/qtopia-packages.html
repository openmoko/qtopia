Introduction

Qtopia is available in a number of different packages. A package is a combination of Qtopia plus other products such as Qt and each package targets a different user group. Depending on the package type the deliverable may be a full source code product or a subset of the source code plus pre-built Qtopia binaries.

Commercial Packages

Qtopia OEM Kit

The Qtopia OEM Kit targets companies developing mobile devices running Qtopia. It is required for installation and distribution of Qtopia on consumer electronic devices and allows the customer to develop Qtopia applications as well as to adapt Qtopia to fit on a device. The Qtopia OEM Kit comprises a number of Qtopia developer licenses and the full Qtopia source code.

To install and distribute Qtopia on commercial devices both the Qtopia OEM Kit and Qtopia Runtime licenses are required.

Qtopia Software Development Kit (SDK) Packages

There are two package options for developing commercial Qtopia applications:

  1. Qtopia SDK Light
  2. Qtopia SDK Professional

Both packages provide pre-built binaries of the Qtopia product and header files. The professional product also includes parts of the source code and a support contract. The Qtopia binaries cannot be re-built with these packages.

Qtopia SDK Light

Qtopia SDK Light targets small companies involved in commercial Qtopia application development not requiring support or maintenance. It allows customers to develop Qtopia applications using Qt/Embedded and the Qtopia API.

The Light SDK package is provided in binary format and includes:

  • pre-built binaries for i386 and ARM
  • header files to allow customers to build and link Qtopia-based applications
  • documentation
  • tools required for the production of Qtopia-based applications.

Qtopia SDK Professional

Qtopia SDK Professional targets mid-sized companies involved in commercial Qtopia application development requiring support and maintenance. It allows customers to develop Qtopia applications using Qt/Embedded and the Qtopia API.

The Professional SDK package is provided in binary format and includes all of Qtopia SDK Light together with:

  • pre-built binaries for i386 and ARM
  • header files to allow customers to build and link Qtopia-based applications
  • Qtopia source code (.cpp) files.
  • documentation
  • tools required for the production of Qtopia-based applications.
  • support and maintenance

Free Packages

Qtopia PDA GPL edition

Qtopia is available in a free (GPL) Licensed version. Currently, this version is based on Qtopia PDA and is intended to support the Open Source community.

For further information about the GPL licensing please read the Qtopia Licensing documentation.
