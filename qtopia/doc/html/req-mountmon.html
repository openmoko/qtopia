Mounted Device Monitor Specification

    • Dependencies

This applet displays the currently mounted storage devices. The applet displays a task bar icon that allows external media to be cleanly unmounted from the Linux system, making possible to reduce the chance of damaging data on the card. It has the following features:

Feature \Description
Supported Storage Media The mounted device monitor detects the following media types:
  • Compact Flash cards (CF)
  • Secure Digital cards (SD)
  • Personal Computer Memory Card International Association Cards (PCMICA)
Automatic Media Detection Icon indicating when a newly inserted card has been correctly mounted and is ready for use.
Ejecting Media Eject option that can be used to safely remove an inserted storage media.

Dependencies

In order for this applet to work hardware connections for the external media are required and there are limits to the number of CF or PCMCIA slots that can be supported.
