Introduction

The growing demand for portable personal digital media and information environments is driving the convergence of computing and communications technologies and influencing the emergence of a new class of devices. The convergence of devices will continue to blur the distinction between mobile phones and personal digital assistants making it increasingly important for manufacturers to be using device-independent platforms such as Qtopia.

Qtopia is Trolltech's comprehensive application platform for embedded Linux based PDAs, mobile phones, web pads, and other electronic devices. The Qtopia product line comprises a number of editions designed to provide and create graphical user-interfaces for Linux-based devices.

Qtopia is both a platform and turn-key product offering unprecedented flexibility and a wealth of choices for manufacturers. It is the Trolltech solution to straightforward, compact, and versatile computing for consumer electronic devices.

The Trolltech Qtopia product line supports the next generation of wireless 3G devices that combine voice and Internet capabilities into wireless handsets. The third generation of cellular technology provides high performance, low-power processors that offer consumers the perfect way to stay in touch with the digital world while staying mobile.

Licensing Schemes

Trolltech offers two categories of licensing for Qtopia:

  1. Commercial Version
  2. Free (GPL) Version

The commercial versions of Qtopia are intended for the creation of commercial, proprietary, and non-free software and are available with a number of licences and license agreements to suit most customers. Please refer to Qtopia Commercial Editions for details.

The free version of Qtopia is intended to support and encourage the development of free software that will be released using a recognized open source license. Please note that free in the context of software refers to freedom as opposed to price. For more details about free software please go to http://www.fsfeurope.org/documents/whyfs.en.html or to learn more about Trolltech's free editions of Qtopia please refer to the Qtopia Free Editions page.

For further information about licenses please refer to the Qtopia Licenses.

Qtopia Editions

Features

Qtopia is a windowing system with a range of PIM and messaging applications complete with a desktop synchronization framework. The two Qtopia editions:

  1. Qtopia Phone
  2. Qtopia PDA

include a range of standard utilities and games, and are capable of multi-language and wireless support.

As a system, Qtopia, offers the following features:

  • Development environment
  • Games and multimedia
  • Input methods
  • Internet applications
  • Java integration
  • Localization support
  • Personalization options
  • PIM applications
  • Productivity applications
  • Synchronization framework
  • Windowing system
  • Wireless support

Qtopia Phone Edition

Qtopia Phone Edition for embedded Linux is designed for smartphones and feature-phones. It offers a comprehensive range of personal, messaging, and entertainment tools and combines them with the now standard phone functions.

Qtopia PDA Edition

Qtopia PDA Edition is designed for Linux OS-based PDAs (Personal Digital Assistants). Offering most of the same personal and entertainment tools as the Phone Edition, the PDA Edition expands the range of productivity tools and takes advantage of the enhanced functionalities of the device.

Qtopia Desktop

Qtopia Desktop is a companion package that allows the user to synchronize their data. It contains versions of the PIM application found in Qtopia and can be extended by third parties.
