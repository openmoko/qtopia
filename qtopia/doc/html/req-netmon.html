Dial-up Monitor Specification

  • Dependencies
    • Demand Dialing

The Network Monitor displays the available services to the user, allows services to be initiated, prompts for passwords if required, and shows the status of networking when relevant. It has the following features:

Features Description
Choosing Services The Network Monitor allows for multiple network services to be used.
Passwords For connections that require a password the user is prompted to enter the code after which the connection is completed.
Current Connection Status The user is presented with connection status information when a service is connected to a network.
Network Plug-ins It is possible to use the network plug-ins coupled with configuration parameters to provide an extendible way to add new software when new hardware is supplied.

Dependencies

Demand Dialing

Demand dialing requires that the PPP service can support the "demand" option.
