Detailed Description

The DocPropertiesWidget class provides controls for modifying an AppLnks properties.

The DocPropertiesWidget and allows modification of the name, location, and category of a file associated with a particular AppLnk. The file can also be deleted, copied and beamed from the DocPropertiesWidget.

First availability: Qtopia 1.6

See also DocPropertiesDialog and Qtopia Classes.
