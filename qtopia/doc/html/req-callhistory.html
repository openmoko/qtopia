Call History provides a record of calls that have been dialed, received, and missed on the phone. A single Call History record represents a single phone call and contains the various attributes of the call. The call record may be further associated with a contact through its phone number in which case the contact information is utilized when displaying the record. Call History allows up to 50 call records.

Features

Display Separate Dialed, Received and Missed Call Lists

Call History is able to display dialed, received and missed call lists separately. Each item in the list shows the following partial information of the call.

  • If the call record is associated with a contact, then the contact name, phone number type and photo otherwise the literal number of the call.
  • The type of phone call (either dialed, received, or missed).
  • The week day and time the call originated.

Display Detailed Call Information

The general dialed, received and missed call lists provide access to a detailed view of the full information of a call record. The following details of the call record are displayed:

  • If the call record is associated with a contact, the contact name, phone number type and photo, otherwise the literal number of the call.
  • The week day, day, month and time the call originated.
  • For calls that resulted in a connection, the duration and the week day, day, month and time the call was terminated.
  • An indicator for dialed calls which never resulted in a connection.
  • Related calls information (see next section).

Display Related Calls

Call records are related if one of the following conditions are met:

  • The calls are either dialed, received or missed calls to/from the same number.
  • The calls are either dialed, received or missed calls to/from the same contact.

The detailed view displays up to 10 related records. The related calls are the closest calls in terms of the time they originated, and are shown in ascending order based on this time. The information shown is the same as that shown in the lists.

Managing Call History

Call History allows the deletion of records so the user has the ability to keep the information as relevant as possible.

When viewing the general call lists, the user is able to clear just the visible list, as well as having the option to clear all lists.

When viewing detailed information, the user has the option to delete the call record being viewed.

Contact Creation

Call History can be a source of new contacts, since the user might often dial or receive phone calls to/from numbers that have no associated contact. Call History provides the option to save the phone number of call record not associated with a contact to an existing contact, or create a new contact based on the call record.

Other Application Integration

For call records associated with a contact, Call History provides the access to view that contact when viewing the call record in one of the lists or the detailed view.

For all call records not associated with a contact, or call records associated with a contact that have a mobile phone number type, the Call History allows the user to create a Message using the phone number of the record.

Dependencies

Contacts

Call History associates a call record with a contact where possible. Call History provides the ability to create new contacts from call records that have no associated contact.

Messages

Call History allows the creation of Messages based on call records.
