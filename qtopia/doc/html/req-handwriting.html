Handwriting Settings Specification

The Handwriting settings application allows the user to modify the way characters are drawn for handwriting recognition.

On touchscreen devices Qtopia provides fullscreen handwriting recognition that can be personalized through the device settings. The Handwriting Settings application also offers a comprehensive guide to drawing letters, numbers and other characters.

Special strokes are provided for elements such as next word, space, backspace, uppercase, new line, and punctuation.

It is also possible to use the scrollbar or other widgets instead of entering text.

Features

Modify Timing for Character Recognition

The user is able to:

  • Increase or decrease time allowed between different strokes of a multi-stroke character.
  • Increase or decrease time the stylus must be held still at the start of a stroke for the handwriting recognition not to attempt recognizing the stroke as a character.

Modify Characters

The user is able to modify the example strokes for a character or add additional examples of the character.

Add New Characters

The user is able to add new characters to the set of recognized characters. New characters are restricted to those available in the current font, plus additional non-printing and control characters.

Dependencies

Touchscreen

Handwriting settings requires that the device has a touchscreen enabled.
