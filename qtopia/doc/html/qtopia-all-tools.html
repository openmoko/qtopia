An Overview of Qtopia Related Tools

There are several tools available to assist in the development of Qtopia software. The following lists and briefly describes the command line and graphical tools available to speed the development process.

Tool Description
makeqpf Used to save QPF font files by rendering and saving fonts.
qmake A makefile tool to create Makefiles from simple platform-independent project files. Used with Qtopia 1.9 (limited beta release) onwards.
tmake An older version of qmake used with Qtopia 1.6 and 1.7.
Phone Simulator A graphical desktop method of testing and viewing phone software.
QVFB A utility that allows Qt/Embedded programs to be developed on a desktop machine.
Designer Manual Guide to Qt Designer.
Linguist Manual Guide to the Qt Translation Tools.
Help Browser Preprocessor An extension to Help Browser that enables the maintenance of a single source of html help files from which device-specific help can be extracted.

Additional tools are available for Qt and Qt/Embedded for example uic and moc.

Note: Designer for Qt/X11-2.3.2 must be used to modify .ui files for Qtopia 2.x
