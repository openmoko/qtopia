Die Funktion von Kategorien

Die Verwendung von Kategorien ist sinnvoll, wenn Sie eine große Anzahl von Elementen (z. B. Kontakte oder Aufgaben) verwalten müssen. Mit Hilfe von Kategorien können Sie die Anzeige auf die Elemente beschränken, die einer bestimmten Kategorie angehören.

Standardmäßig werden alle Elemente unabhängig von ihrer Kategorie angezeigt.

Das Dialogfeld "Kategorien bearbeiten" ist in allen Anwendungen gleich. Sie können damit Kategorien erstellen und diese Elementen zuweisen.
