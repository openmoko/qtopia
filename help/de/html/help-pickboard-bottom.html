Die untere Pickboard-Zeile

Die Eingabe der Wörter erfolgt durch Antippen der Buchstabengruppen. Das Pickboard zeigt am oberen Rand des Eingabebereichs eine Liste mit Buchstaben und Wörtern an. Ist das gewünschte Wort in der Liste enthalten, können Sie es durch Antippen einfügen.

Wenn Sie beispielsweise das Wort "Hotel" eingeben möchten, tippen Sie nacheinander auf die Buchstabengruppe GHI (für das "h"), die Gruppe MNO (für das "o"), die Gruppe STU (für das "t") und schließlich auf die Gruppe DEF (für das "e"). In der Liste wird nun auch das Wort "Hotel" angezeigt, das Sie nur noch antippen müssen, um es einzufügen.
