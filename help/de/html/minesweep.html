Minenjagd

Markieren Sie möglichst schnell alle Minenfelder.

Tippen Sie auf ein Feld und halten Sie den Stift auf dem Feld gedrückt, um es zu markieren bzw. die Markierung zu entfernen. Auf markierten Feldern wird eine Flagge angezeigt. Sie können Felder freilegen, indem Sie darauf tippen. In freigelegten Feldern wird die Anzahl der Minen in den benachbarten Feldern angezeigt.

Tippen Sie auf das Menü Spiel, um den Anfängermodus, Fortgeschrittenenmodus oder Expertenmodus auszuwählen. Durch Tippen auf den gelben Smiley können Sie ein neues Spiel beginnen.
