Der Modus "Fläche"

Geben Sie eine Einheit gefolgt von einer Zahl ein. Tippen Sie beispielsweise auf Sq m und dann auf die Zahl 3, um den Wert 3 m^2 einzugeben. Sie können diesen Wert dann konvertieren, indem Sie die gewünschte Einheit antippen (durch Antippen von Sq yd erhalten Sie beispielsweise 3.5...yd^2).

Die Schaltfläche für die aktuelle Maßeinheit wird vertieft dargestellt. So können Sie schnell erkennen, welche Einheit gerade verwendet wird.
