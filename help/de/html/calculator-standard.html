Der Modus "Standard"

Wenn Sie eine Zahl potenzieren möchten, geben Sie die Zahl ein, tippen Sie auf die Schaltfläche , tippen Sie auf die Potenzzahl und anschließend auf =.

Um die Wurzel einer Zahl zu berechnen, geben Sie die Zahl ein, tippen Sie auf die Schaltfläche , tippen Sie auf die Wurzelzahl (z. B. die Zahl 2 für die Quadratwurzel) und anschließend auf =.

Wenn Sie auf die Schaltfläche +/- tippen, wird das Vorzeichen der Zahl im Anzeigefeld geändert.

Mit den weiteren Schaltflächen werden die dazugehörigen Funktionen auf die Zahl im Anzeigefeld angewendet. Mit der Schaltfläche cos können Sie z. B. den Kosinus einer Zahl berechnen.
