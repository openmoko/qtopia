Mit den Anwendungen arbeiten

Tippen Sie auf das Symbol einer Anwendung, um die Anwendung zu starten.

Wenn Sie Informationen zu einem Element oder einer Funktion benötigen, tippen Sie darauf und halten Sie den Stift gedrückt. Daraufhin wird entweder ein Menü, ein Dialogfeld mit Optionen oder ein Fenster mit zusätzlichen Informationen angezeigt.

Durch Tippen auf lassen sich Fenster schließen. Manche Fenster enthalten außerdem neben der Schaltfläche eine Schließen-Schaltfläche. Wenn Sie auf tippen, wird das Fenster ohne vorheriges Speichern geschlossen.
