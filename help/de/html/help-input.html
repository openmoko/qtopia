Eingabemethoden

In der linken unteren Bildschirmecke wird standardmäßig das Symbol "abc" für die Eingabe von Text angezeigt.

Wenn Sie auf das kleine Dreieck rechts neben diesem Symbol tippen, wird eine Liste der verfügbaren Eingabemethoden angezeigt (z. B. Handschrift, Pickboard, Tastatur und Unicode). Tippen Sie auf die gewünschte Eingabemethode. Anstelle des Symbols "abc" wird dann das Symbol für die gewählte Eingabemethode angezeigt.

Durch Tippen auf das Symbol für die Eingabemethode wird der Eingabebereich eingeblendet; durch erneutes Antippen des Symbols wird er wieder ausgeblendet.
