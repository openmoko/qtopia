Termine hinzufügen

Tippen Sie in der aktiven Ansicht auf das Datum, für das Sie einen Termin eingeben möchten. Falls Sie nicht bereits in der Tagesansicht arbeiten, wird diese aufgerufen.

Tippen Sie auf die gewünschte Uhrzeit und anschließend auf , um über das Dialogfeld Neuer Termin einen Termin einzugeben. Sie können auch auf die Uhrzeit tippen und mit der Eingabe einer Beschreibung für den Termin beginnen. Wählen Sie hierzu eine der Eingabemethoden.
