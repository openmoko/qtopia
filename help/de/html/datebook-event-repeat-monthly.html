Monatlich stattfindende Ereignisse

Tippen Sie auf die Pfeile im Feld "Alle", um den Monatsrhythmus festzulegen, in dem das Ereignis stattfinden soll.

Falls das Ereignis ein Enddatum hat, deaktivieren Sie das Kontrollkästchen "Kein Enddatum", tippen Sie auf den Pfeil rechts neben dem Datumsfeld und wählen Sie in dem daraufhin eingeblendeten Kalender ein Enddatum.

Tippen Sie im Bereich "Wiederholung" auf die Schaltfläche Datum um festzulegen, dass das Ereignis immer an dem gewählten Datum des Monats stattfindet. Durch Tippen auf die Schaltfläche Tag können Sie festlegen, dass das Ereignis jeden Monat am gewählten Tag stattfindet.

Bestätigen Sie die Wiederholung des Ereignisses durch Tippen auf oder brechen Sie den Vorgang durch Tippen auf ab.
