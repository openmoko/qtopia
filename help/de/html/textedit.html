Der Texteditor

Wenn Sie den Texteditor öffnen, wird eine Liste der auf dem Gerät gespeicherten Textdateien angezeigt. Tippen Sie auf den Namen der gewünschten Datei oder auf , um eine neue Datei zu erstellen.

Wenn Sie eine Datei geöffnet oder neu erstellt haben, werden ein Menü und eine Symbolleiste angezeigt.

Geben Sie den gewünschten Text mit einer der Eingabemethoden ein.
