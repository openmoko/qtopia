Alarmeinstellungen für den Kalender

Alarme werden standardmäßig die Anzahl der Minuten vor einem Ereignis ausgelöst, die im Feld "Alarmvorgabe" definiert ist. Tippen Sie in das Kontrollkästchen "Alarmvorgabe" und stellen Sie anschließend über die Pfeilschaltflächen die gewünschte Alarmvorgabe ein.

Speichern Sie die Einstellungen durch Tippen auf oder brechen Sie den Vorgang durch Tippen auf ab.
