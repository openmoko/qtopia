Das Pickboard-Menü

Am rechten Rand des Pickboards befindet sich ein Einblendmenü, das die Optionen "Hinzufügen" und "Zurücksetzen" enthält.

Um ein Wort hinzuzufügen, tippen Sie nacheinander auf die Buchstabengruppen für die einzelnen Buchstaben des Wortes. Tippen Sie dann auf die Pfeilschaltfläche für das Menü und tippen Sie auf Hinzufügen. Das Dialogfeld "Hinzufügen" wird angezeigt, das Buchstabenlisten enthält. Tippen Sie auf die einzelnen Buchstaben des Wortes und anschließend auf OK, um das Wort in das Wörterbuch aufzunehmen.

Um die Auswahl eines Wortes über das Pickboard abzubrechen, tippen Sie auf die Pfeilschaltfläche für das Menü und anschließend auf Zurücksetzen.
