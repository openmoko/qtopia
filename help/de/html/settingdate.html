Das Datum einstellen

Tippen Sie auf das Datum, um den Kalender einzublenden und ein Datum festzulegen.

Tippen Sie in der Liste "Wochenbeginn" auf "Sonntag" oder "Montag".

Tippen Sie in der Liste "Datumsformat" auf das gewünschte Format.

Bestätigen Sie das neue Datum und die neue Uhrzeit, indem Sie auf tippen, oder tippen Sie auf die Schaltfläche {X}, um den Vorgang abzubrechen.
