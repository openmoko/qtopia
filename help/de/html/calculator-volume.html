Der Modus "Volumen"

Geben Sie eine Einheit gefolgt von einer Zahl ein. Tippen Sie beispielsweise auf pt und dann auf die Zahl 4, um den Wert 4 Pint einzugeben. Sie können diesen Wert dann konvertieren, indem Sie die gewünschte Maßeinheit antippen (durch Antippen von gal erhalten Sie beispielsweise 0.5 Gallonen).

Die Schaltfläche für die aktuelle Maßeinheit wird vertieft dargestellt. So können Sie schnell erkennen, welche Einheit gerade verwendet wird.
