Dialogfeld "Kategorien bearbeiten"

Kategorien einrichten
Aktivieren Sie das Kontrollkästchen jeder Kategorie, der das Element angehören soll. Lassen Sie die Kontrollkästchen der nicht benötigten Kategorien deaktiviert.

Kategorien hinzufügen
Tippen Sie auf die Schaltfläche Hinzufügen, tippen Sie links unten in das Bearbeitungsfeld und geben Sie mit einer der Eingabemethoden einen Namen ein. Aktivieren Sie das Kontrollkästchen "Global", wenn die Kategorie in allen Anwendungen zur Verfügung stehen soll.

Sie können Kategorien auch umbenennen und löschen.
