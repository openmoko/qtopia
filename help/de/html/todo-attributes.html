Attribute für Aufgaben festlegen

Tippen Sie auf die Listen "Priorität" und Kategorie, um die Priorität und die Kategorie der Aufgabe festzulegen. Wenn Sie neue Kategorien erstellen oder der Adresse mehrere Kategorien zuweisen möchten, tippen Sie auf die Schaltfläche ..., um das Dialogfeld "Kategorien bearbeiten" aufzurufen.

Aktivieren bzw. deaktivieren Sie das Kontrollkästchen "Erledigt".

Falls die Aufgabe zu einem bestimmten Zeitpunkt ausgeführt werden soll, aktivieren Sie das Kontrollkästchen "Frist" und tippen Sie auf die Datumsschaltfläche, um das Datum über den Kalender festzulegen.
