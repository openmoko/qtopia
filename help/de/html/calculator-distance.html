Der Modus "Distanz"

Geben Sie eine Einheit gefolgt von einer Zahl ein. Tippen Sie beispielsweise auf m und dann auf die Zahl 10, um die Entfernung 10 m einzugeben. Sie können diesen Wert dann konvertieren, indem Sie die gewünschte Maßeinheit antippen (durch Antippen von yd erhalten Sie beispielsweise 10.9...yd).

Die Schaltfläche für die aktuelle Maßeinheit wird vertieft dargestellt. So können Sie schnell erkennen, welche Einheit gerade verwendet wird.
