Täglich stattfindende Ereignisse

Tippen Sie auf die Pfeile im Feld "Alle", um den Tagesrhythmus festzulegen, in dem das Ereignis stattfinden soll.

Falls das Ereignis ein Enddatum hat, deaktivieren Sie das Kontrollkästchen "Kein Enddatum", tippen Sie auf den Pfeil rechts neben dem Datumsfeld und wählen Sie in dem daraufhin eingeblendeten Kalender ein Enddatum.

Bestätigen Sie die Wiederholung des Ereignisses durch Tippen auf oder brechen Sie den Vorgang durch Tippen auf ab.
