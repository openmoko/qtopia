E-Mail-Adressen eingeben

Geben Sie die E-Mail-Adresse des Empfängers in das Feld An:" ein. Verwenden Sie hierfür eine der Eingabemethoden.

Falls die Nachricht an mehrere Empfänger gesendet werden soll, können Sie die einzelnen E-Mail-Adressen durch Komma getrennt eingeben. Wenn Sie die Nachricht außerdem als Kopie oder Blindkopie an weitere Empfänger senden möchten, wählen Sie in der Liste "An:" die Optionen "CC" bzw. "BCC" und geben Sie die gewünschte Adresse ein.

Sie können die Eingabe beschleunigen, indem Sie die Adresse über das Adressbuch eingeben.
