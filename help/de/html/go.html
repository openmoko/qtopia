Go

Erobern Sie mit Ihren Steinen das Spielbrett und lassen Sie sich dabei nicht von Ihrem Gegner gefangen nehmen.

Starten Sie ein Spiel, indem Sie auf Spiel und auf Neues Spiel tippen. Tippen Sie auf einen Schnittpunkt, um einen Stein auf das Brett zu setzen. Wenn Sie keinen Stein setzen möchten, tippen Sie auf Spiel und dann auf Passe. Tippen Sie auf Spiel und dann auf Aufgeben, um sich geschlagen zu geben. Wenn Sie gegen einen menschlichen Gegner spielen möchten, tippen Sie auf Spiel und dann auf Zwei Spieler.

Beenden Sie das Spiel durch Tippen auf .
