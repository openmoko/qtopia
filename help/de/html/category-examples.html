Beispiele für Kategorien

Wenn Sie das {QP}-Gerät sowohl für dienstliche als auch private Zwecke einsetzen, können Sie beispielsweise eine Kategorie "Arbeit" und eine Kategorie "Privat" erstellen.

Wenn Sie an vielen unterschiedlichen Projekten arbeiten, kann es sinnvoll sein, für jedes Projekt eine Kategorie anzulegen, um sich schnell und einfach die Informationen zu dem jeweiligen Projekt anzeigen zu lassen.
