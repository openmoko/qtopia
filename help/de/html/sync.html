Synchronisation mit einem PC

Klicken Sie in der Symbolleiste von Qtopia Desktop auf die Schaltfläche Alles synchronisieren.

Wenn Sie den Ablauf der Synchronisation genau steuern möchten, können Sie auch einzelne Anwendungen separat synchronisieren sowie während der Synchronisation zusätzliche Software installieren. Weitere Informationen hierzu finden Sie in der Onlinehilfe von Qtopia Desktop.
