Sicherheit

Sie können Ihr {QP}-Gerät mit Hilfe einer Passcode-Nummer sichern und so einen minimalen Schutz gegen unbefugte Benutzung des Gerätes gewährleisten.

Tippen Sie auf Passcode einstellen, um die Nummer festzulegen. Geben Sie die Nummer ein und tippen Sie auf "OK". Wiederholen Sie diese Schritte.

Aktivieren Sie das Kontrollkästchen "Passcode beim Einschalten anfordern", um die Passcode-Funktion zu aktivieren. Die Codenummer muss dann bei jedem Einschalten oder Reaktivieren des {QP}-Geräts eingegeben werden.
