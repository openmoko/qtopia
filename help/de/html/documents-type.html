Dokumente nach Typ anzeigen

Tippen Sie unten links auf die Schaltfläche Typ und anschließend auf das gewünschte Auswahlkriterium. Wenn z. B. nur Textdokumente angezeigt werden sollen, tippen Sie auf "Text". Alle Dokumente eines anderen Typs werden dann ausgeblendet.

Tippen Sie auf "Alle", um Dokumente aller Typen anzuzeigen.
