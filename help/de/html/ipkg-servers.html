Paket-Server

Tippen Sie auf die Schaltfläche Upgrade, um die installierte Software über den Paket-Server zu aktualisieren.

Tippen Sie auf das Schaltersymbol, um das Dialogfeld "Paket-Server" zu öffnen. Tippen Sie auf die Schaltfläche Neu, tippen Sie in das Feld "Name" und geben Sie mit einer der Eingabemethoden einen Namen ein. Tippen Sie nun in das Feld "URL" und geben Sie die URL für den Paket-Server mit einer der Eingabemethoden ein. Sie können mit beliebig vielen Paket-Servern arbeiten.

Wenn Sie einen Server aus der Liste entfernen möchten, tippen Sie in der Liste der Server auf den Namen des Servers und anschließend auf Entfernen.

Tippen Sie auf {OK}, wenn Sie Ihre Arbeit in dem Dialogfeld beendet haben.
