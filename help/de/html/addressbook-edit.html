Dialogfeld "Adresse ändern"

Tippen Sie in die einzelnen Felder, die Sie ausfüllen möchten, und geben Sie die Informationen mit einer der Eingabemethoden ein. (Die Reihenfolge der Felder kann über das Dialogfeld Einstellungen festgelegt werden.)

Tippen Sie auf die Bildlaufleiste, um weitere, nicht sichtbare Felder anzuzeigen.

Wenn Sie Kategorien bearbeiten möchten, tippen Sie auf die Schaltfläche ....

Tippen Sie auf die Schaltfläche Notizen, um das Dialogfeld Notiz bearbeiten aufzurufen.

Speichern Sie die Adresse durch Tippen auf oder brechen Sie den Vorgang durch Tippen auf ab.
