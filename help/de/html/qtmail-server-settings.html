Servereinstellungen

Legen Sie die gewünschten Optionen fest, indem Sie die entsprechenden Kontrollkästchen aktivieren bzw. deaktivieren.

Synchronisieren Diese Option sollte in den meisten Fällen aktiviert sein.

E-Mail löschen Wenn diese Option aktiviert ist, werden Nachrichten nach dem Abruf auf dem Server gelöscht.

Große E-Mails auslassen Wenn diese Option aktiviert ist, verbleiben große E-Mails auf dem Server. Sie erhalten jedoch eine Benachrichtigung, über die Sie die Nachrichten vollständig herunterladen können.

Bestätigen Sie die Einstellungen durch Tippen auf oder brechen Sie den Vorgang durch Tippen auf ab.
