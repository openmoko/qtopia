Die Playlist

Wenn Sie auf Playlist|Playlist umschalten, wird der Bildschirm in zwei Hälften unterteilt, so dass Playlist und Medienbibliothek gleichzeitig angezeigt werden. (Durch erneutes Tippen auf Playlist|Playlist umschalten kehren Sie zur normalen Ansicht der Medienbibliothek zurück.)

Tippen Sie im unteren Fenster die gewünschten Dateien in der Reihenfolge an, in der sie in der Playlist angezeigt werden sollen. Tippen Sie auf , um alle Dateien in der Liste nacheinander abzuspielen.

Die Reihenfolge der Dateien in der Playlist kann geändert werden, indem Sie eine Datei antippen und anschließend auf (Auf) bzw. (Ab) tippen. Tippen Sie auf , um eine Datei aus der Playlist zu löschen.
