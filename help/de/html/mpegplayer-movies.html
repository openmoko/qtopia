Filme abspielen

Tippen Sie auf einen Film, um die Steuerelemente zum Abspielen des Films aufzurufen. Tippen Sie anschließend auf die Wiedergabeschaltfläche.

Es stehen folgende Steuerelemente zur Verfügung: Zurückspulen, Beenden, Anhalten, schnelles Vorspulen, Schließen und Vollbildanzeige.

Tippen Sie auf , damit das Wiedergabefenster ohne Skalierung einen möglichst großen Bereich einnimmt. Um die Wiedergabe anzuhalten, tippen Sie auf den Film.

Tippen Sie auf {X}, um die Wiedergabe zu beenden und zur Liste der Mediendateien zurückzukehren.

Wenn Sie mehrere Dateien nacheinander abspielen möchten, können Sie eine Playlist erstellen.
