LAN-Account

Vervollständigen Sie die Angaben, indem Sie die Informationen mit einer der Eingabemethoden eingeben.

Name Geben Sie den gewünschten Namen für dieses Account ein.

Tippen Sie abhängig davon, ob das {QP}-Gerät die TCP/IP-Informationen automatisch ermitteln soll (Standardeinstellung) oder ob Sie die Informationen manuell eingeben möchten, auf eines der Optionsfelder.

Wenn Sie festgelegt haben, dass Sie die TCP/IP-Informationen manuell eingeben, müssen Sie die Angaben auf den Registerkarten TCP/IP und DNS vervollständigen.
