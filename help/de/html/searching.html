Suchen

Tippen Sie auf , um das Dialogfeld "Suchen" aufzurufen.

Geben Sie den zu suchenden Text mit einer der Eingabemethoden in das Feld "Suchbegriff" ein und tippen Sie anschließend auf die Schaltfläche Suchen, um nach dem nächsten Element mit dieser Zeichenfolge zu suchen.

Durch Tippen auf können Sie das Dialogfeld schließen.
