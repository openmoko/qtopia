Der Modus "Gewicht"

Geben Sie eine Einheit gefolgt von einer Zahl ein. Tippen Sie beispielsweise auf kg und dann auf die Zahl 2, um das Gewicht 2 kg einzugeben. Sie können diesen Wert dann konvertieren, indem Sie die gewünschte Maßeinheit antippen (durch Antippen von lb erhalten Sie beispielsweise 4.4...lb).

Die Schaltfläche für die aktuelle Maßeinheit wird vertieft dargestellt. So können Sie schnell erkennen, welche Einheit gerade verwendet wird.
