Patience

Legen Sie die Karten vom Ass bis zum König in der passenden Reihenfolge ab.

Wählen Sie aus dem Menü Spiel Patience oder Freecell. Tippen Sie auf die Karten und ziehen Sie sie auf den gewünschten Stapel. Durch wiederholtes Tippen auf Einstellungen|Kartenrückseite ändern können Sie auf der Kartenrückseite ein anderes Bild anzeigen. Tippen Sie auf Einstellungen|Ausrichten, um die Karten auf dem gewünschten Stapel auszurichten. Tippen Sie auf Einstellungen|Drei Karten ziehen, wenn Sie statt einer drei Karten auf einmal ziehen möchten. Beenden Sie das Spiel durch Tippen auf .
