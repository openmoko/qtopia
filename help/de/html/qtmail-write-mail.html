E-Mail schreiben

Dieses Dialogfeld wird aufgerufen, wenn Sie zum Beantworten einer Nachricht auf tippen oder zum Verfassen einer neuen Nachricht auf tippen.

Geben Sie die gewünschten Adressen ein, tippen Sie in das Feld "Betreff" und geben Sie mit einer der Eingabemethoden einen Betreff ein. Geben Sie den Text der Nachricht in den leeren Bereich unterhalb des Feldes "Betreff" ein. Verwenden Sie dazu eine der Eingabemethoden.

Tippen Sie auf {OK}, wenn Sie die Nachricht erstellt haben.

Die Nachricht wird im Ausgang gespeichert.

Sie können alle Nachrichten im Ausgang senden, indem Sie auf tippen.
