Alarm einstellen

Wenn Sie an den Termin erinnert werden möchten, können Sie das Kontrollkästchen "Alarm" aktivieren und festlegen, wie viele Minuten vor dem Termin der Alarm ausgelöst werden soll.

Sie können den Alarm auf "Leise" stellen, damit die Erinnerung in Form einer Meldung auf dem Bildschirm erfolgt, oder auf "Laut", so dass eine Klingel ertönt. Tippen Sie auf OK, um den Alarm auszuschalten.

Falls der Termin mehrmals stattfinden soll, tippen Sie auf die Schaltfläche Keine Wiederholung. Das Dialogfeld Mehrfaches Ereignis wird angezeigt, in dem Sie die gewünschten Einstellungen vornehmen können.
