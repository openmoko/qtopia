Netzwerkverbindung hinzufügen

Die verfügbaren Verbindungen werden in einer Liste aufgeführt. Tippen Sie auf die gewünschte Verbindung und anschließend auf Hinzufügen. Abhängig davon, welche Art von Verbindung hinzugefügt werden soll, wird das Dialogfeld Modemverbindung oder LAN angezeigt.
