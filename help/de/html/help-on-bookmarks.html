Lesezeichen

Wenn Sie auf das Menü Lesezeichen und anschließend auf Lesezeichen einfügen tippen, können Sie ein Lesezeichen für die aktuelle Seite erstellen.

Sie können diese Seite später erneut aufrufen, indem Sie auf Lesezeichen und anschließend auf den Namen des zuvor erstellten Lesezeichens tippen.

Wenn Sie ein Lesezeichen löschen möchten, tippen Sie auf Lesezeichen, tippen Sie auf den Namen der Seite und anschließend auf Lesezeichen|Lesezeichen entfernen.
