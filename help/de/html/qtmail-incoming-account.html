Eingehendes Account

Tragen Sie die gewünschten Informationen mit einer der Eingabemethoden in die Felder ein.

Account-Typ wählen Tippen Sie in der Auswahlliste auf den Eintrag "POP" (oder "IMAP", falls dieser Account-Typ von Ihrem Serviceprovider unterstützt wird).

Benutzername Geben Sie den Benutzernamen für das Account ein.

Passwort Geben Sie das Passwort für das E-Mail-Account ein.

Server Geben Sie den Namen Ihres E-Mail-Servers ein, den Sie von Ihrem Serviceprovider erhalten haben.

Port In diesem Feld ist standardmäßig der Wert 110 eingestellt.

Sie können nun die Servereinstellungen vervollständigen.
