Proxies

Tippen Sie in der Auswahlliste auf eine der folgenden Optionen:

Keine Proxies - Wählen Sie diese Option, wenn Sie nicht mit Proxies arbeiten möchten.

Proxies automatisch konfigurieren - Tippen Sie anschließend in der Liste "Konfiguration" auf eine Konfiguration, damit das {QP}-Gerät die Proxies automatische konfiguriert.

Proxies einrichten - Mit dieser Option können Sie die Proxies manuell einrichten. Geben Sie den Hostnamen bzw. die IP-Adresse und die Port-Nummer für die zu verwendenden HTTP- und FTP-Proxies ein. Sie erhalten diese Werte von Ihrem Serviceprovider.

Tippen Sie auf {OK}, um die Verbindung hinzuzufügen, bzw. auf {X}, um den Vorgang abzubrechen.
