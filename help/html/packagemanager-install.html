Install Application

  • Use / to view the Downloads tab.
  • Press Options and select Connect to be shown a list of available servers.
  • Select a server to connect to it. A list of available applications are displayed and the connection's status is shown in the bottom pane.
  • Highlight an application , press Options and select Install. You will be asked to confirm installation. An SXE enabled device also will display the capabilities the application requests.
  • To confirm installation press Options and select Confirm. A successful installation dialog will be presented. Press Ok and the newly installed package will be highlighted in the Installed tab. (The application will no longer be available in the Downloads tab.)
