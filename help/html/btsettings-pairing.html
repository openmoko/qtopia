Pairing with other devices

If you need to use a Bluetooth device with your own , you will probably need to "pair" with the other device (as a security precaution) before you can use it. To do this, go to the Paired Devices tab on the main screen, press Options and select Pair with new device... from the menu. This starts a search for nearby Bluetooth devices and displays a list of devices that have been found.

Choose the device that you want to pair with from the list. You will be prompted to enter a PIN code; once you have entered a PIN, and the same PIN code is then entered on the other device, the pairing process is complete. The device will now appear in the Paired Devices list.

To remove a pairing, select the device in the list of paired devices, press Options and select Remove pairing. Once the pairing has been removed, the device will no longer appear in the list.
