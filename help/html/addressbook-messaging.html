Messaging Contacts

Messages can be created and addressed quickly from Contacts.

To create and address a message:

  • View the contact you want to message. (See Viewing Contacts)
  • Select the mobile phone(,) or email address () to message by highlighting it and pressing Select. pressing it.
  • When sending to a mobile phone you must select the message type, i.e. SMS or MMS .
  • Complete and send the message.

This command can only be used with contacts that have a mobile phone or an email address.
