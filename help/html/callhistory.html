Call History

Call History is invoked by pressing from the home screen and displays the last 50 calls.

If a number is in Contacts then the icon shows its type (home, mobile, etc).

Use / to display:

Touch the tabs to display:

  • all calls.
  • outgoing calls.
  • incoming calls.
  • missed calls.

Highlight and select a call to view more details of its history.

The Options menu provides the following actions:

Open Contact: opens the Contacts application to show the details of a contact associated with the selected call history entry.

Save to Contacts: save the number of the selected entry in Contacts if the number has no associated contact already.

Send Message: opens a message composer to send a text message to the number associated with the selected entry.

Related Calls: shows calls to/from the same contact or number.

Clear...: removes all call history entries.
