Call History

Call history is invoked by pressing from the home screen and displays the last 50 calls.

If a number is in Contacts then the icon shows its type (home, mobile, etc).

Use / to display:

Touch the tabs to dispay:

  • all calls.
  • outgoing calls.
  • incoming calls.
  • missed calls.

Highlight and select a call to view more details of its history.

Open Contact: opens Contacts to show the details of the saved contact.

Send Message: opens a message composer if the type of number is a mobile number.

Save to Contacts: allows to save the number in Contacts.

Related Calls: shows calls to/from the same contact or number.

Clear...: allows to remove call history entries.
