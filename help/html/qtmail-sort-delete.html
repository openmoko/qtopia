Sorting and Deleting Email

Messages can be sorted in the Messages page by:

  • Tapping the From, Subject, or Date column headers to sort your messages.
  • Tap an email to view it using the Reading mail dialog.

To delete the current message:

  • Tap Mail, then Delete.
  • The message will be moved into the Trash folder.
  • If the mail is already in the Trash folder a message will prompt for confirmation of deletion.
