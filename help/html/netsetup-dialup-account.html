Dial-Up Account

Used to define internet account details.

Complete the form by entering the information into the following fields:

Name: the name you wish to give this account, eg Bigpond.

Startup mode: the following modes are available:

  • When needed (the service needs to be started manually)
  • Always online (the service is started in demand mode and connects automatically when a data request occurs)

Dial-up number: to be used for dialing up.

Username: used to logon to internet.

Password: used to logon to internet.
