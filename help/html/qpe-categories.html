Category Help

Many Qt Extended applications support the concept of categories. Contacts, Events, Tasks and Documents can all be associated with one or more categories. Categories help in quickly finding a record, such as a: contact, audio or video clip, picture, note, or task.

Most categories only work with one application. Global categories (marked with a ) work across all applications.

To set the category for a record: when editing a record, highlight and select the Category field to see a list of categories; turn on categories applying to this record.

When trying to find a record, choose View Category... from the Options menu and pick one or more categories to view. Only records in those categories will be shown.
