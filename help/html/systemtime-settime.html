Setting the Time

To set the time:

  • Launch Settings by using to focus on and pressing Select to select.
  • Use to scroll down the list and Select to select Time.
  • Use the up / down arrows to reach the hours field.
  • Press to backspace and delete.
  • Use the keyboard to enter the time.
  • Press OK to finish a field.
  • Repeat for the minutes field.
  • Change the time format to match your entry. (Down arrow then Select then arrow to highlight the correct format and Select) to select.
  • Press to save and exit.
