Setting the Time

To set the time:

  • Tap the up / down arrows on the Hour field to set the hour.
  • Tap the up / down arrows on the Minute field to set the minutes.
  • Tap the AM/PM drop-down list to select the time of day.

To change the time format:

  • Tap the Time Format drop-down list.
  • Select either a 12 hour or 24 hour time format.

Note: When the 24 hour time format is selected the AM/PM drop-down list becomes unavailable.
