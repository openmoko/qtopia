Add Tasks

To add a new task:

  • Tap Task, then New Task.
  • Enter the details of the task in the Edit Task dialog.
  • Tap to save the task, or to cancel.
  • Launch the Tasks application.
  • Press and select New from the menu.
  • Fill in the Task fields. Use and to select a field and the keypad to enter the text.
  • Add any free text notes on the Notes tab.
  • Press to save the task and return to the Tasks application list view.
