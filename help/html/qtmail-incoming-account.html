Incoming Account

Fill in the fields using one of the Input Methods.

Choose account type tap the combobox and tap POP or IMAP (Check with your ISP which account you have) If this account is used only for synchronizing with the desktop choose Synchronized instead.

Username enter your email account's username.

Password enter your email account's password.

Server enter the email server name provided by your ISP.

Port defaults to the correct value of 110 for POP and 143 for IMAP (you normally do not need to change these).

Now complete the Server Settings.
