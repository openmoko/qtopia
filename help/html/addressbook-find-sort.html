Finding and Sorting Contacts

To Find a Contact:
- Tap Contact, then Find
- Enter the text you want to search for on the find line (all fields will be searched)
- Tap the Go To button on the find line to locate the first matching entry
- Continue tapping the Go To button to skip from one matching entry to the next

To Sort the Contact List:
- Change to the Contact List view
- Tap the column heading for the column you want to sort by, e.g. "File As"
An arrow will appear beside the column heading indicating the direction of the sort
- Tap the column heading again if you want to reverse the sort order

Once a contact has been assigned to a Category in the Edit Contact Details screen the Contact List can be filtered to show only contacts assigned to a selected category.

You can change the column headings used in the Contact List view using the dropdown list in the lower left corner of the screen or the Configure headers command.
