Pickboard Top Row

This row has the 123 letter group for entering digits, the symbols letter group and some command words.

Tap Space to enter a space, tap Back to backspace delete, and tap Enter to move to a new line. CAPS LOCK is controlled by tapping Shift. The KEY group leads to a set of keyboard navigation keys, e.g. PgUp, PgDn, etc. Tap ESC to return to the normal Pickboard.
