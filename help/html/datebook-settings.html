Calendar Settings

Use Calendar Settings to change the view and alarm preferences for the calendar.

View settings are used to control the appearance of the calendar.

Default view: determines whether Calendar starts in Day View or Month View each time it is launched.

Day starts at: first blank time slot to display; events set for an earlier time will still show.

To change the start time:

  • Tap View, then Settings...
  • Adjust the Start Time by tapping the up / down arrows beside the current start time.
  • Tap to save your changes or to cancel.

Alarm settings are used to specify if an alarm should be preset for newly created events and if so, how many minutes prior to the start time of the event should the alarm go off.
