Add / Edit Event

To add an event to your calendar or edit an existing event:

  • Adding an event: Launch Calendar and press Options, then select New from the menu.
  • Editing an event: Select the event in Calendar and press Options, then select Edit event from the menu.

The details of an event are split over three tabs.

The Event tab has fields for most of the details of the event, including:

  • Desc. description shown for the event.
  • Loc. where the event is.
  • All day event select if the event covers the whole day.
  • Start, End date/time of start and end of event. If the All day event checkbox is selected, only the start/end date of the event will be shown. If you adjust the start time of an event, the end time will also be adjusted to keep the same event duration. To make an event longer or shorter, first set the start time of the event, then adjust the end time.
  • T.Z. time zone where event takes place.
  • Reminder choose whether to have a reminder that the event will start soon. You can choose either a silent reminder or an audible reminder, and when you want the reminder to occur. For an all day event, you can choose the time of day that the reminder should occur.

The Recurrence allows you to configure how the event repeats, including:

  • Repeat choose a type of repetition (daily, weekly etc). Depending on the type of repetition, more options may be available to customize the repetition.
  • Frequency choose how often the repetition will occur. For example, a frequency of 2 and a repetition type of daily means that the event will repeat every 2 days.
  • Repeat until check this option and select a date if the repetitions should stop on or before a specific date.
  • On these days when choosing a weekly repeat, an option to choose additional days of the week to repeat on will appear. The original event start day of the week is always selected. This allows you to create events that will repeat on every weekday, or on any particular days you specify.
  • Repeat on when choosing a monthly repeat, you can specify how the day of the month is interpreted. This allows you to specify events that recur on certain days of certain weeks (for example, the first Tuesday), or a fixed day of the month (for example, the 17th of each month).

Notes this tab contains extra information about the event, including:

  • Category used to group similar events.
  • Notes enter additional information for an event.
Navigating the fields:
  • Use to and Select to select a field.
  • Use to backspace and delete the text, then use the keypad to enter new text.
  • For fields with a list of options, simply select another.
  • Press OK to accept new text in a field or use to move to the next.
  • Press to save changes to the task and exit, or Options then Cancel from the menu.
