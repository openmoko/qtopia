Selecting Messages

Messages you have received are listed in the Inbox folder. Select the Inbox folder from the folder list to see the messages you have received.

Press / to locate a message you wish to view, then press Select to view the message contents.

Viewing Messages

The contents of messages are rendered as text, followed by the details of the origin of the message. When viewing a message, the window title shows the name of the sender of the message.

Phone numbers, email addresses and URLs in messages are displayed as hyperlinks.

Press / to page through a message. If the message contains hyperlinks, the selection location will cycle through the available hyperlinks.

If you select a hyperlink containing a phone number or an email address, a new message to that recipient will be composed.

If a message contains phone numbers, a Dial action will be added to the context menu for each phone number. Select the Dial action to dial the relevant phone number.
