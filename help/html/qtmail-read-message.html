Viewing Messages

The contents of messages are rendered as text, followed by the details of the origin of the message.

Phone numbers, email addresses and URLs in messages are displayed as hyperlinks.

Press / to page through a message. If the message contains hyperlinks, the selection location will cycle through the available hyperlinks.

If you select a hyperlink containing a phone number or an email address, a new message to that recipient will be composed.

If a message contains phone numbers, a Dial action will be added to the options menu for each phone number. Select the Dial action to dial the relevant phone number.

If the sender of a message is unknown, i.e. has no matching entry in the Contacts store then a Save to Contacts action will be added to the context menu. Select the Save to Contacts action to either create a new contacts entry for the sender of the message, or to add to an existing contacts entry.
