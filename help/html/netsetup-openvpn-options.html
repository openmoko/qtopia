Options

Used to define additional OpenVPN options/details.

Complete the form by entering the information into the following fields:

LZO compression: enables fast LZO compression.

Pull paramter: enables the client to receive IP and route parameter when during the process of establishing the connection.

Explicit exit notification: When a UDP client terminates the connection it can send an exit notification to its peer.

Configuration script: all OpenVPN parameter can be provided by a configuration script. This option allows the use of such a script. The most common reason for doing this would be a configuration script provided by a system administrator. Note that when specifying such a script Qt Extended assumes that the script specifies all required parameter.
