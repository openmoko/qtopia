Renaming/Deleting Files or Folders

To rename a file or folder:
- Tap and hold the file or folder to rename
- A pop-up list of commands will appear
- Tap Rename
- Use on of the Input Methods to enter a new name
- Tap a blank area of the screen

To delete a file or folder:
- Tap and hold the file or folder to delete
- A pop-up list of commands will appear
- Tap Delete
- A message box will ask if you are sure
- Tap YES to continue or NO to cancel

See also Files, Folders, and Moving/Copying Files and Folders
