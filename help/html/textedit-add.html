Add a Note

  • Press and select New from the menu.
  • Use the keypad to enter the text.
  • Press to accept the text.
  • Press to save the text and exit.

To create a new file:

  • Tap New.
  • Enter text using one of the Input Methods.
  • Save the note and exit.

Once you've opened or created a file the text editor will provide a menu and toolbar for working with the file.
