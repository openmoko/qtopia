Editing Categories

Categories can only be edited from an application that uses categories. Default categories cannot be edited.

  • Open the Documents list.
  • Highlight any document.
  • Press Options and select Properties.
  • Press the Category button.
  • When the list of available categories is displayed, highlight the category to be edited.
  • Press Options and select Edit.
  • When finished editing, press Back to save and exit.
  • Unless your are going to use the new category immediately, press Back again to exit Documents.
