Editing Categories

Categories can only be edited from an application that uses categories. Default categories cannot be edited.

  • Launch Documents by using and pressing Select.
  • Highlight any document.
  • Press Options and select Properties.
  • Use to highlight the Category field and press Select.
  • When the list of available categories is displayed, use to highlight the category to be edited.
  • Press Options and select Edit.
  • Press Select to open the field, then change the name of the category.
  • Press OK to complete the field.
  • Use and Select to tick/untick the Global check-box.
  • Press Back to save and exit.
  • Unless your are going to use the new category immediately, press Back again to exit Documents.
