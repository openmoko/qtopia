Synchronizing with a PC

  • Run the Qtopia Sync Agent application on your desktop PC.
  • Place the Qtopia device in its cradle.
  • Qtopia Sync Agent will automatically establish a connection to the Qtopia device.
  • Click the Sync All button on the Qtopia Sync Agent toolbar.

Finer control is available for synchronizing individual applications, and installing software.

See the Qtopia Sync Agent online help for more information.
