File Navigation

The files on your device are arranged into folders to make them easier to identify and locate.

%%PDA%%

To view the contents of a folder:

  • Tap the name of the folder (a folder may contain a combination of files and folders).

Once you have opened a folder you can:

  • Tap to return to the previous folder.
  • Tap to move up to the parent folder.
  • Tap Dir, then "/" to return to the initial list of files and folders.

Files and Folders can be sorted by Name, Size, Date or Type in ascending or descending order.

To change the type of sort:

  • Tap Sort, then select the type of sort.

To change the direction of the sort:

  • Tap Sort, then Ascending
