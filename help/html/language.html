Language

Your Qtopia device can operate in a number of languages. The Language Settings application allows you to set the primary language to be used for display and input. It also allows you to set any number of secondary languages, which can be used for input.

To change the primary language: highlight the language, then press Select, press on the language and answer Yes to the confirmation dialog to activate the selected language. The device will then be restarted to apply the new language.

To use a secondary language for input: highlight the language and select Use for input from the Options menu.
