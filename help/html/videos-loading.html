Loading New Videos

CF and SD Cards

Inserting a CF or SD card prompts Qtopia to scan it for files and use the Documents application to display a list of all those it finds (except hidden files).

PC-based Video Files

To store PC-based video files on the PDA drag'n'drop the files in to Qtopia Desktop's documents view and use the synchronization functions to transfer the files to the PDA and make them available for selection in Videos.

Transfering Files

To transfer media files by any other means requires using the File Manager. Tap and hold on each video file, then tap the Add to Documents menu option. The files will now be available for selection in Videos.
