Reading Responding to Mail

To view the contents of your Inbox:
- Tap the Inbox, then tap the Messages page

To view the contents of a message
- Tap the messages subject
- Tap View, then Previous or Next to other messages in the current folder

To reply to the current message:
- Tap Mail, then Reply.
- Enter your message into the message window. See Write Mail for more information.
- Queue the message for sending.

The Mail menu also contains commands for replying to all, forwarding messages and changing the status of a message.

To delete the current message:
- Tap Mail, then Delete
- The message will be moved into Trash folder.
- If the mail is already in the Trash folder a message box will ask for confirmation of deletion.

To view the contents of another folder:
- Tap the Folders tab, and select the tab to view
- Tap the Messages page
- e.g. Tap Outbox to view all messages queued for sending.

If you are viewing the Outbox folder you can send the selected message by tapping the button.

If the mail is Unsent or in the Drafts folder you can edit it by tapping which will open the Write Mail window.

E-mail addresses and URL's in mail messages are displayed in blue and underlined. Tapping an email addresses will give you an option of either composing a message to this address, or adding them to your Contacts. Clicking an URL will open it in the default Web Browser.

Messages can be sorted in the Messages page by:
- Tapping the From, Subject, or Date column headers to sort your messages.

Read email is shown in black, unread email is shown in blue. Emails that are not yet downloaded are shown in red. Tap an email to view it using the Reading mail screen.
