Message Attachments

To attach a document or an image to a message:

  • Compose the message.
  • Tap to display the list of available documents.
  • Select a document and tap Attach to move it into the Attached column.
  • To remove a document from the list of those that will be attached, select the document and tap Remove.
  • When all required attachments are shown in the Attached column tap to return to the email and send as usual.
  • Press and select Attachments. The Attachments screen will be shown (blank unless an attachment has already been added).
  • Press and select Add Attachment.
  • Select the the document to be attached by moving through the list using and pressing to attach a document.
  • When all required attachments are shown on the screen, press to return to the body of the message.
  • When message is complete, press and then .
  • Enter the address details.
  • Press to send the message.
