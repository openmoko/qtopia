Scientific mode

Scientific mode extends the simple mode by adding the following:

asin arc sine function

acos arc cosine function

atan arc tangent function

sin sine function

cos cosine function

tan tangent function

inverse operator

log logarithm (base 10)

ln natural logarithm (base e)

x! factorial function

+/- negation operator

square function (power of 2)

raise-to-the-power-of function (arbitary power)

square root function

root function (arbitary root)
