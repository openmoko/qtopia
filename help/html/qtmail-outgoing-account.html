Outgoing Account

Enter the details for your outgoing mail using one of the Input Methods.

Account name enter your email account's username.
Your name enter your real name.
Email enter your email address.
SMTP server enter the email server name provided by your ISP.
SMTP port default value is 25 (You normally do not need to change this).
Use Signature Select this box if you want to append a signature to your outgoing mails.
- Tap Set... to create a signature.
Default Server Check this box if this is the normal outgoing account to be used for sending messages.

If you have not already done so, set up the Incoming Account.
