New Event

The New Event screen provides several fields that can be used to quickly add information about an event.

For description and location either re-use previous descriptions and locations recorded in the drop-down list or enter new ones.

To assign a category to an event:

  • Use the drop-down list or the '...' button to access the Edit Categories screen.

Note: You can also set New Event Times, and Event Alarms. You can even make it a Repeating Event.
