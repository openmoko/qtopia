Moving around Screens

To move up or down a screen: use the / buttons to move the highlight.

To move from one tab to another: use the / buttons.

To turn a check-box on or off: highlight it and press Select.

To select from a combo-box: highlight it and press Select to activate, then use / to select.

To operate a button: highlight it and press Select.

To edit a text field: highlight it and press Select to enter edit mode. Edit the text and then press Select again to leave edit mode. While in edit mode, the /// buttons will navigate within the text rather than between fields.
