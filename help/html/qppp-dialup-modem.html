Modem

Complete the form, entering the information using one of the Input Methods.

AT-dial - the AT-dial dial code supplied by your ISP, e.g. ATDT.
Speed - your modem speed.
Flow Control - tick the box if you want Hardware flow control.
Wait Time - use the default 5 seconds, or increase the time if you have problems connecting.
Demand Dialing - select your preferred connection option and idle timeout.
