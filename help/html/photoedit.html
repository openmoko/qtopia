Pictures

The Pictures application allows you to view, edit, delete, send and print pictures.

To view an image: highlight the image and press View. tap the image.

To edit an image: select the image. Press Options and select Edit.

To delete an image: select the image. Press Options and select Delete.

To send an image: select the image. Press Options and select Send.

To print an image: select the image. Press Options and select Print.

To edit image properties: select the image. Press Options and select Properties. For more on properties go to Picture Properties.

To view a slide show of the available images: highlight the first image for the slide show. Press Options and select Slide Show. For more on slide shows go to Picture Slide Show.

To view pictures by type: Press Options and select View Type....

To view pictures by category: Press Options and select View Category....
