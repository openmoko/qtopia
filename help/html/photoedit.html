Pictures

This image tool allows you to view, edit, delete, and send pictures.

To view an image: tap the image or highlight the image and press View.

To edit an image: select the image. Press Options and select Edit.

To delete an image: select the image. Press Options and select .

To send an image: select the image. Press Options and select .

To edit image properties: select the image. Press Options and select . For more on properties go to Picture Properties.

To view a slide show: highlight the first image for the slide show. Press Options and select . For more on slide shows go to Picture Slide Show.
