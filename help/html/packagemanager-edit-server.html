View/Edit Server

  • Use / to view the Downloads tab.
  • Press Options and select Edit Servers. A list of servers will be shown.
  • Highlight the desired server, press Options and select View/Edit.... A dialog will show the server details.
  • When you have finished entering/viewing details press the to save any changes and return to the server list.
  • Press again to return to the Downloads tab. Note: If there is only a View menu option for a server, then it is not a user added server and cannot be edited.
