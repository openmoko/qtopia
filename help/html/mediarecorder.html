Voice Notes

Voice Notes allows you to make sound recordings, such as voice memos.

To start recording: select New, then Record.

To stop recording: press Stop.

To play back: select a recording from the list.

To delete: choose Delete from the Options menu.

To change settings: select New, then Settings from the menu.

Voice notes are stored in Documents in the Recordings category and can be played back using the Media Player application or by selecting a recording in the Documents list.
