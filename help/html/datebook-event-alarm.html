Event Alarm

If you want a reminder of the event, select either Silent Alarm or Audible Alarm from the alarm combobox, and the number of minutes before the event you want the alarm to go off.

The alarm can be Silent (a message box will appear when the alarm goes off), or Audible (a bell will sound).

If you want the event to be repeated, select how often (Every Day, Every Week or Every Year) from the Repeat box. To set an event to repeat on a different schedule, select Other. The Repeating Event dialog will pop up for you to complete. Repeating events can go forever, or you can specify an End Date.
