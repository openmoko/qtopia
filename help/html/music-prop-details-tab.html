Music Properties - Details Tab

File: name of the file.

Path: directory path of the file.

Size: of the video file in bytes.

Type: MIME type (read only).

Date: last modification date.

Length: playing time in minutes:seconds.

Audio: audio file format.

Rate: data rate in baud. 0 means variable or unknown.

Chans: number of channels.

Freq: sampling frequency (related to quality).
