Google CalendarTM Accounts

This dialog allows you to add or edit account information for a Google Calendar account. This could be a private calendar that you have previously created, or it may be a public calendar that someone else has created and made available.

  • Email: enter the email address of the calendar account.
  • Password: enter the password for the calendar account, if known. Note that a password is required to view private calendars, but not public calendars.
  • Access: specify the level of access to this calendar account. This can be one of:
    • Private Full - complete event data will be synchronized. A password is required for this level of access.
    • Public Full - complete event data will be synchronized. The calendar account must be shared with everyone.
    • Public Free/Busy - only free/busy data will be synchronized. The calendar account must be shared with everyone.
