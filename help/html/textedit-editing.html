Editing Note

To edit an existing note:

  • Use to highlight the note title and press Select to display the note.
  • Use to position the cursor and to backspace and delete the text, then use the keypad to enter new text.
  • Press OK to accept new text.
  • Press to save the task and exit.
  • From the Options menu, you can Find text, Print the text, and change the display Settings,
