Editing Text

You can select text which can be, cut, copied and then pasted.

Tap to return to the list of files.

The File and Edit menus provide the same options as the toolbar buttons.

Tap Zoom in or Zoom out from the View menu to make the text appear larger or smaller. Some documents are best viewed with a fixed-sized font; enable Fixed-width from the View menu to enable this.

Tap Wrap lines from the View menu to toggle word wrapping on long lines.
