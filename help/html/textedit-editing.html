Editing Text

The File and Edit menus provide commands for creating new documents, opening existing documents displaying a documents properties, cutting, copying and pasting text, and a find command to search for text. These commands are all available on the toolbar.

When editing a document selected text can be, cut or copied and then pasted to a new location in a document.

To return the the list of files to open another file:
- Tap File, then Open

To reduce or enlarge the appearance of text:
- Tap View, then Zoom in or Zoom out

Some documents are easier to view with a fixed-sized font:
- Tap View, then Fixed-width to enable this.

Wrap Lines cause long lines of text to be wrapped in the display so they can be seen fully.

To turn Wrap Lines on and off:
Tap View, then Wrap lines
