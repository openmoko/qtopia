Editing Note

The File and Edit menus provide commands for opening existing documents into edit mode, displaying documents properties, cutting, copying and pasting text, and a find command to search for text.

These commands are all available on the toolbar. When editing a document selected text can be, cut or copied and then pasted to a new location in a document.

To edit an existing note:

  • Tap the note, then Edit.
  • Modify the content.
  • Open a new note or exit to save your changes.

To edit an existing task:

  • Use to highlight the note title and press to display the note.
  • Use to position the cursor and to backspace and delete the text, then use the keypad to enter new text.
  • Press to accept new text.
  • Press to save the task and exit.
