Tasks

Tasks is used to create and maintain a list of prioritized tasks. You can add, edit, delete, view and send tasks.

Tasks are displayed in a list, with the description of the task and some additional information about the task. The additional information can include the due date of the task, if it has one, the date the task was completed, or the current status of the task. Several icons may be displayed for each task, including:

  • shows that a task has been completed.
  • shows that a task has high priority (priority 1 or 2).
  • shows that a task repeats.
  • shows that a task has an audible reminder.
  • shows that a task has a silent reminder.

Common Task Actions

  • To view or edit a task: highlight and select a task, and then to edit choose Edit from the menu.

  • To add a task: choose New Task from the menu.

  • To mark a task as completed: choose Mark Task Complete from the menu.

  • To mark a task as incomplete: choose Mark Task Incomplete from the menu.

  • To only show tasks that are not complete: choose Hide Completed Tasks from the menu, and Show Completed Tasks to show them again.

  • To only show tasks in certain categories: choose View Categories... from the menu.
