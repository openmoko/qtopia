Show Documents by Type

%%PDA%%

The Show Documents by Type drop-down list is located in the lower left corner of the Document screen.

It is used to limit the list of documents being displayed to those of a particular type.

@@PDA@@

To show only text documents:

  • Tap the Show Documents by Type drop-down list, then Text.
  • A list of all text documents will be displayed.

To show all document types:

  • Tap the Show Documents by Type drop-down list, then All types.
  • A list of all documents will be displayed.
