Plugin Manager

The Plugin Manager controls which plugins can be loaded by the Qtopia system and application.

It is not usually necessary to disable plugins, however it is desirable in some situations:

  • Disabling Taskbar Applets can leave more free space on the Taskbar to show running applications.
  • If an application or Qtopia crashes after installing new software that includes plugins, the Plugin Manager can be used to disable the faulty plugin. This is particularly useful if Qtopia crashes and restarts in Safe mode.
