Plug-in Manager Help

The Plug-in Manager controls which plug-ins can be loaded by the Qtopia system and its applications.

It is not usually necessary to disable plug-ins, however it is desirable in some situations:

  • Disabling Taskbar Applets can leave more free space on the Taskbar to show running applications.
  • If an application or Qtopia itself crashes after installing new software that includes plug-ins, the Plug-in Manager can be used to disable the faulty plug-in. This is particularly useful if Qtopia crashes and restarts in Safe mode.
