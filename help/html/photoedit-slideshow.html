Picture Slide Show

You can adjust the display options before starting the slide show.

Slide length: adjust this to change the duration of each slide.

Display names: tick this to display the name of each image during the slide show.

Loop through: tick this to restart once the slide show reaches the last image.

Note that the slide show begins from the highlighted image. If looping will it will restart from the very beginning (not from the highlighted image).

Press Back to exit the slide show once it has started.
