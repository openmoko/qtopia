Package Servers

Tap the Upgrade all button to upgrade all the installed software from the Package Servers.

Tap the switch icon to launch the Package Servers dialog. Tap New, then tap the Name edit box and enter a name using one of the Input Methods. Now tap the URL edit box and enter the URL for the package server using one of the Input Methods. You can use any number of package servers.

To remove a server from the list, tap its name in the Servers list, then tap Remove.

Tap to finish.
