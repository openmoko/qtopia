Stopwatch

Tap the Start button to start the stopwatch. While the stopwatch is running, the Start button will be replaced with Stop, which can be used to pause the stopwatch. Tap the Reset button set the time back to zero.

Split times can be recorded by tapping the Lap/Split button. The Split time is the time total time elapsed up until split was pressed; the Lap time is the difference between the previous split and this one. Use the spinboxes to view each recorded split time.

The stopwatch can also be controlled by using a keypad (if available). The Return button is used to start and stop the stopwatch, and the Up and Down buttons can be used to view the previous and next lap times.
