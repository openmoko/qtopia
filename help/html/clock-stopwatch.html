Stopwatch

The top display shows total time in analog and digital format.

The number next to the arrow buttons shows current lap number.

Split shows end time for this lap, Lap shows duration of this lap.

Press Select to start/stop timer.

Press the # key to mark the lap time, or to reset the timer to zero.

Press the Start button to start the timer. The button will change to Stop and pressing it will stop/pause the timer.

While the timer is running, press the Lap/Split button to mark the lap time.

While the timer is paused, press the Reset button to reset the timer to zero.

When stopped/paused, use the / buttons to view times for previous laps.

Note: If the system time is modified while the stopwatch is running the stopwatch time elapsed may no longer be valid.
