IP PAN

Complete the form by entering the required information into the fields, each of which will require IP addresses, which should fit into a context upon which all participating PAN devices have agreed on.

Select an option by pressing Select to tick/un-tick options.

The selection box in the top determines to which configuration the settings below apply to. New configurations can be created/removed on the account screen.

Autm. IP (DHCP) tick to use DHCP to discover IP details (not available in gateway mode)

If you want to specify the IP details manually un-tick the DHCP check box.

IP Address: enter your IP address.

First DNS: enter preferred DNS.

Second DNS: enter 2nd choice DNS.

Broadcast: enter the broadcast address.

Subnet mask: the subnet mask filter.

Gateway: enter the address of your internet gateway (not available in gateway mode).
