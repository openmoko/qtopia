Proxies Tab

A proxy is a kind of gateway that acts like a server. When viewing frequently requested sites, using a proxy server can speed the response time.

This setting defines how your Qtopia device connects to the internet. Highlight and select to view the list of options.

No proxies: connect directly to specified site.

Auto configure: to make the Qtopia device configure proxies automatically. Select a configuration from the Config drop-down box.

Set proxies: to set proxies manually. Enter the host name or IP address and port number for the HTTP and FTP proxies you wish to use. These values should be supplied by your ISP.

  • HTTP: enter address of proxy and port number (default 8080).
  • FTP: enter address of proxy and port number (default 8080).
  • No proxies for: do not use proxies for this site(s).
