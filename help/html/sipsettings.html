VoIP

Shows the current configuration of VoIP networks and allows the user to modify it.

Identity: create or edit a VoIP identity.

  • User ID: user login id to the SIP server
  • Host: address of the SIP server
  • Name: full name of the user
  • Password: password for the SIP server
  • Auto Register: automactic registration with the SIP server when the phone starts

Proxy: set the proxy SIP server information.

  • Host: address of the proxy SIP server
  • Port: port number on the proxy SIP server to connect to
  • User ID: login id to the proxy SIP server

Advanced: configure the advanced options.

  • Registration Timeout: timeout in seconds for registration request
  • Presence Timeout: timeout in seconds for presence subscription
  • Codec: preferred codec for the audio
