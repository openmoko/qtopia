Setting Date/Time

Date/Time

The Date/Time application is used to set the local time zone, and the current time and date for your .

Time tab

  • Automatic: the time zone, time, and date are updated from the phone network, where available. This setting can be On, Off, or it can Ask what you want to do whenever the time or zone changes.
  • Time Zone: set the current time zone.

  • Time, Date: set the current local time and date.

Format tab

  • Time format: choose either 12-hour (1 pm) or 24-hour (13:00) time format.

  • Week starts: start day for the weeks in the Calendar month view.

  • Date format: choose either the format standard for your chosen input language, or one of the other listed formats.
