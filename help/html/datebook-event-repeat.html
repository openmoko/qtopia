Repeating Events

Repeating events can recur Every Day, Every Week, Every Year. Select the appropriate frequency from the Repeat list.

To set an event to repeat at another interval, e.g. every 2 weeks, select Other..., then specify how the event should repeat. Repeating events can continue forever, or you can specify an End Date. Standard options are:

  • Day: repeat every day, or every 2, 3, etc days - including weekends.
  • Week: repeat every week on the same day(s).
  • Month: repeat every month, or every 2, 3, etc months.
  • Year: repeat each year, or every 2, 3, etc years.
