Software Packages

Software Packages is used to manage software installed on your device. It lets you browse through applications available for download on one or more package servers and to install and uninstall those applications.

How to:

  • Add a server
  • Download and install application from a server
  • Uninstall an application
  • Delete a server
  • View/Edit server details
  • View application details

If the device is SXE enabled you can

  • Re-enable a disabled package
