USB Connection Mode

The USB Connection Mode window allows you to choose how your communicates with a computer over the USB interface.

By default the USB Connection Mode window is displayed when you connect your to a computer via a USB cable. If the Don't ask again item in the Options menu is checked the USB Connection Mode window will not be displayed. Instead the highlighted connection mode will be activated immediately.

To select the connection mode to activate:

  • Tap the desired connection mode to select it.
  • Tap Back to activate the selected connection mode.
  • Use to highlight the desired connection mode.
  • Press to select the connection mode.
  • Press Back to activate the selected connection mode.
