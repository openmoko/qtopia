Outgoing Account

For your outgoing mail to be correctly processed you must enter the details for the Outgoing Account. Select Account Settings and select either Add account... or an existing account from the drop-down list.

  • Acct: account name to show in the list of mailboxes on your phone.
  • Name: the name to show on your outgoing emails.
  • Email: the email address for your outgoing emails.
  • Server: IP address of server (provided by your ISP).
  • Port: default value is 25.
  • Signature: select to append a signature to your outgoing emails.
    • Select Set... to create a signature.
  • Default mail server check this box to make this your default outgoing mail account.

If you have not already done so, set up the Incoming Account.
