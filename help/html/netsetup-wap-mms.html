MMS

MMS Help

Used to define MMS server details. Ask the internet provider for details.

Complete the form by entering the information into the following fields:

MMS server (MMSC): MMS server URL. The URL has the format <server>:<port> ( e.g. http://mms.provider.net:110 )

MMS expiry: defines the expiry time for an MMS when receiver can not receive the message.

Sender visibility: allows the following values:

  • Default (as set by provider)
  • Show (attaches your sender ID to every MMS)
  • Hidden (no sender ID is attached to an MMS)

Delivery report: tick this check box to request a delivery report. A delivery report is generated when the receiver reads this MMS.
