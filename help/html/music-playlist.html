Creating Playlists

A playlist is a user specified list of tracks that can be played. Songs can be listed in any order.

%%PHONE%%

Use categories to create a playlist for your phone.

First create a new category with the name of the playlist.

Then edit the properties of each music file and assign them to the new category.

Now you can select View Category... and display a list of all those tracks in your 'playlist'.

Select the first in the list to begin playing from the playlist. Music will automatically play the next in the list when the previous is finished.

@@PHONE@@ %%PDA%%

To add a single entry to the playlist:

  • Select the song and tap .

To remove an entry from the playlist:

  • Select the Playlist tab, select the song to remove, and tap .

The play order can be modified by selecting a song in the Playlist tab and using / to move the entry up or down in order. The stylus can also be used to drag items up and down the order of the list.

Use Clear from the playlist menu to remove all entries from the playlist.

The Add all Music and Add all from the playlist menu are used to add files to the playlist from the list of available files. Category and location fields located at the bottom of the list can be used to filter the Music lists.
