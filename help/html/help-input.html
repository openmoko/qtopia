Input Methods Help

Input is via Handwriting. Simply write letters on the display, or use one of the special strokes for other input.

Input is via the predictive keyboard. Type letters by pressing them, and accept the current word by pressing on it or quickly stroking the keyboard left-to-right. Backspace by stroking right-to-left. Access capital letters, numbers, and symbols by stroking the board up or down.

Input is via the keypad. Press the key corresponding to the letter you wish to type (eg. press 3 for "D", "E", or "F").
