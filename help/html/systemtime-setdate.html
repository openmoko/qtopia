Setting the Date

To set the date:

  • Launch Settings by using to focus on and pressing Select to select.
  • Use to scroll down the list and Select to select Date.
  • Use the down arrow to reach the fields.
  • Press Select and choose the time zone (T.Z.).
  • To select the date press Select to launch the calendar.
  • Use arrow to highlight the correct date then press Select.
  • To set the day of the week calendar begins on, highlight and select the Week starts field. Choose the day and press Select to select it.
  • Change the date format to suit by selecting from the Date format field menu.
  • Press to save and exit.
