Edit Channel

Channels are the difference frequency spaces that operators use to send messages. The availablility of channels are subject to operator's offerring. Please consult your network operator for available channel list.

Channel 50 is a standard channel that is used for area code of the current cell. This channel cannot be removed.

Use this dialog to edit channel information and how to receive the message.

  • Title: Change the title of the channel.
  • Channel: Change the channel other than 50.
  • Language: Select languages in which the message is written if available.
  • Display Mode: Select how you want to display the message when it arrives.

Display Modes:

  • Home Screen: The message will appear on the idle screen and disappears after a short period of time.
  • Pop-up Message: The message will appear in a pop up message box and stay until it gets cleared by the user. If there is a subsequent message arriving before the previous one is cleared, the previous one will be replace with the new one.
