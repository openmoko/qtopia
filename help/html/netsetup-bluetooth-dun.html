Bluetooth Dial-up Network (DUN)

Qt Extended provides a Bluetooth dial-up networking client. The device can connect through other Bluetooth devices to the internet. The partner device must offer a gateway to a dial-up provider. The actual internet connection is established via e.g. GPRS and then forwarded to the DUN client.

The Bluetooth dial-up dialog has three sub-menus to setup a Bluetooth dial-up connection:

  • Account
  • Dial parameter
  • Advanced

Select a menu option to open the corresponding edit dialog, or press Back to save and exit the service details.
