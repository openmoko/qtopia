Month View

Month View shows a whole month - events show as colored squares, all-day events show as a shaded day, multi-day events show as a line.

To view a day in detail: use to highlight the day, then press Select or the * key.

To view a different day:

  • use / to see next/previous day.
  • use buttons 0-9 to enter a date (in your preferred format).
  • to get back to today, select Today from the menu.

To change the start day for the week: use Date / Time in Settings.
