Message Types

The Messages application supports multiple types of messages: SMS, MMS and Email.

SMS - Short Message Service

A service for sending text-only messages of up to 160 characters, to other mobile phones using GSM communication.

MMS - Multimedia Messaging Service

MMS allows you to send and receive messages that include images and audio in addition to text.

Qt Extended supports the SMIL (Synchronized Multimedia Integration Language) framework for presenting multiple media elements in a slide-show format.

Email - electronic mail

Email allows you to send text messages to email recipients, including personal computers and other communications devices. Email messages can contain other types of media as attachments.

EMS - Enhanced Messaging Service

Used to send and receive ring tones and emotive icons, as well as combinations of simple media to and from EMS-compliant handsets.
