Write Mail

This dialog is invoked by tapping to reply, or to create a new email.

Enter Email Addresses, then tap the Subject edit box, and enter a subject using one of the Input Methods. Enter the text of the message in the empty area below the Subject edit box using one of the Input Methods.

The Field menu is used to select which fields are automatically available. The default settings display To, Subject and CC fields. If you seldom use a CC field you can choose to hide it by deselecting it in the Field menu. Similarly you can set whether the BCC and From field should normally be displayed. The From field displays which outgoing account this message belongs to. Tap the From box to select another account (Applicable only if you have more than one account defined).

Tapping will toggle the display of all fields on or off (to give you more room to write the message or verify that you have filled out the intended fields).

Tapping will store the mail in the Outbox folder, while tapping store the mail in the Drafts folder.

Tapping the close button will close the dialog. If you were editing a mail your changes will be automatically saved. New mails will always be stored in the Drafts folder (unless you press ).

Tap in the main view to send all the email in the Outbox. All mails that were successfully sent will be automatically moved to the Sent folder.
