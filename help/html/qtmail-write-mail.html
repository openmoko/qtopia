Writing Messages

Begin composing by selecting New message from the folder list, or press Options while viewing a message list, and select New.

Select the type of message you wish to compose from the list of options.

Composing

SMS: the total number of characters remaining and the number of SMS messages to be sent is displayed in the top right of the display.

MMS: see Composing MMS Messages.

Email: to add attachments to your message, see Message Attachments.

After entering the text of your message, press Next to proceed to the details page.

If you are not yet ready to send your message, you can save it to edit later by pressing Options and selecting Save in drafts.

To abandon the message completely, press Options and select Cancel.
