Writing Email

This dialog is invoked by tapping to reply, or to create a new email.

  • Enter the email address.
  • Tap the Subject field and enter a subject.
  • Enter the text of the message in the empty area below the Subject field.
  • Tap the From field to select another account (only applicable if you have more than one account).
  • Tap in the main view to send all the email in the Outbox. All mails that were successfully sent will be automatically moved to the Sent folder.

The Field menu is used to select which fields are automatically available. The default settings display To, Subject and CC fields. If you seldom use a CC field you can choose to hide it by de-selecting it in the Field menu. Similarly you can set whether the BCC and From field should normally be displayed. The From field displays which outgoing account this message belongs to.

Composing a message uses several pages. Press , then at each step to move to the next page.

To delete message: choose Cancel from the menu.

To store message to work on it later: choose Save in Drafts from the menu.

Body: enter the text for the message, then press to confirm.

SMS: enter the mobile number if it is an SMS.

To: enter the email address if it is an email.

CC, BCC: send copies to others by adding them in as CC, or as BCC to hide their numbers from the main recipient.

Attachments: you can attach one or more files to an email.
