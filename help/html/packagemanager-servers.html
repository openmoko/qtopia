Package Servers

To add a new server:

  • Tap New, then tap the Name field and enter a name.
  • Tap the URL field and enter the URL for the package server. You can use any number of package servers.

To remove a server from the list:

  • Tap its name in the Servers list, then tap Remove.

To add a new package server:

  • Press and then select Servers->Edit Servers... from the menu.
  • Press and then select New from the menu.
  • Enter a name and URL for the server.

To remove a package server:

  • Press and then select Servers->Edit Servers... from the menu.
  • Use to select a server.
  • Press and then select Remove from the menu.
