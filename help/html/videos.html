Videos Help

The Videos application is used to play video and mutimedia files.

%%PDA%%

To play a video select the file and tap . Any available properties about the video can be displayed by selecting the entry and tapping . There is also a playlist tab where selected files can be added or removed to organise the playlist.

Tap File to access the menu and read about the options for playing files.

Details about the video controls and loading new files are also available.
