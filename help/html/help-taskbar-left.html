Left of the Taskbar

The Q-Menu appears at the far left of the Taskbar.

To start a program using the Taskbar:

  • Tap the icon.
  • Then tap Applications, Games, or Settings.
  • Then the application you want to use.

To the right of this menu is the Input Methods icon. Tap this icon to pop-up the current input method, or tap on the menu arrow further to the right to select which input method you want to use.
