Message Address List

The Message Address List combines entries from Contacts with addresses collected automatically from Email as you send and receive messages.

To address a message to a contact listed in the Message Address List:

  • Tap the ... button to the right of the To, CC or BCC fields.
  • Tap the name of each contact you want to send the message to.
  • Tap to close the list and return to the message.
