Send/Receive Files

The Send/Receive Files window displays information about any files you are currently sending or receiving over Bluetooth or Infrared. Bluetooth. Infrared.

When you are sending or receiving a file, a progress bar will be displayed to show the progress of the file transfer. Once the transfer is complete, will appear next to the file. You can then press to open the file.

To stop a file transfer, press and select "Cancel transfer".

When the Send/Receive Files window is closed, the file transfer list will be cleared of all completed transfers.

You can use the Multitasking and Task Manager features to switch to another application while a file transfer is in progress. The Send/Receive Files window will be displayed in the Task Manager so that you can return to the window to check the progress of the transfer.
