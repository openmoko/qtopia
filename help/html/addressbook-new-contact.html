Add a Contact

Press and select New. This displays the input record for the new contact.

Enter as much information as you wish (you can always add more later).

To store this contact's phone numbers on the SIM (rather than the phone itself), select the Store on SIM option from the menu.

To create a New Entry:

  • Tap Contact, then New.
  • Enter the details for the new contact into the Edit Contact Details dialog.
  • Tap to save the entry or to cancel.
