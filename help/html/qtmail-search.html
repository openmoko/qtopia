Search for Messages

The message search feature can be used to search for messages of a specific type or those containing specific text.

Press Options and select Search from the menu.

The following search conditions can be specified:

In: which folder to search within.

Status: message status flag settings to search for.

From: text to match against the From field of each message.

To: text to match against the To field of each message.

Subj: text to match against the Subject line of each message.

Body: text to match against the body text of each message.

After: if ticked, match only messages after the specified date.

Before: if ticked, match only messages before the specified date.
