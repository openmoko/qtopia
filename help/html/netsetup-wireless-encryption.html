Encryption

Complete the form by entering the required information into the fields.

The first step is to select the network to be configured.

Encryption

  • None (unencrypted operation without authentication)
  • Open
  • Shared key
  • WPA-PSK
  • WPA-EAP (supports TLS, TTSL, PEAP)

Most encryption keys can be a pass-phrase or a hexadecimal key. The user interface will present the appropriate menus where such a choice is required.

WPA-PSK requires a password with a length of at least 8 characters.
