Launcher Help

The Launcher is the main screen from which you start the Qtopia Applications, Games, Settings and Documents.

To customize a launch page:

  • Select the Launcher settings icon .
  • Select the Category drop-down list, then select the launch page to modify, e.g. Games.
  • Modify the launch page properties as required.
  • Select to save your changes or to cancel.
  • Select to save your changes or to cancel.

The following aspects of the launcher can be customized:

  • Background
  • Font
  • Icon Size
