Launcher

The Launcher is the main screen from which you launch the Qtopia Applications, Games, Settings and Documents.

Icons, Background and Fonts used in the launcher can be customized.

To customize a launch page:
- Tap the Launcher settings icon
- Tap the Category dropdown list, then select the launch page to modify, e.g. Games
- Modify the launch page properties as required
- Tap to save your changes or to cancel.
