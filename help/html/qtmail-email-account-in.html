Incoming Settings

You can configure an email account to send or receive email, or both.

For each account, you must enter:

  • Name: the name to show for this account in the list of mailboxes on your .

Incoming Settings

To receive email you must enter the Incoming Settings for your account. Select Account settings... and select either Add account... or choose an existing account from the drop-down list.

You must enter:

  • Type: POP or IMAP (check with your ISP as to which account type you have if you are unsure).
  • Username: for this account.
  • Password: to access the account.
  • Server: name provided by your ISP.
  • Port: usually 110 for POP and 143 for IMAP
  • Encryption: turn on for SSL encryption; normally requires a Port number othen than default.
  • Delete mail: turn on to delete mail from the server once it has been emptied from the Trash folder of Messages.
  • Skip larger: turn on to leave the content of large messages on the server for downloading on demand; set largest message size to be automatically retrieved, in kilobytes.
  • Push enabled: turn on to enable IMAP IDLE based push email, only supported for IMAP accounts.
  • Interval: turn on to have the account automatically check for new messages periodically; set the interval between checks, in minutes.
  • Disable when Roaming: turn on to disable interval checking during network roaming.

IMAP Settings

For IMAP accounts only, you can configure:

  • Base folder: to retrieve only those messages stored within a particular folder from your account, enter the folder name here.

For many of these fields, you will be able to simply use the default setting.

To send email, you must also set up your Outgoing Settings.
