Incoming Account

To receive email you must enter the details for the Incoming Account. Select Account Settings... and select either Add account... or choose an existing account from the drop-down list.

You must enter:

  • Type:POP or IMAP (check with your ISP as to which account type you have if you are unsure). If this account is used only for synchronizing with the desktop choose Synchronized.
  • Username: for this account.
  • Password: to access the account.
  • Server: name provided by your ISP.
  • Port: usually 110 for POP and 143 for IMAP
  • Delete mail: turn on to delete mail from the server or PC once it is stored in your .
  • Skip larger: turn on to leave large messages on the server and save on download time; set largest message size in kilobytes.

For many of these fields, you will be able to simply use the default setting.

For further details, see POP & IMAP Settings

To send email, you must also set up your Outgoing Account.
