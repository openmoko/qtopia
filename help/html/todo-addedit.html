Creating, Editing and Deleting Tasks

To add a new task:
- Tap Task, then New Task
- Enter the details of the task in the Edit Task screen using the Input Methods available
- Tap to save the task, or to cancel.

To edit an existing task:
- Tap the task, then Edit
- Modify the details of the task in the Edit Task screen
- Tap to save your changes, or to cancel.

To mark a task complete:
- Place a tick in the completed check box beside the task

To delete a task:
- Tap the task to delete
- Tap Task, then Delete
- A message box will appear asking you to confirm the deletion
- Tap YES to continue, or NO to cancel.

These commands are also available as buttons on the toolbar.
