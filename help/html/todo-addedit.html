Adding and Editing Todo Items

To add a new item tap . To edit an existing item, tap the item, then tap .

Enter a short description of the item in the Description field. Select a Priority and Status. If applicable to the task, check Due, and choose the appropriate dates from the Due date, Started Date and Completed Date.

Change the associated category for this task by tapping the Category button.

More detailed information about the task can be entered on the Notes tab.

Tap to add the item, or tap to cancel.
