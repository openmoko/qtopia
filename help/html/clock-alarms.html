Alarm Clock

To set the alarm:

  • Check the Enabled box to enable the daily alarm.
  • Set the time the alarm is to be activated.
  • Select the days of the week on which the alarm is to be activated.
  • Set the number of minutes for the snooze function.

When the alarm goes off:

  • The alarm will ring 20 times and then stop.
  • If you set a snooze time you can snooze the alarm by pressing the snooze button.
  • Clicking on the bell also snoozes the alarm.
  • To prevent a snoozed alarm from re-appearing, toggle the alarm off and on.
