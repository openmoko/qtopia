Security

A basic level of security can be achieved by giving your Qtopia device a pass-code number.

To set a pass-code number:

  • Tap Set code.
  • Enter the number you want to use as your pass-code.
  • Tap to continue.
  • Re-enter the number and tap to confirm.

To clear the pass-code:

  • Tap the Clear code button.
  • The Qtopia device will no longer require a pass-code.

To activate the pass-code:

  • Tick the Require code at power-on check-box.

Note: Once a pass-code has been set, you will be required to enter the code number whenever the Qtopia device is switched on or wakes up.

Use the security settings to control pin-codes and set the TCP/IP network with which to synchronize the device.
