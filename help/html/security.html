Security

Qtopia phone can be protected by PIN numbers at two levels: the SIM card level and the phone device level. To change either of the locks you must know the PIN.

Use and to select the type of lock (SIM or Phone).

Check Enable protection to activate or uncheck to deactivate the protection and enter PIN code when prompted.

Press Change PIN button to change PIN.

Auto key lock can be set from Homescreen settings application.
