Security

A basic level of security can be achieved by giving your Qtopia device a passcode number.

To set a passcode number:
- Tap Set code
- Enter the number you want to use as your passcode
- Tap to continue
- Re-enter the number and tap to confirm

To clear the passcode:
- Tap the Clear code button
- The Qtopia device will no longer require a passcode.

To activate the passcode:
- Tick the Require code at power-on tick box.

Note: Once a passcode has been set, you will be required to enter the code number whenever the Qtopia device is switched on or wakes up.

You can restrict the sync access to the Qtopia by modifying or selecting the subnet in the Accept sync from network list. You will need to specify either the IP address of a single machine, or specify a network by entering an IP address with a slash (/) and the number of bits (e.g. 10.0.0.0/8 means all addresses from 10.0.0.0 to 10.255.255.255).
