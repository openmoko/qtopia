Security

A basic level of security can be achieved by giving the Qtopia device a passcode number.

Tap Set code to set the number. You must enter the number, then tap OK. Confirm the new passcode by entering it again followed by OK.

Tap Clear code to purge the passcode. The Qtopia will no longer require a passcode.

Check the Require code at power-on checkbox to activate it. Once set, you must enter the code number whenever the Qtopia is switched on or wakes up.

Machines which are allowed to sync to Qtopia can be restricted. Select or modify the subnet in the Accept sync from network list. You can either specify the IP address of a single machine, or specify a network by an IP address with a slash (/) and the number of bits (eg. 10.0.0.0/8 means all addresses from 10.0.0.0 to 10.255.255.255).
