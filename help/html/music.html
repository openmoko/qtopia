Music Help

The Music application is used to play music and other audio files.

%%PDA%%

To play an audio file select the file and tap .

The properties information of the file can be displayed by selecting the entry and tapping .

There is also a playlist tab where selected files can be added or removed to organise the playlist.

Details about the music controls and loading new files are also available.
