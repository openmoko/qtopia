Text Editor

The text editor is used to create and view text files. Once open it displays a list of available files.

To open a file:
- Tap the name of the file you want to open

To create a new file:
- Tap New Document

Once you've opened or created a file the text editor will provide menu and toolbar for working with the file.

To enter text:
- Use one of the Input Methods.

More information is available on Editing Text
