Notes

Notes are short pieces of text that you can edit and attach to emails. The application can be used to add, edit, and delete notes. Once open it displays a list of available files.

To create a new note: simply select New from the file list, then enter and save the text.

To edit an existing note: just select that note from the file list.

To delete a note: highlight the note you want to delete and select Delete from the Options menu, then press Yes to confirm the deletion.

To change the properties of a note: highlight the note and select Properties from the Options menu.
