Calendar View

To switch between Calendar views:

  1. Press and select one of the following from the menu:
    • Today
    • Month
  2. To go directly to today's date in any view, press and select Today from the menu.

To switch between calendar views:

Tap View, then one of the following:

Day

Week

Month

To go directly to today's date in any view, tap View, then Today.

These commands are also available as buttons on the toolbar. You can change the default Start Time for the Day and Week views using Calendar Settings.
