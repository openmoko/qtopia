Media Controls

For both videos and music files, there are the following controls: previous, rewind, stop, play, pause, fast forward, next and return to playlist.

Play / Pause
When playing, the play button is used to pause, and when paused, pressing it again resumes playing from when it was paused. The hardware select button on the device is also able to be used for this.

Stop
If stop is pressed, the position in the file is reset to the beginning and pressing play will begin playing from the start.

Rewind / Fast Forward
The rewind and forward buttons are for seeking through a media file. It is also possible to use the hardware left and right buttons for this. To use the hardware buttons, hold down the button in the direction you wish to seek, then release the button when it has seeked to the desired position and the player will continue playing from there. It is also possible to use the position slider at the bottom of the window to tap on it to position the playback instantly to a given position. For instance if there is a good scene at the end of a video file you wish to jump to, tap on the slider towards the right end of the slider to be taken near to the end of the video.

Previous / Next
The previous and next buttons are for moving through the playlist. The hardware buttons can also be used for this if you prefer using the left and right buttons with a brief tap.

Close
Tap to stop playing and return to the list of media files. The hardware cancel button can also be used to return to the media file list view.

Fullscreen
For movies files only, tapping on the movie while it is playing will change the mode. If the movie is playing in fullscreen mode, the player will be taken out of fullscreen mode. Tapping on the movie again can be used to change the view mode back to fullscreen mode.
