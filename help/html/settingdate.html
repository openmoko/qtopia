Setting the Date

To set the current date:
- Tap the Date dropdown list to select the current date from the popup calendar.

To set the day of the week calendars begin on:
- Tap the Weeks start on dropdown list, then either Sunday or Monday.

To select your preferred date format:
- Tap the Date format dropdown list.

Tap to confirm the new date and time, or tap the button to cancel.

See also Setting the Time.
