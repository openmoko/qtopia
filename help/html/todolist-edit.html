Edit Tasks

To edit an existing task:

  • Use to highlight the Description and press View to display the task.
  • Press Options and select Edit from the menu.
  • Use to and Select to select a field.
  • Use to backspace and delete the text, then use the keypad to enter new text.
  • For fields with a list of options, simply select another.
  • Press OK to accept new text in a field or use to move to the next.
  • Press to save the task and exit.

To mark a task complete:

  • Use to highlight the check-box.
  • Press Select to place a tick in the box.
  • To add completion notes, highlight the Description and press View to display the task.
  • Press Options and select Edit from the menu.
  • Use and Select to open the Notes tab.
  • Use the keypad to add the completion notes.
  • Press to save the task and exit.
