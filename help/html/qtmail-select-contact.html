Contact List

A message recipient field can contain a list of contacts, each of whom will receive a copy of the message. The selected contacts are displayed in a list to which can add or remove selections.

To add another recipient to your list, press Options and select New recipient. To remove an existing recipient from your list, highlight the relevant address and press Options and select Remove recipient.

Select Contact

The Select Contact dialog allows you to select any contact from your address book, if that contact has an address to which the current message can be sent. The list can be filtered by typing the first characters of the name you want to find.

Select a contact to add it to the recipient list. If the contact you have selected has more than one address that the message could be sent to, the Addresses dialog will prompt you to choose which address to use.

When the list contains all your desired addresses, press Back to return to the Details page.
