Task Editing

To create a new task, or edit an existing task, several tabs with the details of the task are presented:

Task: This tab has essential task information on it.
  • Description: enter a brief description (shown as the title in the list of tasks).
  • Priority: to help you prioritize your tasks:
    • 1 = Very High
    • 2 = High
    • 3 = Normal
    • 4 = Low
    • 5 = Very Low
  • Due: check this option and select a date if the task has a due date; tasks of equal priority will be sorted by due date.
  • Reminder: if you have set a due date, you can choose to have a reminder that the task is due soon. You can choose either a silent reminder or an audible reminder, and when you want the reminder to occur.
Progress: This tab allows you to edit the progress information for a task.
  • Status: to indicate the current status of the task:
    • Not Started
    • In Progress
    • Completed
    • Waiting
    • Deferred
    Completed tasks have a check mark in the list; all tasks can have a percentage complete (except Not Started, which is zero percent complete by definition).
  • Started Date: the date you started the task. This can be set manually, or will be automatically set if you change the task status to anything except Not Started.
  • Completed Date: entered automatically when a task is marked complete.
Recurrence: This tab allows you to set whether a task repeats. This allows you to store tasks like "take out the rubbish", or "create end of month report".
  • Repeat: choose a type of repetition (daily, weekly etc). Depending on the type of repetition, more options may be available to customize the repetition.
  • Frequency: choose how often the repetition will occur. For example, a frequency of 2 and a repetition type of daily means that the task will be due every 2 days.
  • Repeat until: check this option and select a date if the repetitions should stop on or before a specific date.
  • On these days: when choosing a weekly repeat, an option to choose additional days of the week to repeat on will appear. The original due date's day of the week is always selected. This allows you to create tasks that will repeat on every weekday, or on any particular days you specify.
  • Repeat on: when choosing a monthly repeat, you can specify how the day of the month is interpreted. This allows you to specify tasks that recur on certain days of certain weeks (for example, the first Tuesday), or a fixed day of the month (for example, the 17th of each month).
Notes: this tab contains extra information about the task.
  • Category: used to group like tasks.
  • Notes: enter more detailed information for a task.
Navigating the fields:
  • Use to and Select to select a field.
  • Use to backspace and delete the text, then use the keypad to enter new text.
  • For fields with a list of options, simply select another.
  • Press OK to accept new text in a field or use to move to the next.
  • Press to save changes to the task and exit, or Options then Cancel from the menu.
