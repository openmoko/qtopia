Edit Tasks

Use the / to choose Task details or Notes.

You can record and modify the following information:

  • Description: enter a brief description (shown as title in the list).
  • Priority: to help you prioritize your tasks:
    1 = Very High, 2 = High, 3 = Normal, 4 = Low, 5 = Very Low
  • Status: to indicate the current status of the task:
    Not Started, In Progress, Completed, Waiting, Deferred. Completed tasks have a check mark in the list; all tasks can have a percentage complete (except Not Started).
  • Due: check this option and select a date from the drop-down list if the task has a due date; tasks of equal priority will be sorted by due date.
  • Started Date: the starting date can only be set for In Progress tasks.
  • Completed Date: entered automatically when a task is marked complete.
  • Category: used to group like tasks.
  • Notes: enter more detailed information for a task.

Tap to save the task or changes you have made to a task or to cancel.

Press to save changes to the task or then Cancel from the menu.
