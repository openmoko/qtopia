Help Browser

You're using it! The Help Browser is a hypertext help system that works in a similar way to a web browser. Hypertext links are signified by colored text.

To get help within an application tap in the title bar.

Some applications provide short help messages for different parts of their dialogs. Tap and hold in the title bar until the title bar displays the text What's this..., then tap on any part of the dialog.
