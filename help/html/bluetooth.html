Bluetooth FTP Client

The Bluetooth FTP client allows you to transfer files over the bluetooth link between your and other Bluetooth devices that support the Bluetooth FTP profile.

If you're not connected to a device, you can choose to connect to a device by using the Browse action. This will bring up a list of nearby devices that you can connect to. Selecting a device from this screen will attempt to connect to the remote device's Bluetooth FTP service.

Once you're connected, you can transfer a file from the remote device to your local device by highlighting the file and pressing the Select button. tapping on the file.

You can create new directories and put new files onto the remote device by using the Options menu actions. Deleting files or directories can be accomplished by using the Delete menu option. You can also choose to disconnect from the remote device by using the Disconnect menu option.
