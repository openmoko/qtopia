Bluetooth FTP Client

Bluetooth FTP client allows to transfer files over the bluetooth link between your and other Bluetooth devices that support the Bluetooth FTP profile.

If you're not connected to a device, you can choose to connect to a device by using the Browse action. This will bring up a list of nearby devices that you can connect to. Selecting a device from this screenwill attempt to connect to the remote device's Bluetooth FTP service.

Once you're connected, you can create new directories and put new files onto the remote device by using the context menu actions. Deleting files or directories can be accomplished by using the Delete context action. You can also choose to disconnect from the remote device by using the Disconnect context action.

Browsing the directory structure of the remote device is accomplished by using the up/down arrow keys. To switch to a new directory on the remote device, use the Select button.

To obtain files from the remote device and transfer them to your local device, highlight a file and press the Select button.
