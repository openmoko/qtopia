Selecting Messages

Messages you have received are listed in the Inbox folder.

Each folder is displayed along with a summary of the number of messages contained within the folder. The total number of messages in the folder is displayed, and if there are unread messages, the number of unread messages is also displayed.

Select the Inbox folder from the folder list to see the messages you have received.

When viewing the message list, highlight a message and press Select to view the message contents.

When viewing the message list, press on a message to view the message contents.

Marking Messages

In order to perform an operation on multiple messages at once, you can first mark the messages you want to operate on.

To mark messages, press Options and select Mark messages. Each message will now be shown with a checkbox indicating selection. Click on any message to toggle its selection state.

You can mark all available messages at once by pressing Options and selecting Select all.

Once you have marked the messages you wish to operate on, select the operation from the Options menu.

You can cancel message marking by pressing Options and selecting Cancel.
