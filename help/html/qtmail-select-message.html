Selecting Messages

Messages you have received are listed in the Inbox folder.

Each folder is displayed along with a summary of the number of messages contained within the folder. The total number of messages in the folder is displayed, and if there are unread messages, the number of unread messages is also displayed.

Select the Inbox folder from the folder list to see the messages you have received.

When viewing the message list, press / to locate a message you wish to view, then press Select to view the message contents.
