Creating and Editing Contacts

To create a New Entry:
- Tap Contact, then New
- Enter the details for your new contact into the Edit Contact Details screen
- Tap to save the new entry or to cancel.

To Edit an existing Entry:
- Tap the entry to modify
- Tap Contact, then Edit
- Modify the contacts details in the Edit Contact Details screen
- Tap to save your changes or to cancel.

The above commands are also available as buttons on the toolbar.

You can quickly create a new entry that is similar to an existing entry, using the duplicate command, e.g. a new contact at the same company as an existing contact may have the same address, but different phone number and e-mail address.

To duplicate an existing contact:
- Tap the entry to duplicate
- Tap Contact, then Duplicate
- Tap the new duplicate entry
- Tap Contact, then Edit
- Modify the contacts details as appropriate in the Edit Contact Details screen
- Tap to save your changes or to cancel.
