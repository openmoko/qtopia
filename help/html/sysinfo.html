System Info

This application presents a set of tabbed pages each presenting a different aspect of the system's state.

The Memory tab shows memory use. Total available memory is Free plus Cached memory; Qtopia makes use of spare Free memory as Cache to improve speed, but will reclaim Cached memory whenever necessary.

The Storage tab shows how much storage is available for any media currently in use.

The CPU tab shows how hard the processor is working.

The Version tab gives software version numbers.
