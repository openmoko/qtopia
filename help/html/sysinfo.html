System Info

This application presents a set of tabs each containing information about the current system-related status of your device.

Version: version information for the build of Qtopia installed on your device.

Storage: used and available space on the device's internal storage and any removable storage cards.

Memory: free and used amounts of memory.

Data: provides a detailed breakdown of the memory being used.

Security: the state of the safe execution environment<./p>

Modem: technical data about the device's GSM modem<./p>

SIM: the number of used and free SMS and Contacts slots on the SIM card<./p>
