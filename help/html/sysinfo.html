System Info

This application presents a set of tabbed pages each containing information about the current status of the systems Memory, Storage, CPU and operating system Versions.

Memory
Displays memory use.
Used memory is displayed on the left of the bar in red
Total available memory equals Free plus Cached memory; Qtopia makes use of spare Free memory as Cache to improve speed, but will reclaim Cached memory whenever necessary.

Storage
Displays storage available for any media currently in use.

CPU
Displays the CPU usage, or how hard the processor is working.

Version
Displays software version numbers.
