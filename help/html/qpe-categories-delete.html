Deleting Categories

Categories can only be deleted from an application that uses categories. Default categories cannot be deleted.

Items in a deleted category become Unfiled.

Delete a Category

  • Highlight any document.
  • Press and select Properties.

  • When the list of available categories is displayed, use to highlight the category to be deleted.
  • Press and select Delete.
  • Confirm the deletion.
  • Press to save and exit.
