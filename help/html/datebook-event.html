New Event

The New Event dialog provides several comboboxes which you can use to quickly define some features of the event.

For description and location you can either pick from the list (tap the triangle to pop up the list), or enter your own using one of the Input Methods. For Categories, pick from the list or tap the ... button to invoke the Edit Categories Dialog.

You can also set New Event Times, and an Event Alarm, and you can make it a Repeating Event.
