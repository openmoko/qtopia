New Event

The New Event screen provides several dropdown boxes which can be used to quickly information about an event.

For description and location reuse previous descriptions and locations recorded in the dropdown list, or use one of the Input Methods.

To assign a Category to an event:
- Tap the dropdown list or the ... button to access the Edit Categories screen.

Note: You can also set New Event Times, and Event Alarms. You can even make it a Repeating Event.
