The Edit Task Screen

The Edit Task screen is used to record and modify information about tasks. The following fields are available:

Description: Enter a brief description for the task
Priority: To help you prioritize your tasks
1 - Very High, 2 - High, 3 - Normal, 4 - Low, 5 - Very Low
Status: To indicate the current status of the task
- Not Started, In Progress, Completed, Waiting, Deferred
Due: Check this option and select a date from the dropdown list if the task has a due date
Started Date: The starting date can be set for In Progress tasks
Completed Date: Entered automatically when a task is marked complete
Category: Tasks can be organised by assigning them to categories
Notes: Tap the notes page and use one of the Input Methods to enter more detailed information for a task.
- Tap to save the task or changes you have made to a task, or to cancel.
