Proxies

Tap the dropdown list, then select one of the following options:

No proxies - if you don't want proxying.

Auto configure proxies - to make the Qtopia device configure proxies automatically. Select a configuration from the Config dropdown box.

Set proxies - to set proxies manually. Enter the host name or IP address and port number for the HTTP and FTP proxies you wish to use. These values should be supplied by your ISP.
