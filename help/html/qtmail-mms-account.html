MMS Settings

Before you can use MMS you must set up your account.

  • Press Options, then select Account Settings... from the menu.
  • Select MMS from the list.
  • Select Configure Network... from the list to set up an internet connection.

A data connection and WAP account must be created:

    To create a data connection:
  • Press to select the Data tab.
  • Press Options and select New from the menu.
  • Use / to select the appropriate service, e.g. GPRS.
  • Enter the appropriate account settings information, e.g. APN, username and password for GPRS as provided by your service provider.
  • Press back twice to return to the internet connection screen.
    To create a WAP account:
  • Press to select the WAP tab.
  • Press Options and select New from the menu.
  • Select the Account item, and use and Select to choose a data account, e.g. GPRS, then press back.
  • Select the WAP Gateway item. Enter the appropriate WAP Gateway details as provided by your service provider, then press back.
  • Select the MMS Server item. Enter the appropriate MMS server details as provided by your service provider, then press back to return to the internet connection screen.

    • The MMS Server details than can be specified are:

    Server: the URL of your mail server.

    Port: the TCP/IP port of your mail server.

    Expiry: how long the message should remain on the server before it is removed.

    Visibility: if your address or phone number should be visible to the recipient.

    Delivery reports: if ticked, a receipt will be sent indicating that you have viewed the message when the sender requests one. The cost of sending the receipt will be borne by you.

  • Select the newly created WAP account so that it appears as the Standard account, then press back to return to the Network Profile.
  • Press back again to return to the Internet Connection screen, then press back again to return to the network profile screen. In the network profile screen the following option can be specified:

    • Retrieve automatically: if ticked then when Qtopia receives a notification message that a new MMS message has arrived the complete MMS message will be fetched.

  • Press back again to return to the Account Settings screen, and back once more to return to the main Messages screen.
