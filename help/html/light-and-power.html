Power Management

The power management settings are used to configure the power saving functions of your Qtopia device according to the power source being used. The brighter the screen display, the more battery power it is using. To save power, set screen brightness to suit ambient light conditions and use the power management features.

For example, when using the battery you may want the device to dim the display light after a period of inactivity thereby extending the life of the battery.

To configure a power mode:

  • Tap either Battery or External.
  • Adjust the Power saving settings as required.
  • Tap to confirm your choices, or to cancel.

See Also: Power Saving

To change power saving settings, select a power source then configure the power saving values for that power source. Enter the seconds values using the keypad.

Dim light: sets the screen to dim after a set time.

Display off: sets the screen light to go off after a set time.

Suspend: sets the device to go to sleep after a set time.

Brightness: the brightness of the screen light can be controlled using the slider at the bottom of the Power Saving tab. Press and move the stylus along the slider to adjust the brightness of the display.

Use / to highlight the slider, then use / to adjust the brightness of the display.

Always ensure that the screen light is adjusted to a level that allows you to read the screen without straining your eyes.

In all power saving modes, the screen light returns to the configured brightness when any key is pressed.

Add to profile - The current state can be added to a profile so that this state can be applied when the profile is activated.
