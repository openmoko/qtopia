IP Help

Complete the form by entering the required information into the fields, each of which will require an IP address, that should be obtained from your ISP or network service provider.

Select an option by pressing to tick/un-tick options.

Autm. IP (DHCP) tick to use DHCP to discover IP details

If you want to specify the IP details manually un-tick the DHCP check box.

IP Address: enter your IP address.

First DNS: enter preferred DNS.

Second DNS: enter 2nd choice DNS.

Broadcast: enter the broadcast address.

Gateway: enter the address of your internet gateway.

Subnet mask: the subnet mask filter.
