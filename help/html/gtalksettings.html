GTalk

Shows the current configuration of GoogleTalk and allows the user to modify it.

Registration: create or edit a GoogleTalk identity.

  • Server: address of the GoogleTalk / XMPP server
  • Port: Port of the GoogleTalk / XMPP server
  • Username: username to use for login
  • Password: password for username
  • Auto Register: automactic registration with the GoogleTalk / XMPP server when the device starts

Advanced: configure the advanced options.

  • Use Encryption: whether encryption should be used (should be checked)
  • Ignore SSL Errors: whether SSL errors should be ignored (e.g. self signed certificates, generally should be checked)
  • Use Old Style SSL: whether to use old style ssl (generally should be checked)
