Application Associations

The open screen displays a list of applications that can open a document.

To select an application to open a document: Highlight the document, press Options and select Open with... from the menu. Then highlight the application in the displayed list and press Select. tap on the application in the displayed list.

To choose an application not in this list: Highlight Other... at the top of the list and press Select. Tap on Other... at the top of the list. This will expand the list to include all applications.

To add an assocation between an application and a document type: Highlight the application, press Options and select Add association from the menu.

To remove an assocation between an application and a document type: Highlight the associated application, press Options and select Remove association from the menu.

To set the default application for a document type: Highlight the associated application, press Options and select Set as default from the menu.
