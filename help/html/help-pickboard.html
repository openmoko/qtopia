Pickboard Help

The Pickboard is a Trolltech innovation that makes use of the dynamic nature of the Qtopia display to provide reasonably fast text input using a minimum of screen space. To use the Pickboard, pop-up the input area in Pickboard mode.

The Pickboard comprises groups of letters, plus a few symbols and command words.
