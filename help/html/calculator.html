A basic four-function calculator is available with this device.

  • M+ adds the value in the numeric display to the current value in memory. (An "m" appears in the numeric display when the memory contains a value. This will disappear when the memory is cleared.)
  • MC clears the calculator memory.
  • MR recalls the current value from memory.
  • CE/C Tapped once clears the current entry in the numeric display. Tapped twice clears all values entered for this calculation.
  • <BS deletes the last entered digit.

The Qtopia calculator functions in the same way as a handheld calculator. Four calculator modes are available from the drop-down list below the numeric display.

  • Fraction Mode
  • Conversion Mode
  • Simple Mode
  • Scientific Mode
