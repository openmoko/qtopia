LAN

The LAN dialog has several menus for entering the information required to configure a LAN connection:

Note: The wireless options are only available if a wireless LAN service (WLAN) has been selected.

  • Account Tab
  • IP Settings
  • Proxy Settings
  • Wireless Settings
  • Wireless Encryption
  • Wireless Roaming

See also Wireless Scan.

Select a menu option to open the corresponding edit dialog, or press Back to save and exit the service details.
