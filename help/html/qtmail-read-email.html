Reading Email

If you have an email account configured, you can read email messages. Otherwise, you need to create an account: Account Settings

To view email messages, select the Email folder from the folder list. A list of Email folders will be displayed.

If you have a single email account configured, it will be displayed at the top of the folder list. Otherwise, your email accounts will be grouped under an Inbox folder.

If your account is structured with folders, the folder structure will be displayed as an expandable tree.

Each folder's name is displayed along with a message count summary, containing the total number of messages in the folder, and, where relevant, the number of unread messages in the folder. If either number is marked with an asterisk, then there are folders with more messages beneath the marked folder, not currently displayed.

Retrieving Email Messages

Usually, you need to ask the Messages application to retrieve any email messages that have been sent to you.

  • Press Options and select Get all mail from the menu.
  • Any new messages will be retrieved and placed into your Inbox.

If an email message has not been automatically downloaded because it is too large, it will be shown with "Awaiting download" instead of the message body. To download the body, press Options and select Get this mail from the menu.
