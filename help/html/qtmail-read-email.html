Reading Email

If you have an email account configured, you can read email messages. Otherwise, you need to create an account: Create New Account

To view email messages, select the Email folder from the folder list. A list of Email folders will be displayed.

The email Inbox contains a folder for each separate account you have set up.

Retrieving Email Messages

Usually, you need to ask the Messages application to retrieve any email messages that have been sent to you.

  • Press Options and select Get all mail from the menu.
  • Any new messages will be retrieved and placed into your Inbox.

If an email message has not been automatically downloaded because it is too large, it will be shown with "Awaiting download" instead of the message body. To download the body, press Options and select Get this mail from the menu.
