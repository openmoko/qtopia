Profiles

Profiles are used to make the phone sound and behave differently when it receives a call/message. Your Qtopia phone has several existing profiles.

Press for options:

New - create a new profile.

Edit - change the highlighted profile.

Delete - delete a profile. Existing profiles cannot be deleted.

Activate - activate a profile.

Add to Speed Dial... - add the highlighted profile to Speed Dial.

Airplane safe mode: turns off the radio transmitter. No calls can be made or received in safe mode.
