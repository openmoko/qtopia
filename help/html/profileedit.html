Ring Profiles

Ring profiles are used to make the phone sound and behave differently when it receives a call/message. Your Qtopia phone has several existing ring profiles.

To activate a profile: highlight it, then pick Activate from the menu.

To change a profile: highlight it, then pick Edit from the menu.

To change the ring tone: highlight it, then pick Edit from the menu, and go to Tone. Highlight and select Tone to view options.

Where a ring tone has been set for a particular contact it overrides the current ring profile.

Airplane safe mode: turns off the radio transmitter. No calls can be made or received in safe mode.
