Profiles

Profiles are used to make the device sound and behave differently under certain circumstances. Your Qtopia device has several system profiles.

Press Options for options:

New - create a new custom profile.

Edit - change the highlighted profile.

Delete - delete a profile. System profiles cannot be deleted.

Activate - activate a profile.

Add to Speed Dial... - add the highlighted profile to Speed Dial.

Airplane safe mode: turns off the radio transmitter. No calls can be made or received in this mode.
