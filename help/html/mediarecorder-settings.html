Voice Notes Settings

The settings dialog allows you to set the quality and file format of voice recordings.

The first group of radio-buttons allows you to choose from pre-defined quality settings or to choose to use custom settings of your own.

The second group of radio buttons allows you to choose to record mono (one audio channel) or stereo (two audio channels). Some devices will not support stereo recording.

Sample rate: sound quality - more samples means larger file size.

Format: file format
