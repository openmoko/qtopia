Selecting Text

In navigation mode all text in the field is automatically selected. Use to copy the text to the clipboard.

In edit mode it is possible to select a part of the text. Position the cursor at the beginning of the selection and choose Select. Move the cursor to select the text and use to copy the selected text to the clipboard.

Use to insert the previously cut or copied text into the document at the current location of the cursor.

The select, copy and paste icons only appear in menus when they can be utilized.
