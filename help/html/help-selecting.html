Selecting Text

In navigation mode all text in the field is automatically selected. Use the Copy text menu option to copy the text to the clipboard.

In edit mode it is possible to select a part of the text. Position the cursor at the beginning of the selection and choose Select text from the menu. Move the cursor to select the text and use Copy text from the menu to copy the selected text to the clipboard.

Use the Paste text menu option to insert the previously copied text into a text field at the current location of the cursor.

The select, copy and paste icons only appear in menus when they can be utilized.

This device does not have clipboard functionality.
