Simple mode

Simple mode provides the four basic mathematical operations, and memory functions.

M+ adds the value in the numeric display to the current value in memory.

An "m" appears in the numeric display when the memory contains a value. This will disappear when the memory is cleared.

MC clears the calculators memory.

MR recalls the current value from memory.

CE/C Tap once to clear the current entry in the numeric display. Tap twice to clear all values entered for this calculation.

See also Scientific mode.
