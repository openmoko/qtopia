Template text

Template text provides an easy to use method of inserting text into a message being composed by choosing from a list of frequently used text phrases.

You can insert a new phrase into the list of frequently used phrases by selecting the New template text... item.

You can insert a phrase into the message being composed by selecting the phrase.

To remove an item from the list of frequently used text phrases, highlight it then press Options and select Remove template item.

To reset the list of frequently used text phrases to the default list press Options and then select Reset to default.
