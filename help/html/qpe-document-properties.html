Document Properties

The properties screen displays information about a selected document, e.g. its name, location, type, and the categories it belongs to. The properties screen also provides buttons to Copy, Delete or Beam the document.

To view a document's properties:

  • Use to highlight the document.
  • Press and select Properties from the menu.
  • Navigate the fields by using and to open the fields to editing.
  • Tap and hold a document for a few moments.
  • The property information will be displayed for that document.
