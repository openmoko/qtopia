Adding Events

To add an event to your calendar:

  • Launch Calendar and press then select New from the menu.
  • Use and to open fields and the keypad to enter text or select from the drop-down list.
  • Press to complete a field.
  • Press to save the new event and exit.

Unless specified an event is allocated 1 hour in your calendar.

  • Tap Event, then New Event

OR

  • Tap on the toolbar.
  • Enter the details for the new event (for a description of the fields see Edit Event
  • Tap to save the event or to discard it

Events can also be created quickly by tapping the date and time for a new event. When you tap a date in the Week or Month view you are switched to the Day view automatically where you can select a starting time.

When using this technique the date and start time of the event are set automatically.

The end time can also be set by tapping and dragging the stylus from the start to end time of the event.
