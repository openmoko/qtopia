Write Mail To Contact

Mail messages can be created and addressed quickly from Contacts using the Write Mail To commands.

To Create and Address a Mail Message:
- Tap the contact you want to write to
- Tap Contact, then Write Mail To
- Complete and send the message

This command can only be used with contacts who have an e-mail address. See Edit Contact Details for more information.

More information on Creating and Sending messages is available in the E-mail help files.
