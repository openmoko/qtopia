Today

The Today application provides a summary of events occurring today.

The Calendar summary will display events occurring today. Tap on an event to start the Calendar application.

The Email summary will display a number of email headers. Tap a message to start the EMail application.

The Todolist summary will display all open tasks. Tap a task to start the Todo List application.

The information that each summary provides can be customized by tapping in the top left of the summary page. This shows the 'Edit Today View' page.

Today will start automatically each day (at about midnight, or when you first turn the device on in the morning). This can be disabled by unchecking the 'Start automatically every day' check box in the 'Edit Today View' page.
