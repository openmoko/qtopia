New Category

New categories can only be created from an application that uses categories and by opening a document (this provides access to the right menu).

Adding a New Category

  • Launch Documents by using and pressing .
  • Select a document and view it.
  • Press and select Properties.
  • Use to highlight the Cat field and press , then select the ... option.
  • When the list of available categories is displayed, press and select New.
  • Press to open the field, then enter the name of the new category.
  • Press to complete the field.
  • Use and to tick the Global check-box to make the category available to all applications.
  • Press to save and exit.
  • Unless your are going to use the new category immediately, press again and exit Documents.
  • Launch Documents.
  • Select a document from the list and view it.
  • Tap File and select Properties.
  • Select the category field and choose the ... option.
  • Tap the Add button, then tap the edit box at the bottom left, and enter a name.
  • Check the 'Global' check-box to make the category available in all applications.
  • Save and exit.
