New Category

New categories can only be created from an application that uses categories and by viewing the properties of a document (this provides access to the right menu).

Adding a New Category

  • Launch Documents by using and pressing Select.
  • Highlight any document.
  • Press Options and select Properties.
  • Use to highlight the Category field and press Select.
  • When the list of available categories is displayed, press Options and select New.
  • Press Select to open the field, then enter the name of the new category.
  • Press OK to complete the field.
  • Use and Select to tick the Global check-box to make the category available to all applications.
  • Press to save and exit.
  • Unless your are going to use the new category immediately, press again to exit Documents.
