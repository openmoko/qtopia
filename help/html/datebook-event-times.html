New Event Times

The start time and end time of events are automatically set but can be edited.

To indicate that an event lasts all day:

  • Tick the All day tick box.

To set the event's time zone:

  • Use the time zone drop-down box, then select a time zone.
  • If the time zone required is not shown use to add it to the list.

Press to add the event or to cancel.
