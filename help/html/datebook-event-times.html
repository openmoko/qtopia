New Event Times

The start time and end time are automatically set, but you can edit them. To mark an event that lasts all day, check the All day checkbox.

To set the event's timezone tap the timezone combobox, then tap the timezone. If the timezone isn't shown tap and add it using the World Time application.

You can set an Event Alarm, and you can make it a Repeating Event.
