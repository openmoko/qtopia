New Event Times

The start time and end time of events are set automatically, however, you can edit them.

To indicate that an event lasts all day:
- Tick the All day tick box.

To set the event's timezone:
- Tap the time zone dropdown box, then select a time zone. - If the time zone required is not shown tap it can be added to the list using World Time.

You can also set Event Alarms, and create Repeating Events.

Tap to add the event, or to cancel.
