Server Settings

Check or uncheck the checkboxes to suit your preferences.

POP and IMAP Settings

Delete mail if checked, retrieved email will be deleted from the server when they are deleted from the 'Trash' folder.

Skip large emails if checked, large emails will be left on the server. You can retrive the large messages by choosing Get this mail from the Examine mail dialog.

POP only setting

Synchronize with server most users should have this option checked. (When checked the email client remember the list of mails on the POP server)

IMAP only settings

Base folder if your mail folder is not your root folder you can set the base folder here (E.g. INBOX).

... pressing this button will allow you to configure your synchronization settings for the individual mailboxes on the server. If you have not yet connected to the server this button will not be enabled.

You are done configuring your account. Tap to confirm your settings, or to cancel.
