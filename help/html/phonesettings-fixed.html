Fixed Dialing

Use this feature to restrict outgoing calls to those in the Allowed Numbers list.

This feature is pin-code protected to ensure that once activated, only allowed numbers can be dialed.

Check Active and enter password (PIN2) to activate this feature.

To add Allowed Numbers, press Options, then select Add... from the menu.

To remove Allowed Numbers, select a number from the list and then select Remove from the menu.

Activating this feature may be useful when lending your to someone else.
