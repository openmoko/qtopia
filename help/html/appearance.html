Appearance

Use Appearance to change Qtopia's visual appearance including:

  • : change the look of homescreen and top and bottom bars.
  • : choose a color scheme.
  • : choose text or icons for the context bar at the bottom of the screen.
  • : choose background picture from Your images or Wallpapers.

Add to profile - Add the current state to a profile so that this state can be applied when the profile is activated.

The changes will appear when you exit this screen or press the Select key.

  • Colors: choose a color scheme.
  • Fonts: adjust text size.
  • Frame: choose a border.
  • Style: of onscreen elements.

Tap to save any changes you make or to cancel them.

The changes will appear when you exit this screen.
