Appearance

Use Appearance to change Qtopia's visual appearance including:

  • Theme: change the look of homescreen and title and soft menu bars.
  • Color: choose a color scheme.
  • Background: Choose a background image supported by the selected theme.
  • Menubar: choose text or icons for the context bar at the bottom of the screen.

Add to profile - Add the current state to a profile so that this state can be applied when the profile is activated.

Homescreen Settings - Open application to change the appearance of homescreen.

The changes will appear when you exit the application.
