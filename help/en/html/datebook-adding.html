Adding Events

In any view, tap the date you want to add an event to. You will be switched to Day view if you aren't there already.

Tap to display the New Event dialog. You can automatically set the start time of a new event by clicking any time in the Day view.

To create a multi-hour event, tap and hold the start time, drag to the finish time, and release. The start and finish times are set automatically in the New Event dialog.
