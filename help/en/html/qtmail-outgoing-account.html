Outgoing Account

Fill in the fields using one of the Input Methods.

Account name enter your email account's username.

Your name enter your real name.

Email enter your email address.

SMTP server enter the email server name provided by your ISP.

SMTP port default value is 25 (You normally do not need to change this)

Use Signature Select this box if you want your outgoing mails to append a signature. Tap 'Set...' to write your signature.

Default Server Check this box if this outgoing account should normally be used for sending emails.

Now set up the Incoming Account.
