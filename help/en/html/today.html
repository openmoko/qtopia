Today

The Today application provides a summary of events occuring today.

The Calendar summary displays a number of events. Tapping on any displayed event will start the Calendar application.

The Email summary displays a number of email headers. Tapping on any displayed email will start the EMail application.

The Todolist summary displays a number of tasks. Tapping on any displayed task will start the Todo List application.

The information that each summary provides can be customised by tapping the corner button in the bottom right of the summary page.
