Greenphone™ Release Notes

Qtopia® Greenphone SDK is a virtual machine created to run with VMware® Player.

 Please download and install the latest free VMware Player before continuing with the install.
VMware Player is available at http://www.vmware.com/products/player

  • Only wav file support is available in this release (no video playback)
