Detailed Description

The QInfraredAccessoryProvider class provides an interface for infrared devices to integrate into Qtopia.

The QInfraredAccessoryProvider class provides an interface for infrared devices to integrate into Qtopia. Infrared devices inherit from this class and call setMode() to indicate the level of functionality that is supported.

See also QInfraredAccessory.
