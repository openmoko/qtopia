Detailed Description

The QBluetoothServiceController class provides a means to control and access information about Bluetooth services in Qtopia.

To create a Bluetooth service that can be accessed by instances of QBluetoothServiceController, simply subclass QBluetoothAbstractService.

See also QBluetoothAbstractService.
