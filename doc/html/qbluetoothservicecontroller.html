Detailed Description

The QBluetoothServiceController class provides access to information about Bluetooth services within the Qtopia Bluetooth services framework.

The QBluetoothServiceController class provides access to information about Bluetooth services within the Qtopia Bluetooth services framework (i.e. services that are subclasses of QBluetoothAbstractService). It allows the settings for these services to be set and retrieved, and emits signals when any settings are changed.
