Detailed Description

The QHeadsetAccessory class provides access to a headset accessory on the device.

The QHeadsetAccessory class provides access to a headset accessory on the device. Headset device implementations should inherit from QHeadsetAccessoryProvider.

The QHeadsetAccessory can be used to control the microphone and speaker volume on the headset using setMicrophoneVolume() and setSpeakerVolume().

See also QHeadsetAccessoryProvider and QHardwareInterface.

Member Function Documentation

QHeadsetAccessory::QHeadsetAccessory ( const QString & id = QString(), QObject * parent = 0, QAbstractIpcInterface::Mode mode = Client )

Construct a new headset acessory object for id and attach it to parent. The object will be created in client mode if mode is Client, or server mode otherwise.

If id is empty, this class will use the default accessory that supports the headset interface. If there is more than one service that supports the headset interface, the caller should enumerate them with QHardwareManager::supports() and create separate QHeadsetAccessory objects for each.

See also QHardwareManager::supports().
