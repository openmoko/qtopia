Detailed Description

The QModemSimGenericAccess class implements SIM generic access for AT-based modems.

This class uses the AT+CSIM command from 3GPP TS 27.007 to access the SIM.

QModemSimGenericAccess implements the QSimGenericAccess telephony interface. Client applications should use QSimGenericAccess instead of this class to send generic SIM commands.

See also QSimGenericAccess.
