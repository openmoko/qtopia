Detailed Description

The SimControlEventHandlerTask class provides feedback about SIM control events to the user.

When the user initiates a phone call or sends SMS messages on a SIM-equipped mobile phone, the SIM can make modifications to the request before it is sent to the network, or deny the request outright. This process is described in section 9 of 3GPP TS 11.14.

See also QSimToolkit and QSimControlEvent.
