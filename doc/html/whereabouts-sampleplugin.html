This example demonstrates the integration of a Whereabouts plugin that allows location data to be retrieved from custom location data sources through the Whereabouts API. This allows location data to be retrieved from arbitrary sources; for example, from a custom GPS hardware device, or from AGPS sources.

In this example, the plugin is implemented inside the locationplugin project. Once this project is installed, the application in the mylocationapp project will be able to retrieve location data from the locationplugin.

See Location Services for more details on creating and integrating Whereabouts plugins.
