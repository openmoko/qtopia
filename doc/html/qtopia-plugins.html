Plugins

  • System plugins
  • Application plugins
  • Server plugins

System plugins

The following plugins get loaded at application startup:

  • Text codecs
  • Image codecs
  • Font factories
  • Window frame styles
  • Widget styles

Application plugins

The following plugins are used by applications:

  • "Today" plugins
  • Media player
  • Qtopia Desktop

Server plugins

The following plugins will be loaded by the Qtopia server on bootup and restart.

  • Input methods
  • Taskbar applets
  • Network services

See the Writing Qtopia Plugins for information on how to write a plugin.
