Detailed Description

The QTaskDelegate class provides item drawing for QTaskModel items.

The QTaskDelegate class provides drawing of QTaskModel items to aid in consistent look and feel with other applications dealing with QTaskItems. The tasks are presented by placing their description in bold text in a list view.

QFont QTaskDelegate::mainFont ( const QStyleOptionViewItem & o ) const   [virtual]

Returns the font to use for painting the main label text of the item. Due to the nature of rich text painting in Qt 4.0 attributes such as bold and italic will be ignored. These attributes can be set by the text returned for QAbstractItemModel::data() where role is QTaskModel::LabelRole.

By default returns the font of the style option o.
