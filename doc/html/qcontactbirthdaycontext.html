Detailed Description

The QContactBirthdayContext class provides QAppointments based on the QContact birthdays.

QContactModel uses this class to create synthetic QAppointments from QContacts that have a birthday set. The QAppointments are automatically updated (or deleted) as the QContact changes.

See also QContactAnniversaryContext, QContactModel, and Pim Library.
