Detailed Description

The QMediaPipe class connects QMediaDevices in media processing chain.

See also QMediaDevice and QMediaDecoder.

Member Function Documentation

QMediaPipe::QMediaPipe ( QMediaDevice * inputDevice, QMediaDevice * outputDevice, QObject * parent = 0 )

Construct a QMediaPipe with the given inputDevice as source and the outputDevice as destination for all data. The QMediaPipe will be parented to parent.

QMediaPipe::~QMediaPipe ()

Destroy the QMediaPipe object, disconnecting it from the media processing chain.

void QMediaPipe::bytesWritten ( qint64 bytes )   [signal]

Signal emitted when the pipe has written bytes of data to the outputDevice.

qint64 QMediaPipe::read ( char * data, qint64 maxlen )

Read data of maximim length maxlen, this function should normally be called by the outputDevice.

void QMediaPipe::readyRead ()   [signal]

Signal emitted when the pipe has data available from the inputDevice.

qint64 QMediaPipe::write ( const char * data, qint64 len )

Write data of length len, this function should normally be called by the inputDevice.
