Functional Description

Features

  • Basic Location API
  • Integration with FIC Freerunner phone

Basic Location API

GPS location information, if available through hardware, is made accessible through Qt Extended's API libraries.

The Basic Location API provides:

  • Access to all GPS data that is provided by any available GPS data sources - for example, GPS hardware devices, AGPS
  • Access to all available GPS data, for example, latitude-longitude coordinates, satellite information, time, speed, course, bearing, compass
  • Capabilities to perform calculations involving the GPS data - for example, the distance between two coordinates
  • Both synchronous and asynchronous access to GPS data - the developer may choose to retrieve the last known position, or just request event-based notifications of new GPS data

GPS data sources can be configured and added through Qt Extended's plugins mechanism. As a convenience, Qt Extended provides a plugin that uses GPSd (http://gpsd.berlios.de) to access standard GPS data sources.

Integration with FIC Freerunner phone

Qt Extended provides an example device profile for the FIC GTA01 (aka Neo 1973, aka OpenMoko) and FIC GTA02 (aka Freerunner) phones.
