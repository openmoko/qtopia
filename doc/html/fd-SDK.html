SDK
Functional Description

The Qtopia SDK provides developers with libraries and tools with which to develop software for devices running Qtopia.

The Qtopia libraries provide APIs for:

  • UI Controls & Interaction
  • Media and Document Handling
  • Input/Output
  • Data Structures
  • System access

Tools include:

  • Qt Designer
  • Qt Assistant
  • Qt Linguist
  • lrelease
  • lupdate
  • qmake
  • moc
  • uic

Source packages include the sources for the tools and script to build them. In the SDK packages all the above applications are provided as pre-built binaries.
