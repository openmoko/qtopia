SDK
Functional Description

The Qtopia SDK provides developers with several command line and graphical tools to ease and speed the development process, including the following:

  • Qt Designer
  • Qt Assistant
  • Qt Linguist
  • lrelease
  • lupdate
  • qmake
  • moc
  • uic

Source packages include the sources for the tools and script to build them. In the SDK packages all the above applications are provided as pre-built binaries.
