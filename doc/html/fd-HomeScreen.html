Home Screen
Functional Description

The Qtopia Home Screen is the most important element of the phone user interface. It provides important information and allows fast access to the most used features of the phone.

Features

Display Status Information

The Home Screen shows status indicators for:

  • Locked phone
  • Redirected calls
  • Alarm active
  • GSM Network name
  • Cell Location (only if Cell Broadcasts are turned on in the Call Options application)

The exact representation of each of these indicators is determined by the active theme.

Display Missed Calls

Displays the number of missed calls since the Missed Calls list was last viewed. The context menu has a shortcut to the Missed Calls list.

Display New Messages

Displays the number of new messages that have arrived since the messages were last checked. The context menu has a shortcut to the new messages.

Display Current Profile

The current profile is displayed when it is anything other than "General". The context menu has a shortcut to the Profiles application.

Display Important Messages

Messages regarding the state of the phone are prominently displayed on the Home Screen. Available messages include:

  • Searching for GSM/!VoIP networks
  • Airplane safe mode
  • GSM Network unavailable
  • No VoIP network

Guarding the Keypad

The Home Screen provides a way to quickly guard the keypad from accidental key presses. An incoming call can still be answered by pressing the answer button.

Dial a Number

Pressing and releasing a key on the keypad or clicking the keypad icon on a touchscreen displays the dialer, in which the user can enter the remainder of the phone number and initiate a call.

If the phone is locked or keypad guarded it is still possible to make an emergency call.

Activate a Speed Dial Entry

Pressing and holding a number key on the keypad or writing a number or letter on the touchscreen activates the Speed Dial function (if any) associated with that number/letter, provided that the phone is not locked or keypad guarded.

Display Launcher

The launcher can be reached with a single button press (usually Key_Select).

Display a Custom Wallpaper

The user is able to set an image to be displayed in the background of the Home Screen. This setting is made in the Appearance application.

Theming

The Home Screen is themeable without writing and compiling code.
