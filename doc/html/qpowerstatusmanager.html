Detailed Description

The QPowerStatusManager class provides easy access to the power status of the Qtopia device.

Member Function Documentation

QPowerStatusManager::QPowerStatusManager ( QObject * parent = 0 )

Constructs a QPowerStatusManager object with the specified parent.

bool QPowerStatusManager::APMEnabled ()   [static]

Returns true if the device has enabled APM support.

bool QPowerStatusManager::getProcApmStatus ( int & ac, int & bs, int & bf, int & pc, int & sec )   [static protected]

This function implements a standard /proc/apm reader. The results are only valid if APMEnabled() returns true.

  • ac - AC line status
  • bs - battery status
  • bf - battery flag (not used by Qtopia)
  • pc - remaining battery life (percentage of charge)
  • sec - remaining battery life (time units)

void QPowerStatusManager::getStatus ()   [static protected]

Implements the platform specific part of the status reader and is called by readStatus(). This function is defined in custom-{arch}-{platform}-g++.cpp files.

See also readStatus() and getProcApmStatus().

void QPowerStatusManager::powerStatusChanged ( const QPowerStatus & newStatus )   [signal]

Emitted whenever the power status of the device changes. newStatus will be the new power status.

QPowerStatus QPowerStatusManager::readStatus ()   [static]

Reads and returns the current power status of the device.

See also getStatus() and QPowerStatus.
