Detailed Description

The QModemVibrateAccessory class provides vibrate accessory support for AT-based modems.

This class uses the AT+CVIB command from 3GPP TS 27.007.

QModemVibrateAccessory implements the QVibrateAccessory hardware interface. Client applications should use QVibrateAccessory instead of this class to access the modem's vibration settings.

See also QVibrateAccessory.

void QModemVibrateAccessory::setVibrateNow ( const bool value )   [virtual slot]

Sets the vibrateNow() state to value. This is not used in this implementation because AT+CVIB does not support immediate vibrate. Subclasses should override this if the modem supports immediate vibrate.
