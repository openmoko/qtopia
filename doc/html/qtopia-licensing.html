Introduction

Qtopia depends on Qt and there are two types of third-party software in Qt:

  1. Software compiled as part of the Qt library itself - Qt Licenses
  2. Independent packages that may be added to enhance the capabilities of Qt - 3rd Party Qt Licenses

Note: Qtopia 4.1 is available under a Commercial license only.

Qtopia SDK Professional

The Qtopia SDK Professional is targeted at medium to large companies that carry out commercial Qtopia application development and who require the platform source code and full support and maintenance. The Commercial development SDK license enables developers to build commercial Qtopia applications for sale or free distribution. Commercial development SDK license holders are not required to pay runtime license fees. The Qtopia SDK Professional comes with the Qtopia library source as well as twelve months of support and maintenance. It is intended for professional companies that develop software for Qtopia.

Qtopia Runtime License

To distribute Qtopia on consumer electronic devices you need to purchase Qtopia Runtime Licenses. You need one runtime license per devices. The cost per license depends on the number of runtime licenses purchased and the type of Qtopia product. A flexible target volume pricing is offered. The customer must commit to a minimum of 20% of the target volume.

Other License Issues
