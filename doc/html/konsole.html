Detailed Description

The Konsole class is the top-level widget for the Terminal application.

The Terminal allows command-line access to the system and is used for development and debugging activities.

The Terminal (embeddedkonsole) is based on the KDE Konsole application and is distributed under the terms of the GNU General Public License. A primary copyright holder of the code is Lars Doelle.
