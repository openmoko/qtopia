Detailed Description

The QBatteryAccessoryProvider class provides an interface for battery devices to integrate into Qtopia.

The QBatteryAccessoryProvider class provides an interface for battery devices to integrate into Qtopia. Battery devices inherit from this and call setCharging(), setGood(), setCharge and setTimeRemaining() to indicate the level of functionality that is supported.

See also QBatteryAccessory.
