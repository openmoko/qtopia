An Overview of Qtopia Related Tools

There are several tools available to assist in the development of Qtopia software. The following is an overview of the command line and graphical tools that ease and speed the development process.

  • tmake: used with Qtopia 1.6 and 1.7. An older version of qmake.
  • qmake: used with Qtopia 1.9 (limited beta release) onwards. A makefile tool - creates Makefiles from simple platform-independent project files.
  • makeqpf: used to save QPF font files by rendering and saving fonts.
  • Phone Simulator: a graphical desktop method of testing and viewing phone software.
  • qvfb-qtopia.html QVFB : allows Qt/Embedded programs to be developed on a desktop machine.
  • Linguist Manual: Guide to the Qt Translation Tools.
  • Help Browser Preprocessor: an extension to Help Browser that enables the maintenance of a single source of html help files from which device-specific help can be extracted.

Additional tools are available for Qt and Qt/Embedded.
