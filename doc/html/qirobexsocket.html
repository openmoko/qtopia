Detailed Description

The QIrObexSocket class represents an Infrared OBEX connection.

This allows the use of the OBEX protocol over an Infrared connection. For example, a QIrObexSocket object could be used to create a QObexPushClient object in order to push file objects over an Infrared transport connection.

See also QIrObexServer.

QIrObexSocket::QIrObexSocket ( QObject * parent = 0 )

Constructs an new (invalid) Infrared OBEX socket with parent object parent.

QIrObexSocket::QIrObexSocket ( const QString & service, uint addr = 0, QObject * parent = 0 )

Constructs a new Infrared OBEX socket. The service specifies the service to connect to. The addr specifies the address of the remote device to connect to. If the address is 0, all devices are searched and the first one found will be connected to. The parent specifies the parent object.

See also service() and address().

bool QIrObexSocket::connect ()   [virtual]

Tries to establish a connection to the remote device. Returns true if the connection was established successfully, and false otherwise.

NOTE: This call is blocking. Single-threaded applications might freeze for the duration of this call.
