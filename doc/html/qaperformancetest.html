Detailed Description

The QAPerformanceTest class provides performance testing instrumentation to the Qtopia Server.

On receipt of a "runPerftest()" message on the "QPE/PerformanceTest" this task starts and stops a series of applications inside the qpe event loop in order to allow collection of performance data for application startup times. Following the test, the server exists.

Alternatively, sending the "runPerfTestNoQuit()" message performs the same test, but the server continues running.
