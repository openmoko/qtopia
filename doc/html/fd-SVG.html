SVG
Functional Description

SVG is a powerful language for describing two-dimensional vector and mixed vector/raster graphics in XML and is designed to integrate with Web standards such as Xlink, XMLNamespaces, DOM, CSS and XSL. SVG images use stylesheets to control the look and feel of an image and scripting to provide interactivity and animation. SVG can be applied to a wide set of problems such as resolution-independent image description and format transcoding from proprietary formats. Qtopia's compliance with SVG provides not only the means for enhanced graphics capabilities but also the foundation for interaction with third-party tools.
