Detailed Description

The EnvironmentSetupTask class initializes the basic system environment required by Qtopia.

The EnvironmentSetupTask configures the basic environment variables required by Qtopia. It should execute before QtopiaApplication or any other task in the system.

The EnvironmentSetupTask class provides the EnvironmentSetup task.
