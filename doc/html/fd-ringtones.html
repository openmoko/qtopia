User Interface: Personalization: Ringtones
Functional Description

Features

  • Per-Group Ring Tones
  • Ringtones

Per-Group Ring Tones

The user can associate ring tones with Contact and Categories of Contacts.

When a call is received, the personalized ring tone or group ring tone is used instead of the one in the current profile.

The fall-back policy for a ring tone when a caller is known:

  • A personalized ring tone is used if exists.
  • A group ring tone is used if exists.
  • The of first group in alphabetical order is used if the contact is included in more than one category.
  • A current profile's ring tone is used.

Ringtones

The user may associate an audio file with a Profile for use as a Ring Tone to signal either an incoming phone call or a received message.
