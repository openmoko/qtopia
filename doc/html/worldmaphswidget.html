Detailed Description

The WorldmapHSWidget widget displays a worldmap.

The WorldmapHSWidget is a homescreen widget that displays a worldmap showing the currently selected timezone and a daylight mask. The WorldmapHSWidget is added to the homescreen theme including the widget theme element in the theme XML file, for example:

    <widget name="WorldmapHSWidget" rect="0,0,0x90" />
