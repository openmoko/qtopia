Detailed Description

The QAtChatScript class provides a mechanism to run pppd chat scripts.

The QAtChatScript class provides a mechanism to run pppd chat scripts, such as those described in the chat(1) Unix man page.

This class is intended for use by Qtopia's network component. Other components should use QAtChat.

See also QAtChat.

void QAtChatScript::done ( bool ok, const QAtResult & result )   [signal]

Signal that is emitted when the chat script is done. The ok flag indicates whether the script succeeded or failed, and result indicates more information about the failure (busy, no carrier, etc).

int QAtChatScript::successfulCommands () const

Get the number of commands that were successful. If this is less than totalCommands() when the done() signal is emitted, then the script stopped with an error before the whole script could be executed.
