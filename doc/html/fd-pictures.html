Multimedia: Pictures
Functional Description

The Pictures application allows the user to manage an image collection. Image formats such as PNG, JPEG, BMP, XBM, XPM and GIF are supported and the user is able to browse through their collection, view slide shows, view images close-up, adjust orientation, adjust brightness, and perform cropping. Once an image has been edited the user has the option of saving the new image by either creating a new image or overwriting the old image.

Adjust Image Brightness

The user is able to adjust the brightness of images in their collection.

Adjust Image dimensions

The user is able to crop images in their collection.

Adjust Image Orientation

The user is able to adjust the orientation of images in their collection. The user is able to rotate images by 90, 180, 270 degrees.

Browse Collection

View images from an image collection as thumbnails or as whole images. The entire collection may be browsed or only images belonging to a specific category.

Controls permanenty effect state

This feature is provided as part of the Deskphone Reference Design.

Changes to the rotation of an image are persisted between viewings of the image and are also reflected in thumbnails of the image.

Change to the slide-show timeout will apply to all future slide-shows, including those started from future instances of the Photos application.

Delete Image

The user is able to delete images from their collection.

Edit Image Properties

The user is able to edit the properties of images in their collection. The user is able to rename, change storage location, and change the category of an image.

Navigate Image

The user is able to view specific sections of a large image by moving a viewport around the image.

Performance characteristics for home user

This feature is provided as part of the Deskphone Reference Design.

Response to changing between groups of photos or selecting specific photos is rapid.
Caching is used to make repeated actions faster i.e. a few seconds the first time a folder is opened, almost instantaneous afterward.

Save Image

Once the user has finished editing an image they are able to save or discard the changes. The user has two options when saving:
1 overwrite the original image
1 save as a new image.

Send Image

The user is able to send images from their collection over a wireless (Bluetooth, infrared) link.

Set image as Avatar

The Pictures application can send a selected image to an Instant Messaging client for use as an Avatar.

Set image as background

The Pictures application allows the user to set the background image of the device.

Slide Show

View a slide show of an image collection. The entire collection may be viewed as a slide show, or only images belonging to specific categories. The user is able to adjust the duration of each slide, whether or not to loop, and whether or not to display the name of each image.

Supported Formats

Pictures supports the following image formats:

  • PNG
  • JPEG
  • BMP
  • XBM
  • XPM
  • GIF (read only)
  • SVG (read only)

Use content stores for Photos

This feature is provided as part of the Deskphone Reference Design.

The Photos application can view images stored on local storage media.

View Album contents with Flow View

This feature is provided as part of the Deskphone Reference Design.

Images in a photo album can be browsed in a picture flow style view.

Selecting the center image will display it in the image view.

Selecting the rotate (anti)clockwise button will rotate the center image (anti)clockwise.

Selecting the play slide-show button will start a slide-show of the photo album starting from the center image.

Selecting the index view button switches the view to the icon style view.

View Album contents with Index View

This feature is provided as part of the Deskphone Reference Design.

Images in a photo album can be browsed in a horizontally scrolling icon view.

Selecting an image will display it in the image view.

Selecting the play slide-show button will start an album slide-show from the currently selected image.

Selecting the flow view button switches to the picture flow style view.

View Album Slideshow

This feature is provided as part of the Deskphone Reference Design.

Images in a photo album can be viewed in a full-screen slide-show.

Images in the slide-show progress in order of increasing creation date.

Tapping the screen will display the title bar with the name of the current image, a pause/continue button and a back button, the title bar will fade out after a period or inactivity or if the screen is tapped while it is visible.

Dragging the screen horizontally will change the slide-show timeout duration. The current duration can be seen in a bar at the bottom of the screen that will fade after a period of inactivity or if the screen is tapped.

Pressing the pause button will stop the slide-show from progressing until the continue button is pressed.

View Albums by Date

This feature is provided as part of the Deskphone Reference Design.

Photo albums can be viewed in a horizontally scrolling icon view sorted by date.

Selecting an album will display its contents in an icon or flow view.

Selecting a year from the ribbon at the top of the screen will scroll the view to albums from that year.

Selecting the sort by name button will switch to the sort by name album view.

View Albums by Name

This feature is provided as part of the Deskphone Reference Design.

Photo albums can be viewed in a list view sorted by name.

Selecting an album will display its contents in an icon or flow view.

Selecting a character from the ribbon at the top of the screen will scroll the view to albums starting with that character.

Selecting the sort by date button will switch to the sort by date album view.

View Image in Full Screen

The user is able to view and navigate images using the entire screen.

View Photo

This feature is provided as part of the Deskphone Reference Design.

Images can be viewed in a full-screen view.

Tapping the screen will display the controls. They will fade out again after a period of inactivity or if the screen is tapped while they are visible.

The zoom controls change the current zoom level.

The rotate (anti)clockwise button rotates the image (anti)clockwise.

The add to faves button adds the image to the Favorites album.

Dragging a zoomed image will pan it around the screen.

Pressing the left arrow button or dragging the image off the screen to the right will display the previous image in the album. Pressing the right arrow button or dragging the image off the screen to the left will display the next image in the album.

Pressing the play slide-show button will start an album slide-show starting from the visible image.

Zoom Image

The user is able to zoom in and out of images in their collection.
