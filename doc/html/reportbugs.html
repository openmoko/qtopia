How to Report a Bug in Qtopia

If you think you have found a bug in Qtopia, we would like to hear about it so we can fix it.

Before reporting a bug, please check the FAQs, Task Tracker on our web site to see if the issue has already been reported.

The following information must always be included in a bug report:

  1. The name and version number of your compiler
  2. The name and version number of your operating system
  3. The version of Qtopia Core and Qtopia being used
  4. The configure options used when compiling.

If the problem is only visible at run-time, please create a small test program that shows the problem when run. Often, such a program can be created by making minor changes to one of the many example programs in the examples directory.

Please submit the bug report using the Bug Report Form on the Trolltech website.
