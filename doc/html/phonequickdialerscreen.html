Detailed Description

The PhoneQuickDialerScreen class implements a keypad based dialer UI.

This class is a Qtopia server widget. It is part of the Qtopia server and cannot be used by other Qtopia applications.

See also QAbstractServerInterface and QAbstractDialerScreen.
