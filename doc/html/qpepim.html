Qtopia PIM Library

The Qtopia PIM library provides a set of safe APIs for accessing Qtopia PIM data. Specifically, classes are provided to retrieve, edit, and save AddressBook's Contacts and Todo's Tasks. The access class will also keep applications up to date with changes made externally from other processes.

See Qtopia PIM Library Overview
