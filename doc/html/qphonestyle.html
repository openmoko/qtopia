Detailed Description

The QPhoneStyle class provides a phone look and feel.

This style is Qtopia's default GUI style for Qtopia Phone Edition.

See also QWindowsStyle, QWindowsXPStyle, QMacStyle, QPlastiqueStyle, QCDEStyle, and QMotifStyle.
