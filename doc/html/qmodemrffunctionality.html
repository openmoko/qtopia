Detailed Description

The QModemRfFunctionality class provides a method to get or set the level of RF activity on an AT-based modem.

Thhis class uses the AT+CFUN command from 3GPP TS 27.007.

QModemRfFunctionality implements the QPhoneRfFunctionality telephony interface. Client applications should use QPhoneRfFunctionality instead of this class to access the modem's RF functionality level.

See also QPhoneRfFunctionality.
