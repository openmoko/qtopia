Personal Information Management (PIM)

These classes provide access to the user's PIM data, for causual access or to write replacement PIM applications.

See Qtopia PIM Library

QAppointmentHolds the data of a calendar appointment
QAppointmentContextRepresents a storage context of appointment data
QAppointmentModelAccess to the Calendar data
QContactHolds the data of an address book entry
QContactAddressAddress of a QContact
QContactContextRepresents a storage context of contact data
QContactDelegateItem drawing for QContactModel items
QContactListViewList view widget with some convenience functions for use with QContactModel
QContactModelAccess to the the Contacts data
QContactSelectorWay of selecting a contact from a QContactModel
QOccurrenceHolds the data of an occurrence of a calendar appointment
QOccurrenceModelAccess to the Calendar data
QPhoneNumberUtilities of interest to phone applications that manipulate numeric telephony numbers
QPhoneTypeSelectorWay of selecting a single phone number from or for a QContact
QPimContextRepresents a storage context of PIM data
QPimDelegateAbstract delegate for rendering multiple lines of text
QPimModelAbstract interface to the pim model classes
QPimRecordThe base class for PIM data recorded in the Qtopia database
QPimSourceHolds identifying information for a storage source of PIM data
QPimSourceModelModel class for managing the set of QPimSource objects of a PIM Model
QTaskHolds the data of a todo entry
QTaskContextRepresents a storage context of task data
QTaskDelegateItem drawing for QTaskModel items
QTaskListViewList view widget with some convenience functions for use with QTaskModel
QTaskModelAccess to the the Tasks data
QTaskSelectorWay of selecting a task from a QTaskModel
