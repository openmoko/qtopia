Detailed Description

Provides the Qtopia CallNetworks service.

The CallNetworks service is used by the server to detect the presence of a call networks settings program. When the user selects a call network feature, the program associated with the CallNetworks service will be activated.

Normally the phonenetworks program that is distributed with Qtopia Phone Edition provides the CallNetworks service. If the phonenetworks program is replaced, then the service association will have to be redirected to the replacement program.
