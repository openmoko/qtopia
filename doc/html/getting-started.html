How to Obtain Qtopia Products

Trolltech provides both licensed and free editions of Qtopia. Details of how to obtain these editions is provided below. For specific licensing information please refer to: Qtopia Licensing.

Free Editions

Qtopia packages for the OpenSource, GPL License community, can be downloaded from: Trolltech's FTP site.

Licensed Editions

After becoming a Qtopia customer an email will be sent from sales@trolltech.com confirming your license details. The details include:

  • Customer ID
  • License ID
  • Qtopia package download FTP site location.

Once you have received these details login to the FTP site using any browser and download the packages. Depending on your license the FTP site provides either a:

  • qtopia source package
  • qtopia sdk package

The FTP directory is available for the full duration of the contract, and will be updated regularly with new releases. Customers are notified via email about each new release that is relevant to their account.

Note: The FTP site may also provide access to other files such as various Qt versions, snapshots of packages, language packages and much more, however, this document refers only to the Qtopia specific-files available on the FTP site.

Package Naming Conventions

The FTP download directory contains multiple packages and at first it may be unclear which packages are required. Depending on the type of license you have, one or more of the following packages is needed:

PackageDescription
qtopia-[edition]-[commercial|opensource]-src-[x.y.z].[zip]Full source package: this package contains all sources to build Qtopia, including the prerequisite source for Qt.
qtopia-device-[devicename]-src-[x.y.z].[zip]Device source package: this package contains additional sources that are required to build Qtopia for a specific device. A device package is installed 'on top' of the Qtopia source package. Multiple device packages can be installed.
qtopia-language-[languagename]-[x.y.z].[zip]Language package: this package contains translations for a specific language. A language package is installed 'on top' of the Qtopia source package. Multiple language packages can be installed.
qtopia-[thirdparty-addon]-[commercial|opensource]-src-[x.y.z].[zip]Thirdparty add-on source package: this package contains sources for a thirdparty add-on module that can be used to enhance Qtopia. A thirdparty add-on package is installed 'on top' of the Qtopia source package. Multiple add-on packages can be installed.
qtopia-[edition]-sdk-[x.y.z]-[distro].i386.rpm Software Development Kit (SDK) package: this package contains a combination of pre-built binaries and library header files. SDK packages are typically built for a specific distribution, for example, RedHat, SuSE, etc.
qtopia-[edition]-sdk-devel-[x.y.z]-[distro].i386.rpmSDK extension package: containing Qtopia library source files. The development package is installed 'on top' of the SDK package.

Naming Convention Tags

In the above list the following tags have been used:

TagDescription
[edition]Qtopia product name, one of:
  • core
  • platform
  • phone
  • refboard.
[x.y.z]Product version number, for example 4.1.3
[zip]Indicates that a package is made available as a compressed package such as:
  • tar.gz
  • tar.bz2.
[distro]Operating system or distribution on which the package can be used, for example:
  • redhat9
  • suse82
  • suse9
  • macosx
  • win etc...
[exe]Package type, one of:
  • .rpm
  • .dmg
  • .tar.gz
  • .exe.
