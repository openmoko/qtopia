Detailed Description

The Note class provides a simple container for messages.

This structure has the following fields:

  • QString text - The text field contains the text to be displayed to the user.
  • QString pixmap - The pixmap field contains the path (e.g. ":image/ok") to the picture associated with the text.
  • int id - The id field contains the unique identifier for this note on the MessageBoard. It can be a number between 0 and 65536.

See also MessageBoard.
