User Interface: Launcher
Functional Description

The Qtopia Launcher is the access point for all applications installed on the device. It also allows the user to manage their documents. The primary requirement of the Launcher is to provide fast access to the most frequently used applications.

See also Application Execution.

Features

  • Application, Games and Settings Lists
  • Documents List
  • Grid Menu
  • Homescreen widget plugins

Application, Games and Settings Lists

These lists display all applications, games, and settings programs available to the user.

The user is able to navigate the list using the arrow keys or by typing the name of the application using the keypad. Selecting an item in the list launches that item.

The user is able to set items in the lists as Speed Dialing shortcuts.

Documents List

This list displays all documents on the device.
The user is able to filter documents by category, by mime type, or by both.

The user is able to navigate the list using the arrow keys or by typing the name of the document using the keypad.
Selecting a document launches the associated application and opens the document in that application.

The user is able to edit the properties of documents from the list. The user is able to rename, change storage location, and change the category of the document. In addition the user is able to delete documents and send documents over an infrared link from the list.

If a particular document is DRM protected, the user is also able to open the URL of the document's rights issuer from the list.

Grid Menu

This menu allows the user to directly access the most frequently used applications. It also provides access to lists of applications, games, settings and documents.

Items in this menu are positioned in a grid. The user is able to launch items by using the keypad or arrow keys. Pressing a key in the keypad launches the corresponding item in the grid. Pressing an arrow key moves a selector around the grid.
The user is able to launch the item under the selector by pressing the select key.

For keypad phones, the icons used in the grid menu are magnified and play a short animation when they become highlighted by the selector.

It is possible for the OEM/ODM to configure the arrangement and contents of the grid menu.

Homescreen widget plugins

Widgets can be placed in the homescreen to provide additional rich content. For example, a summary of the day's appointments, or shortcuts to selected applications.
