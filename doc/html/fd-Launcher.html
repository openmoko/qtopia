User Interface: Launcher - Qtopia 4.3 Roadmap
Functional Description

The Qtopia Launcher is the access point for all applications installed on the device. It also allows the user to manage their documents. The primary requirement of the Launcher is to provide fast access to the most frequently used applications.

See also Application Execution.

Features

  • Homescreen widget plugins

Homescreen widget plugins

Widgets can be placed in the homescreen to provide additional rich content. For example, a summary of the day's appointments, or shortcuts to selected applications.
