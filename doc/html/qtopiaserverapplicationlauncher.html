Detailed Description

The QtopiaServerApplicationLauncher class acts as a proxy for the Qtopia Server within the application launcher framework.

The QtopiaServerApplicationLauncher provides a Qtopia Server Task. Qtopia Server Tasks are documented in full in the QtopiaServerApplication class documentation.

Task NameQtopiaServerApplicationLauncher
InterfacesApplicationTypeLauncher
ServicesNone

To enable the Qtopia server itself to receive IPC messages, it must be known to the application launcher framework. The QtopiaServerApplicationLauncher task provides this.

It is essential that the QtopiaServerApplicationLauncher is the first ordered ApplicationLauncherType provider.
