OpenOBEX wrapper API

Bluetooth and Infrared use OBEX protocol to implement various data transfer mechanisms, such as push/pull, folder browsing, and synchronization. Qtopia uses the third party OpenOBEX library, which is very awkward and time consuming to use directly. Qtopia provides a Qt-based, easy to use wrapper API for OpenOBEX, which makes it easy to develop OBEX based clients and services.
