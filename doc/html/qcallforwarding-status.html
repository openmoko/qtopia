Detailed Description

The Status class provides information about a single call forwarding rule.

The Status class contains information about a single call forwarding rule. The cls member is the class of call (or calls) that the rule applies to. The number member is the phone number that calls will be redirected to, The time member is the amount of time to wait for QCallForwarding::NoReply before forwarding the call. The time member will be zero if the rule is not for QCallForwarding::NoReply, or the timeout is unknown.
