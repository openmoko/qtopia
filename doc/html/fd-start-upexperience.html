User Interface: Start-up Experience
Functional Description

When Qt Extended starts up, a splash screen appears. This can be an animated GIF.

Features

  • Initial Settings

Initial Settings

The "first-use" dialog helps the user to setup a new device. It simplifies the personalization process because the user doesn't have to start a variety of other applications to perform the initial personalization process.

The first-use dialog guides the user through the configuration of the following:

  • calibration of the touchscreen (if present)
  • language selection
  • time zone selection (if needed)
  • time, date format and other preferences
