Qtopia Commercial Edition: Commercial SDKs

There are several commercial versions of Qtopia available.

PDA OEM KitPhone OEM KitCommercial SDKDeveloper SDKs
  • Aimed at companies manufacturing PDAs that run Qtopia.
  • Aimed at companies producing mobile phones that run Qtopia.
  • Aimed at small companies developing commercial Qtopia applications.
  • No support or maintenance required.
  • Aimed at medium/large companies developing commercial Qtopia applications.
  • Platform source code required.
  • Full support and maintenance required.

Commercial, proprietary and non-free software may only be written if one of the commercial editions of Qtopia has been purchased. Various licenses and license agreements are available to suit the development and distribution needs of customers. If free software is going to be developed for release using a recognized open source license then one of the Qtopia Free Editions should be used.

The purchase of a Qtopia commercial edition includes technical support and upgrades.
