User Interface: Personalization: Appearance
Functional Description

The Appearance settings application allows the user to select a color scheme, theme, font and background image.

Features

  • Background Image Selection
  • Color Scheme Selection
  • Context Menu Display Style Selection
  • Integration with Profiles
  • Theme Selection

Background Image Selection

The user is able to choose a background image. This image is displayed in the Home Screen and is also displayed, in a 'faded' form, behind all transparent windows (i.e. in the Launcher and in most application windows).

It is possible to choose from both the user's own stored images and Qtopia's pre-defined wallpapers when selecting a new background.
Any image that is chosen is scaled to fit the window size whilst maintaining its aspect ratio.

Color Scheme Selection

The user is able to choose a color scheme. A color scheme defines the colors
used for visuals such as text and background areas. For example,
a color scheme may be comprised of colors that are shades of green.

Context Menu Display Style Selection

The user is able to choose a display style of the menu bar between icon and text.

Integration with Profiles

Appearance settings can be associated with a Profile.

Theme Selection

The user is able to select a theme to define the arrangement and appearance
of the homescreen, title bar, context bar and window decorations. For more
information on theming see Qtopia Theming.
