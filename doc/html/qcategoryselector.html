Detailed Description

The QCategorySelector widget allows users to select Categories with a combobox interface.

QCategorySelector is useful to provide a QComboBox of Categories for filtering (such as in the Contacts table view) or to allow the user to select multiple Categories.

In filtering mode, the All and Unfiled categories are added.

In selecting mode, the QCategorySelector may either be a QComboBox and a QToolButton or a QListView with checkable items depending on the screen size.

QCategorySelector automatically updates if Categories has been changed elsewhere in the environment.

Signals and slots are provided to notify the application of the users selections. A QToolButton is also provided so that users can edit the Categories manually.
