Detailed Description

The PowerManagerTask class implements the device's power management functionality.

The PowerManagerTask class provides the PowerManager task. Qtopia's Light-and-power application configures the functionality implemented by the PowerManager task. The PowerManagerTask monitors QPowerStatusManager and calls powerStatusChanged() whenever the power status changes.

See also QtopiaPowerManager and Hardware Configuration.
