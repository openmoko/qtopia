Plugins (QTOPIA-1405)

Qtopia plug-ins are provided to extend Qtopia. They are integrated into Qtopia and provide additional functionality to applications while the plug-in framework provides a well-defined development environment for new features.

Qtopia includes the following interfaces for plug-ins:

  • Content recognition
  • DRM handling
  • GUI Styles
  • Image Formats
  • Icon Formats
  • Input Methods
  • Media Codecs
  • Media Engines
  • Modem drivers
  • Network interfaces
  • Printing
  • Telephony Multiplexer
  • Text encodings
