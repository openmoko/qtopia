Communications: Telephony: Call Management
Functional Description

Call Management is handled via Call Screen which displays current incoming, outgoing, and active calls.

Features

  • Active Calls
  • Call Details
  • Explicit Call Transfer
  • Handset Volume
  • Held Calls
  • Incoming Call
  • Multi-party Calls
  • Multiparty voice calls
  • Outgoing Call

Active Calls

The user is able to end active calls by pressing the Hang-up button, or selecting End from the context menu. The user is able to place and active calls on hold by
pressing Select (default action) or selecting Hold from the context menu.

Call Details

Call Screen presents the following details for each call:

  • Name or number: if the number matches an address book entry, the name is shown, otherwise only the number is shown.
  • Contact's photo: if the number matches an address book entry.
  • Call duration: the duration the call has been connected.
  • Whether the call is on hold.
  • Whether the call is part of a multi-party call.

Explicit Call Transfer

This feature is provided as part of the Touchscreen Phone Reference Design.

By using the GSM key sequence 4 SEND or selecting Transfer from the options menu, the user can transfer a current call to a held call. The user is disconnected from both calls.

Handset Volume

The volume of the earpiece is controlled by the left and right keys on the direction pad.

Held Calls

The user is able to activate calls on hold by pressing Select or selecting Resume
from the context menu. If there are both calls on hold and active calls, then
the Select button swaps between the active and held calls.

Incoming Call

Call Screen is shown when an incoming call arrives, with the incoming call positioned at the top of the call list. The user is able to:

  • Accept the call by pressing the Call key or selecting Answer from the context menu.
  • Reject the call by pressing the Hang-up key or selecting Send Busy from the context menu.
  • Ignore the call by selecting Mute from the context menu.

Multi-party Calls

If there are active call(s) and held call(s), then the user is able to join the
calls to form a multi-party call by selecting Join from the context menu.

The user is able to split from a multi-party call by selecting Split from the
context menu.

Multiparty voice calls

A multiparty voice call can be created by setting the active call on hold, either by creating a new call via the dialer or accepting an incoming call, then subsequently selecting the Join option from the menu.

Once established the multiparty call can be split, allowing the user to create a private communication with one call and placing all other calls on hold.

In the Home edition, up to three (3) participants are supported by the call manager user interface. Also, if the desired call to be added is an incoming call, the option to merge appears on the incoming call notification.

Outgoing Call

Call Screen is shown when an outgoing call is initiated with the outgoing call positioned at the top of the call list. The user is able to cancel the call by
pressing the Hang-up key, or selecting End from the context menu.
