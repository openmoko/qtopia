Introduction

This document describes the prerequisites and steps required to build Qtopia using Teambuilder.

Prerequisites

The following are the minimum requirements to use Teambuilder to build Qtopia:

  1. /opt/teambuilder/bin/make contains the following:
        #!/bin/sh
    export TEAMBUILDER_COMPRESS=0
    if [ "$TEAMBUILDER" = 0 ]; then
        /usr/bin/make "$@"
    else
        /usr/bin/make -j15 "$@"
    fi
  2. Entries for your toolchains have been added to /etc/tbdaemon.conf.

Teambuilder Example

The following example demonstrates how to configure Teambuilder to use the Embedix toolchain for the Zaurus.

  1. Set path to compilers and Teambuilder.
        export PATH=/opt/teambuilder/bin:/opt/Embedix/tools/bin:$PATH
  2. Set up Teambuilder environment.
        export TEAMBUILDER=1
    export TEAMBUILDER_CC_VERSION="arm-linux-*:2.95*;*:3.3*"

The following table describes how Teambuilder is affected by the environment variables.

VariableDescription
TEAMBUILDERUse the -j option when invoking make.
TEAMBUILDER_CC_VERSION
  • If arm-linux-* use the first compiler that matches version 2.95* (for example, Embedix toolchain for Zaurus).
  • For any other compiler use the first compiler that matches version 3.3* (desktop).

Using the TEAMBUILDER_SYSTEM Variable

The following example demonstrates the use of TEAMBUILDER_SYSTEM environment variable to select the compiler for a specific target platform.

  1. Set path to compilers and Teambuilder.
        export PATH=/opt/teambuilder/bin:/opt/toolchains/arm9tdmi/arm-linux/gcc-3.3.3-glibc-2.3.2/bin:$PATH
  2. Set up Teambuilder environment.
        export TEAMBUILDER=1
    export TEAMBUILDER_SYSTEM="arm:9tdmi"
    export TEAMBUILDER_CC_VERSION="arm-linux-*:3.3.3;*:3.3*"

The following table describes how Teambuilder is affected by the environment variables.

VariableDescription
TEAMBUILDERUse the -j option when invoking make.
TEAMBUILDER_SYSTEMSelects a compiler with the specified label on other machines. Matches compilers tagged with [arm:9tdmi] in /etc/tbdaemon.conf.
TEAMBUILDER_CC_VERSION
  • If arm-linux-* use the first compiler that maches version 3.3.3.
  • For any other compiler use the first compiler that matches version 3.3*.
