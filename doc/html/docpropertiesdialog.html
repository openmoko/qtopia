Detailed Description

The DocPropertiesDialog class allows the user to examine attributes associated with a AppLnk object.

The DocPropertiesDialog uses a DocPropertiesWidget to allow the user to examine and modify attributes associated with a file.

First availability: Qtopia 1.6

See also DocPropertiesWidget and Qtopia Classes.
