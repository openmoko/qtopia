Detailed Description

The ThemedSecondaryDisplay class implements the secondary display for Qtopia Phone.

This class is a Qtopia server widget. It is part of the Qtopia server and cannot be used by other Qtopia applications.

See also QAbstractServerInterface and QAbstractSecondaryDisplay.
