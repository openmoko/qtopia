Detailed Description

The QModemAdviceOfCharge class provides advice of charge information on AT-based modems.

This class implements the QAdviceOfCharge telephony interface. Client applications should use QAdviceOfCharge instead of this class.

The default implementation in this class uses the AT+CAOC, AT+CACM, AT+CAMM, and AT+CPUC AT commands from 3GPP TS 27.007. It also understands the +CCCM and +CCWV unsolicited results from 3GPP TS 27.007.

See also QAdviceOfCharge and QModemService.
