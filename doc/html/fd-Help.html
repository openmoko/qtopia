Help
Functional Description

Features

Context Sensitivity

The user is offered context-sensitive help via:

  • PDA UI:
    • Click on "?" in heading for help on current screen.
    • Press-and-hold on "?" in heading for "What's this?" help.
  • Phone UI:
    • Select "Help" from context menu for help on current screen.

Help Browser

Help is displayed by a Help Browser application which allows the user view help pages. The user is able to browse through system help pages or application specific help pages. The user is able to view help pages in rich text and navigate using hyperlinks. The user is able to jump backwards and forwards through the browser page history.

Rich Text

Help is presented using rich text elements. Support for rich text is provided by the QStyleSheet class. See the QStyleSheet reference for more detail.

Hyperlinks

Help includes hyperlinks. The navigation path of the user is saved in the browser page history.

Step Back

The user is able to step back one page in the browser page history.

Step forward

The user is able to step forward one page in the browser page history.

Home Page

The user is able to jump directly to the system help home page.

Improved Document Navigation

Use the left and right keys to move forward and back through the page history.

Bookmarks

Users can tag pages as bookmarks. A list of bookmarks can be opened from the context menu. Bookmarks can be deleted from this list.
