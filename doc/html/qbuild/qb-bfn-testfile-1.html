testFile()

Usage

    testFile(file,contents)

Description

The first argument is a filename. The second argument is the contents of the file. Note that the parser makes it impossible to pass arbitrary characters into this function. This is designed to be used by the build system, in conjunction with the writeFile() function.

See also QBuild Functions and QBuild Script.
