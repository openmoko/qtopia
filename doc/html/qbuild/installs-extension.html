installs Extension

The installs extension handles installing applications, libraries and plugins as well as processing many of basic install hints. For information about the available install hints, see Install Hints.

Variables

TypeNameDescription
InputQTOPIA_IMAGEThe location of the image to create.
OutputINSTALLS_INSTALLPICThe location of the installpic script is stored in this variable.
OutputINSTALLS_INSTALLHELPThe location of the installhelp script is stored in this variable.

POSTIMAGE_RULESRun rules after the "image" rule.
PREIMAGE_RULESRun rules before the "image" rule.
