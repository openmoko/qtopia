Detailed Description

The QTimeZoneWidget widget allows configuration of time zone information.

Member Function Documentation

QTimeZoneWidget::QTimeZoneWidget ( QWidget * p = 0 )

Creates a new QTimeZoneWidget with parent p. The combobox will be populated with the available timezones.

QTimeZoneWidget::~QTimeZoneWidget ()

Destroys a QTimeZoneWidget.

QString QTimeZoneWidget::currentZone () const

Returns the currently selected timezone as a string in location format, for example,

    Australia/Brisbane

See also setCurrentZone().

bool QTimeZoneWidget::localIncluded () const

Returns true if no timezone is included as an option; otherwise false.

See also setLocalIncluded().

void QTimeZoneWidget::setCurrentZone ( const QString & id )

Sets the current timezone to id.

See also currentZone().

void QTimeZoneWidget::setLocalIncluded ( bool b )

If b is true, allow no timezone as an option. If b is false then a specific timezone must be selected.

See also localIncluded().

void QTimeZoneWidget::signalNewTz ( const QString & id )   [signal]

This signal is emitted when a qtimezone has been selected. The id is a QString in location format, for example,

    Australia/Brisbane
