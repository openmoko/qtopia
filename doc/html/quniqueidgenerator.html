Detailed Description

The QUniqueIdGenerator class provides a generator of unique identifiers.

The QUniqueIdGenerator class provides a generator of unique identifiers. They are 32 bits, 4 bits of which make up a context and the remaining 24 bits an index component. This allows separate contexts to generated ids to be stored in a single list. Unlike QUUid, QUniqueId is only unique upon a single device.

Ids are generated with a QUuid Context identifier and an index. The context identifier should be generated with a program such as quuid and represent the data store or context within which the id is generated. For example, XML, SQL , and SIM card storage for QContacts each have their own Context Identifier. The Context is mapped to a 4bit value allowing for 255 contexts for a single device.

The usual approach to constructing QUniqueIds is:

    QUniqueIdGenerator g(myContext());
    id = g.createId();

QUniqueIdGenerator is suitable in cases where you need to generate an average of 100 identifiers a day or less and require a low number of bits when stored. If your program is likely to need more than 100 unique identifiers a day or does not benefit from the low 32bit storage size it is recommended you look at another type of unique id, such as QUuid.

Member Function Documentation

QUniqueIdGenerator::QUniqueIdGenerator ( const QUuid & context )

Create a QUniqueIdGenerator with the scope context. It is recommended that the context be hard coded into your application. Generating context at run time is redundant.

QUniqueIdGenerator::QUniqueIdGenerator ( const QUniqueIdGenerator & other )

Create a deep copy from other.

QUniqueIdGenerator::~QUniqueIdGenerator ()

Destroy QUniqueIdGenerator.

QUniqueId QUniqueIdGenerator::constructUniqueId ( uint index )

Constructs a identifier based on index for the context of the generator. Does not ensure the constructed id is not in conflict with past or future QUniqueId objects.

See also createUniqueId().

QUniqueId QUniqueIdGenerator::createUniqueId ()

Generate and return a new identifier. The identifier will be unique from other ids generated with this function.

See also constructUniqueId().

bool QUniqueIdGenerator::isValid () const

Returns true if the generator is valid and can be used to create or construct unique ids. Otherwise returns false.

This function should only be needed during debugging as only a generator constructed using null QUuids will be invalid.

uint QUniqueIdGenerator::mappedContext ( const QUuid & context )   [static]

Returns a locally mapped int for context, suitable for checking if a QUniqueId was generated from an equivalent QUniqueIdGenerator.

See also QUniqueId::mappedContext().

QUniqueId QUniqueIdGenerator::temporaryID ( uint index )   [static]

Creates a temporary identity based on index.

Temporary identifiers are scoped so as not to conflict with other identifiers.
