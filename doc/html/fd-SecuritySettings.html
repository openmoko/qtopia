Content: Installed Software: Security Settings
Functional Description

The Security Settings application allows the user to set restrictions on access to the device including both physical access and syncing from hosts. The security features of the Qtopia phone include:

  • availability of a user-specified PIN number to control access to the keypad and the SIM card
  • set up of secure synchronization between the device and a personal computer or network
  • automatic display lock option in the event of prolonged activity.

Features

  • Locks
  • Network sync Restriction

Locks

The user is able to activate and set the Phone
and the SIM PIN access codes. To successfully change a code the user is required
to enter the existing access code.

Network sync Restriction

The user is able to use Security Settings to limit the device to
synchronizing only with specified hosts.
