Detailed Description

The QSDAPSearchResult class encapsulates a result of a SDAP query.

The QSDAPSearchResult class encapsulates a result of a SDAP query, which is returned by the QSDAP object. The query consists of a list of QSDPService objects.

See also QSDAP and QSDPService.
