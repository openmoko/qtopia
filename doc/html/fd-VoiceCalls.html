Message Waiting Indications for Voice Mail

NOT YET RELEASED

Message waiting is a mechanism for telephone services to advertise to a handset that there are voice mail messages waiting for the user.

SIP-based systems typically use the mechanisms described in RFC 3842, whereas GSM systems typically use CPHS.
