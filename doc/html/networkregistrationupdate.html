Detailed Description

The NetworkRegistrationUpdate class provides notifications for the user when the network registration of any network (e.g. VoIP or Cell) changes.

The notifications are posted to the MessageBoard. Therefore any component that displays the message board content will receive the registration updates.

This class is a Qt Extended server task and cannot be used by other Qt Extended applications.

See also MessageBoard.
