Detailed Description

The QTaskListView class provides a list view widget with some convenience functions for use with QTaskModel.

The QTaskListView is a QListView with some additional functions to aid in using a list view of a QTaskModel.

Returns the list of complete tasks selected from the view. If a large number of tasks might be selected this function can be expensive, and selectedTaskIds() should be used instead
