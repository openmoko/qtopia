Per-media Database

Removable storage may be configured so that the Qt Extended Document System maintains a separate database for each device, stored on the physical media itself. The document system takes responsibility for determining which database content belongs to and merging the results from queries against multiple databases, giving the appearance of working against a single source.

Storing the databases on removable media allows content information to be persisted when the media is removed, this improves scan time as only changes need be written to the database, and prevents loss of user assigned information such as categories.

Removable Storage Scanning

Qt Extended scans removable storage upon insertion and populates the Document system with the documents found on the storage media.
