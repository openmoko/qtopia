Display
Functional Description

Qtopia supports a wide range of display sizes and orientations.

High Resolution Phone

Qtopia Phone supports and is tested with resolutions from 176x220 up to 240x320 and has been designed to scale to higher resolutions with little manual adjustment necessary.

Fonts sizes are chosen based on the resolution (DPI) of the display. This ensures that a 9pt font is the same real size independent of the display hardware.

Icons are supplied in SVG format rather than pixmaps, and either used directly, or used to generate pixmaps of the correct sizes at install time.

The Phone Widget style also uses the DPI of the display to ensure that UI elements are sized correctly for the display.

The combination of fonts and icons sized correctly for the resolution of the target device ensures that Qtopia appears as it was intended on many devices with minimal device specific modifications. This simplifies the task of developers and testers both now and in the future.
