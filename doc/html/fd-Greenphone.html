Enablers: SDK: Greenphone
Functional Description

Qtopia Greenphone is a Linux mobile development device. It is an optional hardware add-on to the SDK targetted at developers wishing to develop software for 'pure' Qtopia devices rather than any custom OEM hardware. It is as far as possible based purely on released Qtopia versions.

The Greenphone supports all Qtopia functionality, with the following exceptions and caveats:

  • IrDA - no hardware
  • MultipleDisplays - no hardware

Features

  • Bluetooth
  • Bluetooth Headset

Bluetooth

Hardware support for the Greenphone Bluetooth adapter is provided via a backported BlueZ stack to the Greenphone kernel.

Bluetooth Headset

Qtopia Greenphone supports Bluetooth Headsets. The user is able to utilize a Bluetooth Headset for GSM phone calls, audio playback and audio recording.
