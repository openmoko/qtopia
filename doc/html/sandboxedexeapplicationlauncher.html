Detailed Description

The SandboxedExeApplicationLauncher class supports launching untrusted downloaded application executables.

Note: This class is only relevant if SXE is enabled

The SanboxedExeApplicationLauncher class provides the ApplicationLauncherType implementation for simple but untrusted executable based applications (which have been downloaded and installed via packagemanager). The executable is run under sandboxed conditions to minimize the potential damage the application may cause.

The SandboxedExeApplicationLauncher class provides the SanboxedExeApplicationLauncher Task.
