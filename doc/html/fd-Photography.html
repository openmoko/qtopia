Photography
Functional Description

The Camera application allows the user to take photos using an integrated camera. In addition to this the user is able to quickly view recent photos, browse all photos, edit photos, take photos for contacts, send photos, and adjust camera settings.

Features

Supported Formats

Camera supports the JPEG image format.

Take Photo

The user is able to take a photo using an integrated camera and preview the photo using the viewfinder.

View Recent Photo

The user is able to quickly access and view the five most recent photos.

Browse Photos

The user is able to browse through all photos taken by Camera. This service is provided by the Pictures application.

Edit Photo

The user is able to rotate, crop and adjust the brightness of photos taken by Camera. This service is provided by the Pictures application.

Delete Photo

The user is able to delete photos taken by Camera.

Photo Identifier

The user is able to take a photo to be used as an identifier for a contact. This service is provided by the Contacts application.

Send Photo

The user is able to send photos via MMS. This service is provided by the Messages application.

Adjust Camera Settings

The user is able to adjust the capture settings of photos and to adjust the resolution, quality and storage location of the capture.

Dependencies

Integrated Camera

Camera requires the device be equipped with an integrated camera with Video4Linux support in order to function.

Pictures

Camera uses services provided by Pictures for browsing and editing photos.

Contacts

Camera uses services provided by Contacts for applying photo identifiers to contacts.

Messages

Camera uses services provided by Messages for sending MMS messages.
