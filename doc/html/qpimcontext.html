Detailed Description

The QPimContext class represents a storage context of PIM data.

This includes storage such as SIM Card contacts or data stored on the device in Qtopia's native format. The class can be used to perform operations that relate to a specific context of PIM data.

QPimContext should not be subclassed directly. Instead, one of the data type specific contexts should be used. These are QContactContext, QTaskContext and QAppointmentContext.

Currently there is no way for applications to implement their own contexts. This feature is being considered for future versions of Qtopia.

Generally, it is easier to use a QPimModel to access QPimRecords.

See also QPimRecord, QPimSource, and QPimModel.
