User Interface: Input Methods: Hard Keys
Functional Description

Hard keys are the physical keys and key-like switches on the device. The actions invoked by the keys are determined by the "defaultbuttons.conf" file.

See also Services, Speed Dialing.

Features

  • Global Volume Control Keys
  • Volume Keys

Global Volume Control Keys

The mediaplayer responds to volume keys regardless of whether it is the active window.

Volume Keys

Media player responds to hardware volume keys. From anywhere within the media player the user can change the volume by pressing the hardware volume keys. These keys are typically mapped to the Qt::Key_VolumeUp and Qt::Key_VolumeDown keys.
