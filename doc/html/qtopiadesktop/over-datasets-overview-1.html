Datasets Overview

In order for two or more plugins to pass data between them, they must agree on a structure for the data. Qtopia Sync Agent uses XML to pass data between plugins. The structure of the XML data is determined by the dataset that it belongs to.

Qtopia Sync Agent supports a number of datasets. They are documented in Datasets.

Extra datasets can be added. As long as two or more plugins handle a dataset, it will be synchronized.
