200

Usage

This command includes an untranslated status message that can be ignored.

    200 Command okay

Description

Sent from the device to the desktop. This is sent in response to a successfully handled CALL or CALLB command.

Note that sending of this command can be suppressed with an argument to the HELPER_INIT command.

Note that the desktop does not send these commands to the device.

See also qdsync Protocol.
