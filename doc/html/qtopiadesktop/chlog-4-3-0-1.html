Changes for 4.3.0

Overview

As this is the first release of Qtopia Sync Agent, this page describes differences between Qtopia Desktop 2.2 and Qtopia Sync Agent.

  • Ground-up rewrite.
  • Supports TCP/IP and Serial connections.
  • Enables synchronization between Qtopia and Outlook.
  • Outlook security dialogs are avoided.
  • Not compatible with Qtopia versions before 4.3.0.
  • No longer supported on Windows versions prior to 2000.

Known issues

  • Appointments starting and ending at midnight are mis-reported as all day events.
