Detailed Description

The BtFtpServiceTask class provides server side support for the Bluetooth FTP profile.

This task listens for incoming Bluetooth FTP connections and handles them.

The BtFtpServiceTask class provides the BtFtpServiceTask task. This class is part of the Qtopia server and cannot be used by other QtopiaApplications.
