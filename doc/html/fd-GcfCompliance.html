GCF Compliance
Functional Description

Introduction

GCF stands for "Global Certification Forum". GCF is a partnership between network operators and terminal manufacturers that has been formed with the objective of establishing an independent program to ensure global interoperability of GSM terminals.

The GCF specifications are huge, covering many areas from the behavior of the radio in the modem to user-interface requirements. Much of this would be tested by the handset and modem manufacturers, and not us, but there may be implications for Qtopia. e.g. A GCF test that checks if multi-party calls can be done by the modem according to the radio requirements isn't very useful if there is no UI to access the feature.

The Munich Wiki has quite a bit of information about GCF, and the work that has been done so far to get a handle on how well Qtopia supports it.

This goal of this specification page is to bring together the various pieces of information that are relevant to Qtopia development to generate requirements for other areas of the specification.
