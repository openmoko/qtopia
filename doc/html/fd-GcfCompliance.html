Introduction

GCF stands for "Global Certification Forum". GCF is a partnership between network operators and terminal manufacturers that has been formed with the objective of establishing an independent program to ensure global interoperability of GSM terminals.

The main technical GSM specification is 3GPP TS 51.010, which summarizes the requirements in the form of test cases. These test cases cover many areas, from the behavior of the radio in the modem to user-interface requirements.

Qtopia is mainly concerned with the user-interface requirements of GCF, with the radio aspects handled by the modem. The user-interface requirements are documented in a number of GSM specifications, including:

  • Mobile Station (MS) Features (3GPP TS 02.07)
  • Man-Machine Interface (MMI) of the User Equipment (UE) (3GPP TS 02.30, 3GPP TS 22.030)

Qtopia currently conforms to a subset of these specifications. For full details of compliance in specific versions, contact Trolltech.
