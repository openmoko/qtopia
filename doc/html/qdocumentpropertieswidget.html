Detailed Description

The QDocumentPropertiesWidget class provides controls for modifying the properties of a QContent.

The QDocumentPropertiesWidget allows modification of the name, location, and category of a file associated with a particular QContent. The file can also be deleted, copied and beamed from the QDocumentPropertiesWidget.

Note: The API of this class will change as a result of the new document model design.

See also QDocumentPropertiesDialog.
