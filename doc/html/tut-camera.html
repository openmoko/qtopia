Tutorial: Extending the Camera Device Model

The Qtopia camera application has support for VideoForLinux(V4L) 1 and 2 camera devices

To use a camera device with Qtopia's camera application there are 2 options:

  1. Either derive from the VideoCaptureDevice abstract base class and implement all virtual functions e.g
        namespace camera {

    class MyCameraDevice : public VideoCaptureDevice
    {
        MyCameraDvice(){}

        bool hasCamera () ...
        void getCameraImage(QImage& img, bool copy = false) = 0;

        QList<QSize> photoSizes();
        QList<QSize> videoSizes();

        QSize recommendedPhotoSize(0;
        QSize recommendedVideoSize();
        QSize recommendedPreviewSize();

        // etc...
    }

    }
  2. Or extend and customize one of the existing Video For Linux classes, namely V4L1VideoCaptureDevice, V4L2VideoCaptureDevice
