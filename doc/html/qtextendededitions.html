Reference designs

The concept

A reference design provides a preselection of modules for a particular use case and hardware. Currently Qt Extended supports the following reference designs:

  • Touchscreen Mobile Phone
  • Video IP deskphone

A reference design is directly connected to a particular hardware. Therefore a particular reference design is chosen via configure's -device flag.

Available reference designs

NameDescription
Video IP DeskphoneThe Video IP reference design includes the following modules:
  • Base
  • Connectivity
  • Development Tools
  • Essentials
  • IPComms
  • Media
  • Messaging
  • PIM
  • Telephony
  • UI

This reference design provides a user interface for IP based communication devices (such as a desk phone) which use a touchscreen display. The interface has been optimized towards finger based navigation.

Touchscreen Mobile PhoneThe Mobile reference design includes the following modules:
  • Base
  • Bluetooth
  • Cell
  • Connectivity
  • Development Tools
  • Essentials
  • Games
  • InputMethods
  • IPComms
  • Location
  • Media
  • Messaging
  • Package Management
  • PIM
  • Telephony
  • UI
  • QtUiTest

This reference design provides software that is suitable for development a fully featured phone stack. It incorporates the most extensive range of connectivity features combined with applications supporting most use cases.
