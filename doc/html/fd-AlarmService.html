Alarm Service
Functional Description

Alarms are used for real time events such as Calendar events and the Daily Alarm.

Qtopia communicates with atd to schedule alarms. A sample implementation of this standard Linux components is included.

The atd is responsible for waking the device from "off" and other deep suspend modes. It usually does this by interaction with the Real Time Clock, although this can be hardware-dependent.
