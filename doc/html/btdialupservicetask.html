Detailed Description

The BtDialupServiceTask class provides server side support for the Bluetooth DUN profile.

This task listens for incoming Bluetooth DUN connections, forwards the request to Qtopia's modem emulator and manages the life time of these connections. This task relies on QBluetoothSerialPortService.

The BtDialupServiceTask class provides the BtDialupServiceTask task. This class is part of the Qtopia server and cannot be used by other QtopiaApplications.

See also QBluetoothSerialPortService.
