Qtopia - 1.7.0 Release Notes

The Qtopia 1.7 series adds a small number of key features upon the 1.6 series.

The main features added to Qtopia 1.7 are:

  • Applications can be complied to allow Quicklaunching, which improves application startup time considerably by pre-loading and initializing core functionality.
  • Improved document loading and scanning of removable media.
  • Server source code improved. The code is better divided between the GUI and non-GUI parts, allowing custom launcher user interfaces to be written.
