Detailed Description

The ApplicationShutdownTask class provides a way of terminating all running Qtopia applications before the Qtopia server itself terminates.

This shutdown handler instance is responsible for the proper termination of all running Qtopia applications. Usually this task is only performed when the user requests a reboot or restart of the entire Qtopia system.
