Detailed Description

The QDocumentPropertiesDialog class allows the user to examine attributes associated with an QContent object.

The QDocumentPropertiesDialog uses a QDocumentPropertiesWidget to allow the user to examine and modify attributes associated with a file.

Note: The API of this class will change as a result of the new document model design.

See also QDocumentPropertiesWidget.
