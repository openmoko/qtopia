Plugins

Qtopia plug-ins are provided to extend Qtopia. They are integrated into Qtopia and provide additional functionality to applications while the plug-in framework provides a well-defined development environment for new features.
Plug-in component architecture is implemented via an Integration Layer which is a framework that abstracts device-specific functionality from the rest of Qtopia.

Qtopia includes the following interfaces for plug-ins:

  • Window Decoration styles
  • Image Formats
  • Input Methods
  • Network interfaces
  • Modem drivers
  • SQL drivers
  • GUI Styles
  • Text encodings
  • Media Codecs
