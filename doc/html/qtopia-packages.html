Introduction

To develop commercial Qtopia applications you have two options: Qtopia Binary SDK or the Qtopia Developer License. To install and distribute Qtopia on commercial devices you need the Qtopia OEM Kit and Qtopia Runtime licenses.

Commercial Packages

Qtopia OEM Kit

The Qtopia OEM Kit is targetted at companies that develop mobile devices running Qtopia. The Qtopia commercial OEM license permits OEMs, ODMs, and system integrators to build and sell Qtopia devices. To install and distribute Qtopia on consumer electronic devices you need the Qtopia OEM Kit. The Qtopia OEM Kit consists of a number of Qtopia Developer Licenses and the full Qtopia source code. The Qtopia OEM Kit software licenses allow the customer to develop Qtopia applications as well as adapt Qtopia to fit on their device(s).

Qtopia Binary SDK

The Qtopia Binary SDK is targetted at small companies involved in commercial Qtopia application development and who do not require support or maintenance. The Qtopia Binary SDK allows customers to develop Qtopia applications using Qt/Embedded and the Qtopia API. It comes in binary form and does not include support or maintenance.
