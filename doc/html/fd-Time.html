Utilities: Time - Qtopia 4.3 Roadmap
Functional Description

The Date/Time application allows the user to set current time, date and time zone.

Features

  • Automatic time adjustment
  • Localized Format

Automatic time adjustment

If an external time source is available, such as GSM Cell Tower Time, the user can indicate that they want the time to automatically be adjusted based of that time source.

Localized Format

The default format is that defined by the system for the user's current Language Selection.
