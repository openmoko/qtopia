PIM
Functional Description

Qtopia provides a high-level collection of functionality associated with Personal Information Management (PIM); that is, information relevant primarily to the user and their life. For example: Contacts, Schedule (Calendar and Tasks), Notes (including Voice Notes). This information generally changes regularly so a synchronization process is provided for other devices, including the desktop, to keep all instances of the information up-to-date.

  • Calendar
  • Contacts
  • Notes
  • Tasks
  • Voice Notes
