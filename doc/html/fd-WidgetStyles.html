User Interface: Personalization: Widget Styles - Qtopia 4.3 Roadmap
Functional Description

Widget styles allow the appearance of the controls to be customized.

Features

  • Default Phone Widget Style

Default Phone Widget Style

The default phone widget style is a style optimized for small screen devices. It makes extensive use of anti-aliasing, gradients, alpha blending and other advanced graphics techniques to give provide a polished, modern look.

Unique features include:

  • A stronger highlight than most desktop styles, making it easier to navigate between widgets using a keypad.
  • Tabs with text and icons are rendered as two-line tabs to minimize the need for horizontal scrolling.
  • Widgets are scaled based on the device DPI (Dots Per Inch)
