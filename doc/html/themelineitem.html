Detailed Description

The ThemeLineItem class represents a visual line in a ThemedView.

The ThemeLineItem class implements the line element from the theme XML.

Normally you do not want to call this item's functions directly, but rather specify the relevant attributes and data for this item in the themed view XML.

See also line element.
