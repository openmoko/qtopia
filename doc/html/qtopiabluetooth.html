Bluetooth Classes

These are the public classes for Qtopia application development that are specific to Bluetooth.

QBluetoothAbstractServiceAbstract class for running a Bluetooth service within the Qtopia Bluetooth framework
QBluetoothAbstractSocketRepresents an abstract Bluetooth client socket
QBluetoothAddressRepresents a bluetooth address (bdaddr)
QBluetoothAudioGatewayInterface to a Bluetooth Audio Gateway
QBluetoothDeviceSelectorAllows the user to perform a bluetooth device discovery and select a particular device
QBluetoothLocalDeviceRepresents a local bluetooth device
QBluetoothObexServerListens for OBEX client connections
QBluetoothPasskeyAgentRepresents a passkey entry proxy
QBluetoothPasskeyRequestEncapsulates a Bluetooth PIN request
QBluetoothRemoteDeviceRepresents a remote bluetooth device
QBluetoothReplyWrapper object that contains a reply, and an error indicator
QBluetoothRfcommSerialPortRepresents a serial device which enables access to a bluetooth serial port protocol
QBluetoothRfcommSocketRepresents an RFCOMM client socket
QBluetoothScoSocketRepresents an SCO client socket
QBluetoothServiceControlServerControl interface for a QBluetoothAbstractService using Qtopia IPC mechanisms
QBluetoothServiceControllerAccess to information about Bluetooth services within the Qtopia Bluetooth services framework
QObexPushClientEncapsulates an OBEX PUSH client
QSDAPEncapsulates operations performed by the SDAP profile
QSDAPSearchResultEncapsulates a result of a SDAP query
QSDPAllows interaction with the local SDP server
QSDPServiceRepresents a bluetooth SDP record
QSDPUUIDRepresents a unique identifier used by the SDP protocol
