Detailed Description

The QPhoneProfile::Schedule class controls timed auto-activation of a profile.

Qtopia profiles may be automatically activated on certain times and dates. The QPhoneProfile::Schedule class represents the times at which the profile should be activated. A profile's schedule can be read through the QPhoneProfile::schedule() method, and set through a QPhoneProfile::setSchedule() call.
