Detailed Description

The QtopiaStyle class encapsulates the common Look and Feel of a Qtopia GUI.

This class implements some of the widget's look and feel that is common to all Qtopia GUI styles.

See also QWindowsXPStyle, QMacStyle, QPlastiqueStyle, QCDEStyle, and QMotifStyle.
