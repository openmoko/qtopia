Detailed Description

The QTaskSelector class provides a way of selecting a task from a QTaskModel.

The QTaskSelector dialog allows selecting an existing or new task.

Member Function Documentation

QTaskSelector::QTaskSelector ( bool allowNew, QWidget * parent )

Constructs a QTaskSelector with parent parent. If allowNew is true will also provide the abilityt to select that a new task should be created.

bool QTaskSelector::newTaskSelected () const

Returns true if the dialog was accepted with a new task selected. Otherwise returns false.

QTask QTaskSelector::selectedTask () const

Returns the task that was selected. If no task was selected returns a null task.

void QTaskSelector::setModel ( QTaskModel * m )

Sets the model of representing tasks that should be selected from to m.

bool QTaskSelector::taskSelected () const

Returns true if the dialog was accepted with an existing task selected. Otherwise returns false.
