Detailed Description

The QHandsetAccessoryProvider class provides an interface for handset devices to integrate into Qtopia.

The QHandsetAccessoryProvider class provides an interface for handset devices to integrate into Qtopia. Handset devices inherit from this and call setMode(), setSpeakerPhone() and setTty() to indicate the level of functionality that is supported.

See also QHandsetAccessory.
