Detailed Description

The GSM class is a placeholder for licensing information. GSM is a codec optimised for recording human speech, and is used by the MediaRecorder.

The GSM code was extracted from verison 12.17.3 of the Sox sound tools. The author of Sox is Chris Bagwell. Sox is distributed under the terms of the GNU General Public License. More information can be found at http://spies.com/Sox/
