Detailed Description

The QSerialCommAccessoryProvider class provides an interface for serial communication devices to integrate into Qtopia.

The QSerialCommAccessoryProvider class provides an interface for serial communication devices to integrate into Qtopia. Serial communication devices inherit from this class and call setConnectionType() to indicate the level of functionality that is supported.

See also QSerialCommAccessory.
