Detailed Description

The QServiceIndicationReader class reads Service Indication (SI) documents

The QServiceIndicationReader class extends QWbXmlReader to provide support for reading documents in the WAP Service Indication (SI) language. This language is defined in the WAP standard wap-167-serviceind-20010731-a.pdf.

See also QWbXmlReader.
