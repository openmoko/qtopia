Qtopia - 1.6.2 Release Notes

Qtopia 1.6.2 is a maintenance release for the Qtopia 1.6 series. Since it is a maintenance release, all changes are bug fixes for existing functionality. No new features have been added.

The total number of issues resolved between 1.6.1 and 1.6.2 is 144. The bugs fixed are shown below.
