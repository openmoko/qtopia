Qtopia - 1.6beta3 Release Notes

  • High resolution support

    We have made improvements to Qtopia applications to improve their appearance on high resolution screens.

  • Integration of PDA and Desktop PIM applications

    We are working to integrate the PIM applications from Qtopia into Qtopia Desktop as plugins. This will lead to a more unified experience for the user. In this release Qtopia Desktop is currently unstable and we do not recommend its use, however the Qtopia applications are still compatible with previous releases of Qtopia Desktop.

  • Input Method support enhancements

    We have made improvements to the API for Input Methods, making it easier to write Input Methods for Qtopia.

  • Ffmpeg plugin for Media player

    The media player now uses a new ffmpeg based plugin. This allows for better support for a wider range of audio and video formats.
