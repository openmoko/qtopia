Sample WebKit-based browser

Qt Extended includes a sample browser based on QtWebKit. It includes key integration points:

  • Telephony integration (tel: URLs)
  • EMail integration (mailto: URLs)
  • External application integration (handling content by MIME type)
  • Qt Extended application integration, providing JavaScript access to:
    • Contacts
    • Call History
    • Dialer
    • Calendar
    • Messaging
  • Shortcuts integration (add bookmarks to "speed dial")

These integrations demonstrate the mechanisms by which any web browser would integrate with the rest of Qt Extended.
