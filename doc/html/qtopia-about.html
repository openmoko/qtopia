Introduction

Qtopia is Trolltech's comprehensive application platform for embedded Linux based PDAs, mobile phones, web pads, and other mobile computing devices. The Qtopia product line consists of three editions and is designed to provide and create graphical user interfaces for Linux-based devices.

Qtopia is both a platform and turn key product offering unprecendented flexibility and a wealth of choices for manufacturers. It is the Trolltech solution to straightforward, compact, and versatile computing for consumer electronic devices.

Qtopia Editions

Qtopia is a windowing system with a range of productivity, PIM and messaging applications complete with a desktop synchronization framework. The Phone and PDA Editions also include a range of standard utilities and games, and are capable of multi-language and wireless support.

Commercial vs Free Editions

Trolltech offers two categories of Qtopia; the commercial and the free software.

The commercial versions of Qtopia are intended for the creation of commercial, proprietary and non-free software and are available with a number of licences and license agreements to suit most customers. Please refer to Qtopia Commercial Edition: Commercial SDKs for details.

The free version of Qtopia is intended to support and encourage the development of free software that will be released using a recognized open source license. Please note that free in the context of software refers to freedom as opposed to price. For more details about free software please go to http://www.fsfeurope.org/documents/whyfs.en.html or to learn more about Trolltech's free editions of Qtopia plaese refer to the Qtopia Free Editions page.

Qtopia Phone Edition

Qtopia Phone Edition for embedded Linux is designed for smartphones and feature phones. It offers a comprehensive range of productivity, personal, messaging, and entertainment tools and combines them with the now standard phone functions.

Qtopia PDA Edition

Qtopia PDA Edition is designed for Linux OS-based PDAs (Personal Digital Assistants). Offering most of the same personal and entertainment tools as the Phone Edition, the PDA Edition expands the range of productivity tools and takes advantage of the enhanced functionalities of the device.

Qtopia Desktop

Qtopia Desktop is a companion package that allows the user to synchronize their data. It contains versions of the PIM application found in Qtopia. It is extensible by third parties.

Qtopia features:

As a system, Qtopia, offers the following features:

  • Windowing system
  • Synchronization framework
  • Development environment
  • Localization support
  • Games and multimedia
  • PIM applications
  • Input methods
  • Personalization options
  • Productivity applications
  • Internet applications
  • Java integration
  • Wireless support

The Future

Our lifestyles are becoming increasingly mobile creating a growing demand for personal digital media and information environments that we can carry with us. This is driving the convergence of computing and communications technologies and seeing the strong emergence of a new class of devices.

The micro-convergence of devices will continue to blur the distinction between mobile phones and personal digital assistants making it increasingly important for manufacturers to be using device independent platforms such as Qtopia. The Trolltech Qtopia product line supports this next generation of wireless 3G devices that combine voice and Internet capabilities into wireless handsets. The third generation of cellular technology provides high performance, low-power processors that offers consumers the perfect way to take their information and access to the digital world with them wherever they go.
