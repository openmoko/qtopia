Qtopia Modules

Qtopia 2.1 consists of the following libraries:

library description
qtopia The basic Qtopia library. Binary compatible with Qtopia 1.6
qtopia1 Extension to the basic Qtopia library. Binary compatible with Qtopia 1.7
qtopia2 Extension to the basic Qtopia library. Available in Qtopia 2.0 onwards
qtopiacalc Contains calculator funtions. Binary compatible with Qtopia 1.6
qtopiaphone Contains the actual telephony functions. Available in Qtopia 2.0 onwards
qtopiapim The basic Qtopia PIM library. Binary compatible with Qtopia 1.6
qtopiapim1 Extensions to the basic Qtopia PIM library. Available in Qtopia 2.0 onwards
