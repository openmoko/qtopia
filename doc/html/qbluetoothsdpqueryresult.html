Detailed Description

The QBluetoothSdpQueryResult class encapsulates a result of an SDAP query.

The QBluetoothSdpQueryResult class encapsulates a result of an SDAP query, which is returned by the QBluetoothSdpQuery object. The query consists of a list of QBluetoothSdpRecord objects.

See also QBluetoothSdpQuery and QBluetoothSdpRecord.
