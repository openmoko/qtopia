hint=pics

Usage

    foo.files=pics/*
    foo.path=/pics/$$TARGET
    foo.hint=pics

Description

Register an image install. This uses installpic to install pictures. If you cannot use pics/* then use pics/*.png pics/icons pics/i18n so that icons and translated images can still be handled.

See also Install Hints and installpic Algorithm.
