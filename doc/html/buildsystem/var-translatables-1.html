TRANSLATABLES

Usage

    COND_SOURCES=foo.cpp
    TRANSLATABLES+=$$COND_SOURCES
    cond:SOURCES+=$$COND_SOURCES

Description

TRANSLATABLES lists all of the translatable files for a project.

See also Internationalization, no_tr, and no_auto_translatables.
