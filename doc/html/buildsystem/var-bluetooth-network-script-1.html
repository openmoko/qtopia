BLUETOOTH_NETWORK_SCRIPT

Usage

    BLUETOOTH_NETWORK_SCRIPT=SuSE10.2
    BLUETOOTH_NETWORK_SCRIPT=$$PWD/btdun-network

Description

The Bluetooth DUN network plugin installs a helper script to control the network interface. There are several reference scripts available and this variable lets you pick one to install (usage case 1). It also lets you choose a device-specific script if the included scripts are not suitable (usage case 2). The reference scripts are in src/plugins/network/bluetooth.

Note: This variable is only useful when set in <device>/config.pri.

See also Variables, Device Profiles, BLUETOOTHPAN_NETWORK_SCRIPT, DIALING_NETWORK_SCRIPT, and LAN_NETWORK_SCRIPT.
