Changes for 4.1.7

Overview

This page describes changes to the 4.1.7 build system.

Changes

  • make sdk actually works now.
  • A project can be built debug/release independantly of Qtopia Core's debug/release status.
  • Packages for external projects go into <project root>/pkg instead of <project dir>. Helpful for larger external trees.
  • Don't install static libraries into the image.
  • New hint=background for images that should be scaled to the screen size.

See also What's New?.
