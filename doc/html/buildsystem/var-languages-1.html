LANGUAGES

Usage

    LANGUAGES=de ar

Description

This variable is set by the build system based on your configure switches and should not generally be changed.

An example of when this needs to be changed is for third parties that want to install translations for a specific set of languages rather than the ones selected when the SDK was built.

See also Variables and Internationalization (build system).
