hint=themecfg

Usage

    foo.files=myconf.conf
    foo.path=/etc/themes
    foo.trtarget=MyTheme
    foo.hint=image themecfg

Description

Specifies a theme configuration file so that the theme's translatables strings are handled. This causes the theme_lupdate script to run. theme_lupdate is similar to nct_lupdate but it handles all of the elements specific to themes.

See also Install Hints, Internationalization (build system), and Translating Themes.
