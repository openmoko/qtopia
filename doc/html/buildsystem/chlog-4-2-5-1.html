Build system changes for 4.2.5

Overview

This page describes changes to the 4.2.5 build system.

  • You can now install an SDK into a sandbox. See Installing into a sandbox for details.
  • custom-qtopia.h is now included in SDKs. This is the file described in custom.h/custom.cpp.
