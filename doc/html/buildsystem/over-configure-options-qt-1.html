configure Options (Qt)

These options can be passed by using -extra-qt-config. For other switches that Qtopia supports see configure Options. Note that quoting can get a bit tricky with -extra-qt-config so be careful. The -extra-qt-config switch can be passed multiple times which can simplify quoting issues in many cases.

For example, the following two are equivalent.

    configure -extra-qt-config "-arg -arg word -arg 'words and stuff'"
    configure -extra-qt-config -arg\
              -extra-qt-config -arg=word\
              -extra-qt-config "-arg 'words and stuff'"
