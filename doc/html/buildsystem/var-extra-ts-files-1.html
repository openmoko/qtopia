EXTRA_TS_FILES

Usage

    EXTRA_TS_FILES+=QtopiaHandwriting

Description

Translations are generally handled for each .pro file but there are many common resources in Qtopia that are processed in one go from the server's .pro file. However, apps are still responsible for installing their own .ts files. This variable simplifies the process since you only need to list the name of the .ts file. For third party projects, this is not very useful since the server will not be creating .ts files for you.

Note that EXTRA_TS_FILES is deprecated and the more flexible hint=extra_ts should be used instead.

See also Variables, hint=nct, and no_tr.
