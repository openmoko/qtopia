hint=help

Usage

    foo.source=help
    foo.files=foo.html
    foo.hint=help

Description

Register a help install. This uses installhelp so that both untranslated help files (help/html/*) and translated help files (help/<lang>/html/*) are installed.

See also Install Hints.
