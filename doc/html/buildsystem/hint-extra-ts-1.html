hint=extra_ts

Usage

    foo.file=qt___LANG__
    foo.outfile=qt
    foo.source=$$QT_DEPOT_PATH/translations
    foo.hint=extra_ts
    INSTALLS+=foo
    foo.file=QtopiaDefaults
    foo.source=$$QTOPIA_DEPOT_PATH/i18n/__LANG__
    foo.hint=extra_ts
    INSTALLS+=foo

Description

Register extra .ts files. The first example shows how to install Qt translations. The second example shows the replacement for the EXTRA_TS_FILES variable.

The string __LANG__ is replaced with the current language and can appear in the file name or the source path. If the output file needs to have a different name to the input file (as in the first example) outfile should be set.

The first example will take files named $$QT_DEPOT_PATH/translations/qt_<lang>.ts and install them into /i18n/<lang>/qt.qm.

The second example will take files named $$QTOPIA_DEPOT_PATH/i18n/<lang>/QtopiaDefaults.ts and install them into /i18n/<lang>/QtopiaDefaults.qm.

See also Install Hints.
