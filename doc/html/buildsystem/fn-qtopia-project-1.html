qtopia_project()

Usage

    qtopia_project(keyword ... keyword)

Description

All projects should have one of the following:

  • app

    This is an application.

  • lib

    This is a library.

  • plugin

    This is a plugin.

  • subdirs

    This project lists other projects to build.

  • stub

    This project does not build anything.

These keywords specify the Qt version to use:

  • external

    This is a non-Qt/Qtopia Core project.

  • embedded

    This is a Qtopia Core project.

  • desktop

    This is a Qt project.

These keywords are useful for external projects:

  • qtopia

    This is a Qtopia project.

  • qtopiadesktop

    This is a Qtopia Desktop project.

Qtopia and Qtopia Desktop projects may wish to use:

  • core

    Don't depend on the usual libs (ie. libqtopia/libqtopiadesktop).

The etc/themes root has the following keywords:

  • theme

    This is a theme (alias for stub).

The src/plugins/designer root has the following keywords:

  • designer

    Make a designer plugin (use both designer and plugin).

Note that designer plugins are not tested and probably do not work.

There are some internal types in use.

  • qtopiacore

    This is a stub project used to control building of Qt and Qtopia Core projects.
