hint=background

Usage

    foo.files=mybg.png
    foo.path=/pics/wallpapers
    foo.hint=background
    INSTALLS+=foo

Description

Specify images that should be scaled to the size of the screen (as specified by configure -displaysize WxH). This is most useful for theme backgrounds.
