idep()

Usage

    idep(LIBS+=-l$$TARGET)
    idep(INCLUDEPATH+=/usr/include/foolib,INCLUDEPATH)

Description

idep() is like dep() but works for indirect dependants too. This is useful for things like INCULDEPATHS that your headers rely on and DEFINES. All LIBS additions should generally be with idep() so that the final link can find indirectly-used libraries.

See also Functions (build system), Dependencies, dep(), and depends().
