AVAILABLE_LANGUAGES

Usage

    AVAILABLE_LANGUAGES=en_US de ar

Description

This variable is set by the build system based on the detected languages and should not generally be changed.

An example of when this needs to be changed is for third parties that want to provide translations for a specific set of languages rather than the ones detected when the SDK was built.

See also Variables and Internationalization (build system).
