resolve_include()

Usage

    PREFIX=BLAH
    BLAH_SOURCES=foo.cpp
    BLAH_PRIVATE_SOURCES=bar.cpp
    resolve_include()

Description

The value in PREFIX is used to locate FORMS, HEADERS and SOURCES. It also collapses any PRIVATE_[FORMS|HEADERS|SOURCES] so that they are seen by qmake.

See also Functions, Libraries, and PREFIX.
