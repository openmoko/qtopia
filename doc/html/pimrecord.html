Detailed Description

The PimRecord class is the base class for PIM data recorded in the Qtopia database.

The Pim record class contains data that is common to all data used by the PIM applications, in particular a unique ID and a set of categories.

A PimRecord's ID is returned by uid() and is set with setUid(). Each category is represented by an integer. A PimRecord's categories are returned by categories() and are set with setCategories().

PimRecord comparisons are provided by operator==() and operator!=().

See also QPEPIM Library.
