Integration Layer
Functional Description

The integration layer is a framework which implements and abstracts device-specific functionality from the rest of Qtopia. It does this through a plug-in component architecture, allowing integrators to simply include or exclude components to tailor Qtopia to their particular requirements.

Features

Value Space

The Qtopia Value Space unifies the large number of distinct mechanisms that exist in Qtopia for access to conceptually hierarchal data. Examples of these mechanisms include INI files, active theming data, service mappings and network lists.

Complete documentation for the Value Space can be found in the Reference Documentation.
