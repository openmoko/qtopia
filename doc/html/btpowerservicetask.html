Detailed Description

The BtPowerServiceTask class provides the BtPowerService.

The BtPower service enables applications to notify the server of Bluetooth device useage, such that the server can intelligently manage the bluetooth device for maximum power efficiency.

The BtPower service is typically supplied by the Qtopia server, but the system integrator might change the application that implements this service.
