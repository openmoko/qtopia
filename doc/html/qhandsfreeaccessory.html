Detailed Description

The QHandsfreeAccessory class provides access to a handsfree accessory on the device.

The QHandsfreeAccessory class provides access to a handsfree accessory on the device. Handsfree device implementations should inherit from QHandsfreeAccessoryProvider.

See also QHandsfreeAccessoryProvider and QHardwareInterface.

Member Function Documentation

QHandsfreeAccessory::QHandsfreeAccessory ( const QString & id = QString(), QObject * parent = 0, QAbstractIpcInterface::Mode mode = Client )

Construct a new handsfree acessory object for id and attach it to parent. The object will be created in client mode if mode is Client, or server mode otherwise.

If id is empty, this class will use the default accessory that supports the handsfree interface. If there is more than one service that supports the handsfree interface, the caller should enumerate them with QHardwareManager::supports() and create separate QHandsfreeAccessory objects for each.

See also QHardwareManager::supports().
