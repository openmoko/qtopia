Detailed Description

The QContactAnniversaryContext class provides QAppointments based on the QContact anniversaries.

QContactModel uses this class to create synthetic QAppointments from QContacts that have an anniversary set. The QAppointments are automatically updated (or deleted) as the QContact changes.

See also QContactBirthdayContext, QContactModel, and Pim Library.
