Detailed Description

The QServiceLoadingReader class reads Service Loading (SL) documents

The QServiceLoadingReader class extends QWbXmlReader to provide support for reading documents in the WAP Service Loading (SL) language. This language is defined in the WAP standard wap-168-serviceload-20010731-a.pdf.

See also QWbXmlReader.
