Helix DNA Client multimedia codec library is integrated with Qtopia, allowing access to the many codecs available for Helix.

Qtopia uses Helix DNA Client 3.1 (Atlas).

TCK Compliance

Qtopia passes the RealNetworks' Helix Client Code TCK (excluding device-specific tests).
