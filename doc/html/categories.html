Categories

These classes provide interfaces to content categorization.

QCategoryDialogAllows users to select Categories with a dialog interface
QCategoryFilterFilters category sets
QCategoryManagerFunctionality for modifying the set of available Categories
QCategorySelectorAllows users to select Categories with a combobox interface
