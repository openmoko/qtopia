Bidirectional Interface Support

Qtopia 4 provides user interfaces that work for right-to-left (RTL) languages, such as Arabic and Hebrew. All user interface elements including text and widgets support bidirectional layouts. An implementation of the Bidi algorithm ensures that Arabic and Latin characters are properly displayed on the screen.
