Detailed Description

The QHardwareInterface class is a base-class for accessory classes.

QHardwareInterface is part of the accessory system which provides information about the available physical accessories. QHardwareInterface is used as a base-class for hardware accessory classes.

Default accessories are defined in <Qtopia Runtime Prefix>/etc/default/Trolltech/HardwareAccessories.conf. The file contains a list of interface names and the ID of the default accessory, for example:

    [Defaults]
    QBatteryAccessory = StandardBattery

See also QHardwareManager.
