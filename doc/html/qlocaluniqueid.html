Detailed Description

The QLocalUniqueId class provides a convience class for storing QUniqueId objects locally on a device with reduced processing and space usage.

It should not be used for when writing or reading id to another device or file that may be transfered.

See also QUniqueId and QUniqueIdGenerator.
