Clock
Functional Description

The Clock application provides three functions:

  • Displaying the current date and time.
  • Providing a stopwatch.
  • Allowing a daily alarm to be set and enabled/disabled.

Features

Clock

The application displays an analog clock, digital clock and the date in one view.

Stopwatch

The application displays an analog and digital stopwatch timer. The digital timer has a resolution of 1/100 second.

Start/Stop

The * hardware button is used to Start/Stop the stopwatch.

Lap/Reset

The # hardware button is used for the Lap/Reset function.

Lap/Split

Lap and Split times are displayed (up to 99 laps). The lap and split times are able to be reviewed while the stopwatch is running.

Daily Alarm

The application allows a daily alarm to be enabled/disabled and set. The time of day and days of the week may be specified.

When the alarm is activated an audible alarm can be sounded.

When the alarm is activated a dialog box is shown with the alarm details.
