Communications: Internet: 802.11 Wireless IP support - Qtopia 4.3 Roadmap
Functional Description

Wireless IP (WLAN) support is provided by a network plug-in and allows the use of managed and ad-hoc WLAN networks .

Features

  • Wireless Encryption

Wireless Encryption

Qtopia supports the following wireless encryption technologies:

  • WEP (40 bit and 128 bit)
  • WPA-PSK (TKIP & AES)
  • WPA-EAP (TLS, TLLS and PEAP)
