Detailed Description

The BluetoothPasskeyAgentTask class provides a default passkey agent for performing Bluetooth passkey authentications.

The BluetoothPasskeyAgentTask class implements a Qtopia global passkey agent. The internal implementation uses the QPasswordDialog to ask the user for the passkey.

This class is part of the Qtopia server and cannot be used by other QtopiaApplications.

See also QBluetoothPasskeyAgent.
