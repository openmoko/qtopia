Detailed Description

The TaskbarNamedAppletInterface class provides an interface for Qtopia taskbar applet plugins, including a name and icon.

TaskbarNamedAppletInterface extends TaskbarAppletInterface with a name and an icon. These are used in the taskbar settings dialog.

If the TaskbarNamedAppletInterface is not available, a name for the applet will be created from the plugin library name.

See also TaskbarAppletInterface.

Member Function Documentation

The icon() function returns an icon for the applet. This may be displayed in the taskbar settings dialog.

QString TaskbarNamedAppletInterface::name () const [pure virtual]

The name() function returns the name of the applet. This will be displayed in the taskbar settings dialog.
