Qtopia - Installing Qtopia on a Device

This document provides some recommendations for installing Qtopia on a device.

Device-Specific Code

Device specific code is found in src/libraries/qtopia/, in the files custom-*.*. See the existing files, especially the linux-generic file, for guidance.

Device-Specific Files

The default hardware button mappings for a device are defined by the etc/defaultbuttons-*.conf files. See the existing files, especially the generic file, for guidance. Images of the buttons, which should correspond to the appearance of the device hardware, are in pics/Button/n.png, with n corresponding to the button number in the configuration file.

Non-installed Components

There are a number of applications/features supplied with Qtopia that are not intended to be shipped with a production device:

  • Rotation Setting - This is provided to demonstrate Qtopia's ability to display in different orientations.
  • Shutdown Setting - This is provided as a development tool. It should not be necessary in a production device. It can be removed by deleting apps/Settings/quit.desktop
