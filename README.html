Welcome to Qtopia

Overview

This is Trolltech's Qtopia device application development kit. It is intended for original equipment manufacturers who are developing applications to run on the device of their choice.

This package contains the full source code for Qtopia plus parts of the source code from Qt and TMake. Apart from this source package it should not be necessery to install any other Trolltech package to be able to compile the Qtopia product.

Documentation

The package contains the Qtopia reference documentation including all relevant Qt documentation. The documentation includes the following:

  • Qtopia documentation index : the starting page for all documentation.
  • Getting Started page : for help with installing, building and running Qtopia.
  • Building from Source page : for detailed help with building Qtopia from source.

For the most up-to-date information, please refer to the Trolltech Documentation Site.
