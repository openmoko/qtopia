Welcome to Qt Extended

Overview

This is Trolltech's Qt Extended Open Source edition. See the LICENSE file for the licensing information of this software.

This package contains source code for Qt Extended plus parts of the source code of Qt. Apart from this source package it should not be necessery to install any other Trolltech package to be able to compile Qt Extended.

Documentation

The package contains the Qt Extended reference documentation including all relevant Qt documentation. The documentation includes the following:

  • Qt Extended documentation index: the starting page for all documentation.
  • Getting Started page: for help with installing, building and running Qt Extended.
  • Guide to Configuring and Building Qt Extended: for detailed help with building Qt Extended from source.

For the most up-to-date information, please refer to the Trolltech Documentation Site.
