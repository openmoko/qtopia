You will need the software listed below to be able to compile and use Qtopia. This software is included in this package.

  • Qtopia 1.6.x
  • Qt/Embedded 2.3.x (upon which Qtopia is built)
  • Qt 2.3.2 for X11 (for emulated desktop development)

How to compile

Below is a description of how to get everything built quickly without having to read the detailed instructions provided with each of the packages above.

  1. Unpack Qtopia... 
    
   tar xfz qtopia-free-1.6.x.tar.gz
   cd qtopia-1.6.x
   export QPEDIR=$PWD
   cd ..
  2. Install Qt/Embedded... 
    
   tar xfz qt-embedded-2.3.x-free.tar.gz
   cd qt-2.3.x
   export QTDIR=$PWD
   export QTEDIR=$QTDIR
   export PATH=$QTDIR/bin:$PATH
   export LD_LIBRARY_PATH=$QTDIR/lib:$LD_LIBRARY_PATH
   cp $QPEDIR/src/qt/qconfig-qpe.h src/tools/
   ./configure -qconfig qpe -qvfb -depths 4,8,16,32
   make sub-src
   cd ..
  3. Install Qt/X11 2.3.2... 
    
   tar xfz qt-x11-2.3.2-free.tar.gz
   cd qt-2.3.2
   export QTDIR=$PWD
   export PATH=$QTDIR/bin:$PATH
   export LD_LIBRARY_PATH=$QTDIR/lib:$LD_LIBRARY_PATH
   ./configure
   make
   make -C tools/qvfb
   mv tools/qvfb/qvfb bin
   cp bin/uic $QTEDIR/bin
   cd ..
  4. Build Qtopia... 
    
   cd qtopia-1.6.x
   export QTDIR=$QTEDIR
   export QPEDIR=$PWD
   export PATH=$QPEDIR/bin:$PATH
   cd src
   ./configure
   make
   cd ../..

How to run

  1. Run the Qt Virtual Framebuffer (an X11 application)... 
    
   qvfb &
  2. Run Qtopia (a Qt/Embedded application, will show in QVFb)... 
    
   export QTDIR=$QTEDIR
   qpe &

How to use

With the above procedure, the Qtopia environment will be running the Qt Virtual Framebuffer.
