Qtopia PDA Kit

Overview

Welcome to the Qtopia PDA Kit.

This is Trolltech's Qtopia kit for personal digital assistants (PDAs). It is intended for original equipment manufacturers who are developing PDAs that will run Qtopia.

This package contains all of the files required for full library and application source code access.

Support

Trolltech maintains support mailing lists where the open-source community can ask and answer questions.

Please go to Trolltech Mailing List index to join our mailing lists or to browse the archives. There is a specific mailing list for Qtopia relatedissues.

Documentation

Qtopia documentation includes the relevant Qt documentation and is installed as part of the package. This release includes both developer documentation and end-user documentation. The developer documentation includes the following:

  • Qtopia documentation index.
    • Qtopia Platform introduction.
    • Getting Started page
  • Qt documentation index

The end-user documentation can be found in the help subdirectory 

(image/opt/Qtopia/help/html)
and is meant to be browsed by the Qtopia Help Browser (which includes a preprocessor that removes the tags visible in a normal browser).

For the most up-to-date information, please refer to the Trolltech Documentation Site.
